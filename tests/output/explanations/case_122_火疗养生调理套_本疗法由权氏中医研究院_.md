# Case Analysis

**Input:** 火疗养生调理套。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。我们不做治疗，只做调理。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。

**Risk Level:** 8/10

**Verdict:** scam

---

### 风险分析报告

**产品名称**：火疗养生调理套

**风险等级**：8/10（高度可疑，接近诈骗）

**主要风险信号**：

1. **虚假权威背书**：声称由‘权氏中医研究院（已注销）’创始人研发。机构已注销，说明其可能已失去合法运营资格或存在不良记录，却仍被用来背书，属于典型的虚假权威营销。

2. **伪科学概念**：
   - ‘火疗打开毛孔排毒通道’：人体排毒主要通过肝脏、肾脏和肠道，皮肤排毒（如出汗）作用有限，且‘排毒通道’是伪科学术语。火疗本身有烫伤风险，无科学证据支持其排毒功效。
   - ‘灵芝孢子粉激活自愈力’：灵芝孢子粉作为保健品有一定免疫调节作用，但‘激活自愈力’是夸大宣传，暗示能治疗疾病，违反广告法。

3. **让步免责声明**：明确说‘不做治疗，只做调理’，这是典型的免责话术，试图在暗示疗效的同时规避监管。真正合法的保健品或服务不会使用这种矛盾表述。

4. **内服外用组合营销**：将火疗（外用）和灵芝孢子粉（内服）捆绑销售，制造‘系统调理’的假象，增加消费者信任感，但两者均无可靠证据支持其宣称的协同效果。

**结论**：该产品利用已注销机构背书、伪科学概念和免责声明，具有高度欺诈风险。建议消费者避免购买，并向市场监管部门举报。
