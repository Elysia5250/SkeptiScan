# Case Analysis

**Input:** 精准基因防癌筛查套餐。我们不是医疗机构，不提供诊断。与北京某三甲医院（退休医生个人合作）提供报告解读服务。但据统计，90%的早期癌症患者因未及时发现而错过最佳治疗期。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 7/10

**Verdict:** scam

---

## 详细分析

### 1. 产品性质与风险识别
该产品为“精准基因防癌筛查套餐”，但明确声明“不是医疗机构，不提供诊断”，这本身存在矛盾：筛查服务通常属于医疗行为，非医疗机构提供此类服务可能违反相关法规。同时，其声称与“北京某三甲医院（退休医生个人合作）”提供报告解读，这种模糊的权威背书难以验证，退休医生个人合作不等于医院官方认可，存在虚假背书风险。

### 2. 营销手段分析
- **恐惧营销**：引用“90%的早期癌症患者因未及时发现而错过最佳治疗期”这一统计数据，制造紧迫感和恐惧心理，诱导消费者购买。该数据来源不明，且可能被断章取义。
- **免责声明**：通过“不提供诊断”和“广告内容由品牌方提供”等声明，试图规避医疗责任和平台责任，这是典型的欺诈手法。
- **模糊权威**：利用“三甲医院”的声誉，但实际仅为退休医生个人合作，消费者难以核实，容易误以为有医院背书。

### 3. 风险等级评估
综合以上分析，该产品存在以下高风险信号：
- **虚假权威背书**：退休医生个人合作无法等同于医院认可。
- **恐惧营销**：利用癌症数据制造紧迫感。
- **法律风险**：非医疗机构提供筛查服务可能违法。
- **数据欺诈**：90%的数据缺乏可靠来源。

因此，风险等级评定为7分（高风险），属于可疑的欺诈性产品。

### 4. 购买建议
强烈不建议购买。消费者应选择正规医疗机构进行癌症筛查，如三甲医院的体检中心或疾控中心。对于任何声称能“防癌”的非医疗产品，应保持警惕，避免因恐惧而上当。
