# Case Analysis

**Input:** 祖传秘方免费试用装。但用了没效果的人，99%是没有按疗程使用。祖传七代秘方（创始人记忆中追溯），纯中药制剂（未取得药品批准文号）。免费申领3天试用装，只需支付快递费39元（实际快递成本12元）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品宣称“祖传七代秘方”“纯中药制剂”，但未取得药品批准文号，以“免费试用装”为名，要求用户支付39元快递费（实际成本约12元）。同时暗示“无效是因为未按疗程使用”，将责任转嫁给消费者。

### 风险信号识别

1. **虚假/夸大疗效暗示**：
   - “祖传七代秘方”无法验证，属于典型的伪传统营销话术。
   - “纯中药制剂”但无药品批准文号，涉嫌违反《药品管理法》——未取得批准文号的药品按假药论处。
   - “无效是因为未按疗程使用”是一种典型的免责声明，暗示产品有效，但将无效归咎于用户，属于欺诈性营销。

2. **变相收费（钓鱼式营销）**：
   - “免费申领3天试用装，只需支付快递费39元”——实际快递成本仅12元，差额27元实为产品费用，属于变相销售。
   - 此类手法常见于针对老年人的保健品骗局，利用“免费”吸引，再通过后续高价疗程或产品牟利。

3. **缺乏合法资质**：
   - 未取得药品批准文号，意味着该产品未经国家药监局安全性、有效性审查，存在严重健康风险。
   - “市场价仅供参考，实际价格以门店为准”进一步模糊定价，可能后续有隐藏收费。

### 风险等级评定
- **8分（高风险）**：
  - 明确存在无证销售药品的违法行为。
  - 使用“祖传秘方”等无法验证的伪科学概念。
  - 以免费为诱饵收取不合理费用。
  - 暗示疗效并推卸责任。

### 购买建议
- **绝对不要购买**。该产品无合法资质，疗效无保障，且营销手法具有欺诈性。
- 如需治疗疾病，请到正规医院就诊，购买有国药准字号的药品。
- 如已购买，可向当地市场监督管理局或12315投诉举报。

### 针对老年用户的特别提醒
老年人常成为此类骗局的目标。请记住：任何“免费试用”但要求支付高额快递费的产品，都极有可能是骗局。真正的药品一定有国药准字号，且不会以“祖传秘方”为卖点。
