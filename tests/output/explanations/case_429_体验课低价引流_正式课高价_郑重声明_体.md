# Case Analysis

**Input:** 体验课低价引流-正式课高价。郑重声明：体验营无强制消费。体验营期间可免费体验：针灸、拔罐、艾灸各一次。报名体验营即送500元抵用券。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

## 详细分析

该商品宣传的核心是“体验课低价引流-正式课高价”，这是一种典型的营销策略。虽然声明“体验营无强制消费”，但并未说明体验后是否会进行强力推销或设置消费门槛。赠送500元抵用券看似优惠，但未明确使用条件，可能要求消费满一定金额或仅限特定项目，属于常见的赠品陷阱。

“市场价仅供参考，实际价格以门店为准”这一表述模糊了价格透明度，消费者无法提前知晓真实费用，容易在体验后被收取高价。此外，针灸、拔罐、艾灸属于医疗保健行为，若操作人员无资质，可能存在健康风险。

综合来看，该商品存在明显的营销诱导和价格不透明问题，风险等级为6（可疑），建议消费者保持警惕，尤其是老年人更易受低价和赠品诱惑，需仔细核实所有细节。
