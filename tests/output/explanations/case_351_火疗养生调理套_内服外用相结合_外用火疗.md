# Case Analysis

**Input:** 火疗养生调理套。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。已有超过10万人受益。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。个体效果存在差异，请理性看待。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 1. 伪科学概念
- **排毒通道**：医学上不存在“排毒通道”这一概念，火疗无法科学证明能打开毛孔排毒。
- **激活自愈力**：灵芝孢子粉虽有研究，但“激活自愈力”是模糊夸大，无具体机制或数据支持。

### 2. 虚假权威背书
- **权氏中医研究院（已注销）**：机构已注销，说明其合法性存疑，创始人背景无法验证。
- **三十年研发**：无具体专利或论文支持，仅靠时间长度制造信任。

### 3. 数据欺诈
- **超过10万人受益**：无任何统计方法、样本来源或第三方验证，属于典型夸大宣传。

### 4. 营销策略
- 结合“内服外用”增加产品复杂度，掩盖缺乏证据的事实。
- 使用“道医养生”等模糊术语吸引对传统医学不熟悉的消费者。

### 5. 风险等级评估
- 综合伪科学、虚假权威、数据欺诈，风险等级为8（高），属于严重可疑产品。

### 6. 建议
- 消费者应避免购买，尤其是老年人易受“排毒”“自愈”等概念误导。
- 如需调理，应选择有国药准字或正规医疗机构推荐的产品。
