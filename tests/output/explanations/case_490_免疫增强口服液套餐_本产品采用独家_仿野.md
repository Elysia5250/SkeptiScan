# Case Analysis

**Input:** 免疫增强口服液套餐。本产品采用独家「仿野生」灵芝孢子粉+虫草多糖复合配方，通过调节树突状细胞活性增强机体免疫功能。中国科学院微生物研究所（合作关系，非官方认可）提供技术支持。由XX集团（前身：天津天狮生物工程有限公司）荣誉出品。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 6/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为免疫增强口服液套餐，主打灵芝孢子粉和虫草多糖复合配方，声称能通过调节树突状细胞活性增强免疫力。产品提及中国科学院微生物研究所提供技术支持，但明确标注为“合作关系，非官方认可”，这是一种典型的模糊权威背书策略。出品方XX集团前身为天津天狮生物工程有限公司，天狮曾因传销和虚假宣传被多次曝光，这一背景增加了产品的风险性。

### 风险信号识别
1. **伪科学/夸大宣传**：声称“调节树突状细胞活性”属于高度专业的免疫学概念，但未提供任何具体研究数据或临床试验支持，普通消费者难以验证。
2. **虚假权威背书**：虽然标注了“非官方认可”，但提及中科院仍可能误导消费者认为产品有官方科研背景，属于打擦边球。
3. **公司历史风险**：前身公司天津天狮曾涉及传销和虚假宣传，可能暗示该产品延续了类似营销模式。
4. **缺乏监管认证**：未提及任何国家药品或保健食品批准文号（如国药准字、国食健字），产品合法性存疑。

### 评分理由
综合以上风险，该产品存在模糊权威背书、夸大宣传、公司历史不良等问题，但未直接宣称治疗疾病或使用极端伪科学术语，因此风险等级定为6分（中等偏高）。建议消费者高度警惕，避免购买。
