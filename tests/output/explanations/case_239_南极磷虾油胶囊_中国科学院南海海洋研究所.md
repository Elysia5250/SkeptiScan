# Case Analysis

**Input:** 南极磷虾油胶囊。中国科学院南海海洋研究所（联合实验室合作项目）参与研发。适宜人群：血脂偏高者。本品为保健食品，保健功能：辅助降血脂。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 1/10

**Verdict:** not_scam

---

### 详细分析

**1. 伪科学包装（0分）**
产品描述中未出现量子、磁疗、排毒、小分子水等典型伪科学术语，仅提及磷虾油这一常见保健食品原料，无伪科学包装风险。

**2. 虚假医疗声称（0分）**
产品明确标注“保健功能：辅助降血脂”，属于保健食品的法定功能范围，未声称治疗或替代药物，无虚假医疗声称。

**3. 虚假权威背书（5分）**
产品宣称“中国科学院南海海洋研究所（联合实验室合作项目）参与研发”，但未提供具体合作项目名称、研究人员姓名或可验证的官方链接。这种模糊的权威背书容易让消费者误以为产品有官方背书，但实际无法核实，存在一定风险。

**4. 违法/传销特征（0分）**
描述中无多级返利、拉人头、保本收益等传销或非法金融活动特征。

**5. 诱导欺诈营销（0分）**
无限时优惠、免费领、无效退款等诱导性营销语言。

**6. 数据欺诈（0分）**
未引用任何研究数据或实验结论，无数据欺诈风险。

**7. 可信信号（-5分）**
产品明确标注为保健食品，且功能声称符合批准范围，属于正常营销语言。但权威背书缺乏可验证性，因此仅扣减5分。

**综合判断**：该产品整体描述规范，主要风险在于模糊的权威背书，建议消费者核实合作方信息后再决定购买。
