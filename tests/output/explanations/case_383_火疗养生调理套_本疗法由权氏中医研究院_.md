# Case Analysis

**Input:** 火疗养生调理套。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。我们不做治疗，只做调理。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。

**Risk Level:** 8/10

**Verdict:** scam

---

该产品宣传存在多个高风险信号。首先，声称“火疗打开毛孔排毒通道”属于典型的伪科学排毒概念，缺乏科学定义和验证标准。其次，“灵芝孢子粉激活自愈力”是模糊的健康声称，没有提供任何临床数据或权威认证。第三，提及“权氏中医研究院（已注销）”，利用已注销机构增加权威感，属于虚假权威误导。产品还模糊了“调理”与“治疗”的界限，暗示对疾病有效，但声称不做治疗，规避监管。综合来看，该产品风险较高，建议消费者避免购买。
