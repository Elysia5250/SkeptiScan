# Case Analysis

**Input:** 免疫增强口服液套餐。购买满3000元成为VIP会员，享受团队业绩分红。由XX集团（前身：天津天狮生物工程有限公司）荣誉出品。中国科学院微生物研究所（合作关系，非官方认可）提供技术支持。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 风险等级：8（高度可疑，接近诈骗）

### 风险信号识别

1. **虚假/夸大功效暗示**：产品名称为“免疫增强口服液”，直接暗示具有增强免疫力的功效。但未提供任何临床试验数据或科学依据。根据《食品安全法》，普通食品不得宣传疾病预防或治疗功能，保健食品需有蓝帽子标志。该产品未明确标注为保健食品，属于违规暗示。

2. **权威机构背书滥用**：宣称“中国科学院微生物研究所提供技术支持”，但括号内注明“合作关系，非官方认可”。这是一种典型的“偷换概念”手法——利用中科院的声誉增加可信度，但实际合作可能仅限于名义上的技术交流，而非产品研发。消费者容易误以为产品得到中科院官方认证。

3. **传销式营销模式**：要求购买满3000元成为VIP会员，并享受“团队业绩分红”。这符合传销的典型特征：设置高额入门费、以发展下线（团队）获取回报。XX集团前身天津天狮生物工程有限公司曾因传销活动被多次查处，历史风险极高。

4. **免责声明**：“本广告内容由品牌方提供，不代表本平台立场”是平台推卸责任的常见手段，但并不能免除品牌方的法律责任。消费者应警惕此类声明背后的虚假宣传。

### 事实核查建议

- 查询国家企业信用信息公示系统，确认XX集团及天津天狮的行政处罚记录。
- 联系中国科学院微生物研究所，核实其与该公司是否有技术合作，以及合作是否涉及该口服液。
- 在国家药监局官网查询该口服液是否有保健食品批准文号（国食健字）或药品批准文号（国药准字）。

### 购买建议

**强烈不建议购买**。该产品结合了虚假功效宣传、权威机构背书滥用、传销式营销三大高风险特征。尤其是“团队分红”模式，极可能涉及非法传销，参与者不仅无法获得承诺收益，还可能面临法律风险。对于普通消费者，尤其是老年人，应远离此类产品，避免财产损失。
