# Case Analysis

**Input:** 免费领智能血糖仪。活动说明：血糖仪免费送，耗材有偿提供。我们靠耗材赚钱。免费领！原价598元的智能语音血糖仪（市场同类产品实际价格50-80元），现在0元领取！只需支付耗材费299元（含50条试纸+采血针），后续试纸需从我司购买（单价6.9元/条，市场价2元/条以下）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 产品/活动概述
该活动宣称“免费领智能血糖仪”，但要求用户支付299元耗材费（含50条试纸+采血针），并规定后续试纸必须从该公司购买，单价6.9元/条。活动文案中标注“原价598元”，但承认市场同类产品实际价格仅50-80元。

### 风险信号识别

1. **虚假原价与价格欺诈**：声称原价598元，但实际市场价仅50-80元，属于典型的虚高原价，制造虚假优惠。
2. **钓鱼式营销（bait-and-switch）**：以“免费”为诱饵，实际通过高价耗材和后续绑定消费获利。用户支付299元后，后续试纸单价6.9元/条，远高于市场均价（2元/条以下）。
3. **消费锁定**：要求后续试纸必须从该公司购买，限制了用户选择权，形成长期高额消费。
4. **信息不透明**：活动说明中“市场价仅供参考，实际价格以门店为准”为模糊条款，可能进一步加价。

### 风险等级评估
- **风险等级：7/10**（属于高风险，接近诈骗）
- 理由：该活动不涉及虚假医疗功效或伪科学，但通过价格欺诈、钓鱼营销和消费锁定，对消费者（尤其是老年人）造成显著经济损失。符合“钓鱼/后续收费”风险信号，且虚高原价属于欺诈行为。

### 购买建议
- **不推荐参与**：直接购买正规品牌血糖仪（如罗氏、强生、鱼跃等）和试纸，总成本更低且无绑定消费。
- 如需血糖仪，建议选择一次性购买，避免任何“免费”陷阱。

### 对老年用户的特别提醒
老年人常被“免费”字样吸引，但此活动实际成本极高。请务必告知家中长辈：不要相信免费领取，直接去药店或正规电商平台购买知名品牌产品，更安全、更划算。
