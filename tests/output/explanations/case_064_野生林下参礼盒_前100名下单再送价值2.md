# Case Analysis

**Input:** 野生林下参礼盒。前100名下单再送价值299元的养生壶一台。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。产地直销，无中间商赚差价！只要198元！原价1280元的林下参礼盒，中秋特惠仅需198元。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

## 风险分析报告

### 商品概述
该商品为“野生林下参礼盒”，宣传中强调生长年限15年以上、产地直销、原价1280元现价198元，并附赠养生壶。

### 风险信号识别
1. **虚假宣传**：明确标注“生长年限15年以上（实际林下参5-8年）”，这是典型的年限虚标，属于虚假宣传。林下参的生长年限是核心价值指标，虚标年限会误导消费者。
2. **价格欺诈**：原价1280元现价198元，折扣幅度极大。但“市场价仅供参考，实际价格以门店为准”的免责声明，暗示原价可能被故意抬高，属于价格锚定策略。
3. **限时促销**：前100名赠送养生壶，制造紧迫感，诱导冲动消费。
4. **产地直销话术**：“无中间商赚差价”是常见营销话术，但无法验证，且不能保证价格合理。

### 风险等级评估
综合以上信号，该商品存在虚假宣传和价格欺诈，但未涉及医疗功效或伪科学，因此风险等级为6（可疑）。

### 购买建议
- 不建议购买，尤其是对林下参不了解的消费者。
- 如需购买人参类产品，应选择有信誉的品牌或正规药店，并索要产地证明和年限检测报告。
- 对“原价”和“限时优惠”保持警惕，理性消费。
