# Case Analysis

**Input:** 免疫增强口服液套餐。多位肿瘤患者服用后白细胞计数显著回升，部分案例见内部资料汇编。本产品采用独家「仿野生」灵芝孢子粉+虫草多糖复合配方，通过调节树突状细胞活性增强机体免疫功能。中国科学院微生物研究所（合作关系，非官方认可）提供技术支持。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 详细分析

### 风险评分：7/10（高度可疑，接近诈骗）

### 主要风险信号

1. **虚假疗效暗示**：
   - 声称“多位肿瘤患者服用后白细胞计数显著回升”，但未提供任何可验证的临床数据，仅提及“内部资料汇编”。这属于典型的**健康焦虑营销**，利用患者对癌症和白细胞降低的恐惧来推销产品。
   - 白细胞回升是化疗后常见的医学需求，但任何声称能实现此效果的保健品都必须有严格的临床试验支持。该产品未提供任何公开、可查证的证据。

2. **伪科学包装**：
   - “独家「仿野生」灵芝孢子粉+虫草多糖复合配方”——“仿野生”是营销术语，无科学定义；灵芝和虫草在传统医学中确有应用，但现代医学对其免疫调节作用尚无定论，且口服制剂能否达到有效浓度存疑。
   - “通过调节树突状细胞活性增强机体免疫功能”——树突状细胞是免疫学中的专业概念，但该产品未提供任何机制研究或人体数据来支持这一说法，属于**概念偷换**，用科学术语包装无依据的宣称。

3. **虚假权威背书**：
   - “中国科学院微生物研究所（合作关系，非官方认可）提供技术支持”——括号内的“非官方认可”是典型的**让步免责声明**，既想借中科院的名头，又避免被追究虚假宣传责任。真正的合作通常会有官方公告或可查证的协议，而非这种模糊表述。

4. **价格陷阱**：
   - “市场价仅供参考，实际价格以门店为准”——这是**模糊定价**手段，可能后续以高价销售，或设置隐形消费（如要求购买配套产品）。

### 事实核查结果

- 未能在公开渠道（如国家药品监督管理局、国家市场监督管理总局）找到该产品的保健食品或药品批准文号。
- 中科院微生物研究所官网未发现与该产品的合作信息。
- 网络上无该产品的第三方评测或用户反馈（除可能存在的营销软文外）。

### 最终建议

该产品属于**高风险保健品**，其宣传内容严重缺乏科学依据，且利用权威机构名称和模糊案例误导消费者。**强烈不建议购买**，尤其是肿瘤患者应坚持正规治疗，避免因使用此类产品延误病情或产生经济损失。
