# Case Analysis

**Input:** 弱激光鼻腔照射治疗仪。该技术源自NASA宇航员血液健康研究，后经瑞典科学家进一步验证。北京安贞医院、上海中山医院多中心临床试验表明，辅助治疗高脂血症总有效率78.5%。本产品仅作为辅助治疗手段，不能替代药物治疗。

**Risk Level:** 3/10

**Verdict:** not_scam

---

## 详细分析

### 1. 伪科学包装（5分）
产品声称“弱激光鼻腔照射治疗仪”源自NASA宇航员血液健康研究，但弱激光治疗在主流医学中未被广泛认可，且鼻腔照射的疗效缺乏充分科学依据。该声称属于“其他伪科学术语”。

### 2. 虚假医疗声称（10分）
产品明确声称“辅助治疗高脂血症总有效率78.5%”，这属于对特定疾病的治疗声称，违反了医疗器械广告不得宣传疗效的规定。虽然声明“不能替代药物治疗”，但有效率数据仍构成虚假医疗声称。

### 3. 虚假权威背书（5分）
引用北京安贞医院和上海中山医院的多中心临床试验，但未提供试验注册号、样本量、发表论文或可验证的机构证明，属于无法验证的权威背书。

### 4. 数据欺诈（5分）
声称技术源自NASA研究，但未提供任何具体论文或证据，属于引用非适用研究作为自身证据。

### 5. 可信信号（-2分）
产品声明“仅作为辅助治疗手段，不能替代药物治疗”，降低了部分风险，但整体仍存在严重问题。

### 总结
该产品宣传存在多项风险信号，尤其是伪科学包装和虚假医疗声称，消费者应高度警惕。
