# Case Analysis

**Input:** 体验课低价引流-正式课高价。郑重声明：体验营无强制消费。体验营期间可免费体验：针灸、拔罐、艾灸各一次。报名体验营即送500元抵用券。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 5/10

**Verdict:** not_scam

---

## 详细分析

### 风险评分：5/10（可疑）

### 风险信号识别

1. **低价引流策略**：宣传中明确提到“体验课低价引流-正式课高价”，这是一种典型的营销手段，通过低价或免费体验吸引消费者，然后通过高价正式课程获利。虽然不一定是欺诈，但存在诱导消费的风险。

2. **赠送抵用券**：报名即送500元抵用券，但未说明使用条件。这种“优惠”往往有严格限制，如必须购买特定课程或达到一定消费金额，容易让消费者产生冲动消费。

3. **医疗行为涉及**：体验项目包括针灸、拔罐、艾灸，这些属于中医医疗行为。根据中国法律，从事针灸等医疗活动需要具备《医疗机构执业许可证》和执业医师资格。宣传中未提及任何资质信息，存在非法行医的风险。

4. **价格模糊**：“市场价仅供参考，实际价格以门店为准”这一表述模糊，消费者无法提前了解真实价格，容易被现场推销高价课程。

### 未发现的风险信号

- 没有虚假疗效承诺（如“治愈疾病”）。
- 没有使用伪科学术语（如量子、磁疗）。
- 没有伪造权威背书。

### 综合评估

该宣传属于典型的“低价引流-高价转化”营销模式，本身不一定是诈骗，但存在以下风险：
- 消费者可能被低价吸引后，被诱导购买高价课程。
- 涉及医疗行为的体验项目，机构可能不具备合法资质。
- 抵用券可能附带隐形消费条件。

建议消费者保持警惕，优先选择有正规资质、口碑良好的机构，避免因贪图小便宜而遭受经济损失或健康风险。
