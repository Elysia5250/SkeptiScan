# Case Analysis

**Input:** 免疫增强口服液套餐。购买满3000元成为VIP会员，享受团队业绩分红。由XX集团（前身：天津天狮生物工程有限公司）荣誉出品。中国科学院微生物研究所（合作关系，非官方认可）提供技术支持。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 3/10

**Verdict:** not_scam

---

## 详细分析

### 1. 伪科学包装 (0分)
未发现量子、磁疗、排毒等典型伪科学术语。产品声称“免疫增强”，但未提供具体机制或科学依据，属于模糊健康声称，但未达到伪科学包装的扣分标准。

### 2. 虚假医疗声称 (0分)
未明确声称治疗疾病或替代药物，仅提及“免疫增强”，属于保健食品常见宣传，但缺乏证据支持。

### 3. 虚假权威背书 (10分)
- **中国科学院微生物研究所（合作关系，非官方认可）**：明确标注“非官方认可”，但利用“中国科学院”名称仍可能误导消费者，属于模糊背书，扣5分。
- **XX集团（前身：天津天狮生物工程有限公司）**：天狮集团历史上曾因传销被处罚，利用其品牌效应可能暗示可靠性，但未直接声称权威认证，扣5分。

### 4. 违法/传销特征 (12分)
- **购买满3000元成为VIP会员，享受团队业绩分红**：明确要求付费入会，且“团队业绩分红”暗示多级分销，符合传销特征，扣8分。
- **VIP会员制度**：属于会员制招募，扣4分。

### 5. 诱导欺诈营销 (0分)
未发现限时优惠、免费转付费等典型诱导手段。

### 6. 数据欺诈 (0分)
未引用研究数据或实验室结果。

### 7. 可信信号 (0分)
未提供任何药品或保健食品批准文号，品牌方非知名正规品牌，且销售模式可疑。

### 综合判断
该产品风险较高，主要风险点在于传销模式和虚假权威背书。消费者应避免购买，并警惕类似销售模式。
