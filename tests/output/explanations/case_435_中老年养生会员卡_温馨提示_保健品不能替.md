# Case Analysis

**Input:** 中老年养生会员卡。温馨提示：保健品不能替代药品。本卡一经售出，概不退换。每月15日会员日，全场买二送一！仅限会员参加。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 4/10

**Verdict:** not_scam

---

该商品为中老年养生会员卡，文字内容包含多个风险信号。首先，温馨提示提到保健品不能替代药品，但未明确说明会员卡是否涉及保健品销售，存在模糊宣传的可能。其次，'市场价仅供参考，实际价格以门店为准'是典型的价格欺诈手法，商家可能虚标市场价再打折，实际价格并不优惠。'本卡一经售出，概不退换'属于霸王条款，限制了消费者的合法权益。营销上利用会员专享和买二送一制造紧迫感，诱导消费。综合判断，风险等级为4，属于可疑水平，建议消费者仔细核实后再决定是否购买。
