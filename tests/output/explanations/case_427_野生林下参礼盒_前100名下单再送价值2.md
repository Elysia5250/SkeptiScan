# Case Analysis

**Input:** 野生林下参礼盒。前100名下单再送价值299元的养生壶一台。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。产地直销，无中间商赚差价！只要198元！原价1280元的林下参礼盒，中秋特惠仅需198元。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

### 风险分析

**1. 商品真实性存疑**
- 宣称“生长年限15年以上”，但括号内注明“实际林下参5-8年”，存在明显矛盾，可能故意夸大年限以抬高价值。
- 林下参通常生长缓慢，15年以上较为罕见，价格远高于198元，低价可能意味着品质低劣或假冒。

**2. 价格欺诈风险**
- 原价1280元，现价198元，折扣幅度极大，但“市场价仅供参考，实际价格以门店为准”表明原价可能虚高，属于典型的“价格锚定”策略。
- 赠送养生壶（价值299元）进一步吸引消费者，但养生壶实际价值可能远低于宣称。

**3. 营销手段分析**
- 限时特惠（中秋）和限量赠送（前100名）制造紧迫感，促使冲动消费。
- “产地直销，无中间商赚差价”是常见话术，但无法验证，且可能掩盖产品质量问题。

**4. 风险等级评估**
- 综合以上，该商品存在虚假宣传、价格欺诈和营销套路，风险等级为6（中等偏高），建议消费者谨慎对待。

**5. 购买建议**
- 如需购买人参类产品，应选择有国药准字或知名品牌的产品，并索要正规发票。
- 对林下参的年限和产地保持怀疑，可要求商家提供检测报告或产地证明。
