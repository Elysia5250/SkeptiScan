# Case Analysis

**Input:** 互联网+消费创富系统。直推奖励20%，间推奖励10%，三代奖励5%。月入过万不是梦：推荐10人，团队裂变至100人，月被动收入超5万元！已有来自全国32个省市超过50万人参与，累计分配利润超过20亿元。平台资质：已取得ICP备案（正在办理中）、与某国有银行（仅签署意向协议）达成资金存管合作。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 10/10

**Verdict:** scam

---

## 风险分析报告

### 总体评价
该推广文案描述的是一个典型的“庞氏骗局”或“传销”模式，风险等级为**10（最高）**。其核心是通过多层级推荐奖励（直推20%、间推10%、三代5%）激励用户不断发展下线，而非通过销售真实产品或服务创造价值。这种模式在中国法律下属于非法传销，参与者不仅无法获得承诺的收益，还可能面临资金损失和法律风险。

### 风险信号识别

1. **多层级返利模式**：直推、间推、三代奖励构成三级分销，超过法律允许的两级限制，属于传销特征。
2. **高额收益承诺**：“月入过万”“月被动收入超5万元”等承诺缺乏合理依据，是典型的诱饵。
3. **虚假或模糊的资质**：ICP备案“正在办理中”却声称“已取得”，银行合作仅为“意向协议”，均属于误导性宣传。
4. **无法验证的数据**：“50万人参与”“分配利润20亿元”没有第三方审计或公开来源，极可能为虚构。
5. **免责声明**：“本广告内容由品牌方提供，不代表本平台立场”试图规避责任，但平台仍可能承担连带责任。

### 事实核查建议
- 通过工信部ICP备案查询系统核实该平台是否已完成备案。
- 联系所称的国有银行官方客服，确认资金存管合作是否真实。
- 搜索该平台名称加上“骗局”“传销”等关键词，查看是否有用户投诉或媒体报道。
- 咨询当地市场监管部门或公安机关，确认该模式是否已被查处。

### 最终建议
**绝对不要参与！** 任何要求您拉人头并承诺高额回报的“创富系统”几乎都是骗局。请立即远离，并向当地公安机关或12315平台举报。保护自己和家人，避免陷入传销陷阱。
