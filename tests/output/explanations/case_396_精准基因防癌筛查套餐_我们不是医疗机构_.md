# Case Analysis

**Input:** 精准基因防癌筛查套餐。我们不是医疗机构，不提供诊断。与北京某三甲医院（退休医生个人合作）提供报告解读服务。但据统计，90%的早期癌症患者因未及时发现而错过最佳治疗期。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 7/10

**Verdict:** scam

---

### 详细分析

**1. 风险信号识别**
- **虚假紧迫感**：广告中‘90%的早期癌症患者因未及时发现而错过最佳治疗期’这一数据未提供来源，属于典型的制造焦虑手法，诱导消费者快速下单。
- **模糊权威背书**：声称与‘北京某三甲医院（退休医生个人合作）’合作，但‘退休医生个人合作’意味着并非医院官方行为，无法保证服务质量和医疗合规性。
- **免责声明**：明确表示‘不是医疗机构，不提供诊断’，这实际上是在规避医疗责任，暗示检测结果可能不具临床诊断价值。

**2. 营销模式分析**
- 系统预检已识别出‘制造虚假紧迫感的高压销售’，这与广告中使用的统计数据相吻合。
- 广告末尾‘本广告内容由品牌方提供，不代表本平台立场’进一步削弱了可信度，表明平台未对内容进行审核。

**3. 事实核查建议**
- 基因防癌筛查属于高风险检测，必须确认检测技术是否获得国家药监局（NMPA）批准。
- 合作医生的资质和医院授权情况需公开可查，否则属于虚假背书。
- 统计数据应提供原始研究出处，否则视为误导。

**4. 购买建议**
- 对于普通消费者，尤其是老年人，此类宣传极易引发健康焦虑。建议优先选择公立医院或正规体检机构进行癌症筛查。
- 如果确实需要基因检测，应咨询医生并选择有明确资质和监管的产品。

**5. 总结**
该商品利用模糊的权威背书和未经证实的统计数据，结合免责声明，构成高风险营销。风险等级为7分（可疑），建议避免购买。
