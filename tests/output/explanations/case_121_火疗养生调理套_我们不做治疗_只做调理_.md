# Case Analysis

**Input:** 火疗养生调理套。我们不做治疗，只做调理。零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持。一个疗程调理：高血压、糖尿病、风湿骨痛、妇科问题全面改善。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为“火疗养生调理套”，宣传材料中声称可以调理高血压、糖尿病、风湿骨痛、妇科问题等多种慢性病，同时推广“零风险创业”加盟模式，承诺月入十万。

### 风险信号识别

1. **虚假疗效宣称（高风险）**：
   - 明确列出“高血压、糖尿病、风湿骨痛、妇科问题”等具体疾病，并声称“一个疗程调理……全面改善”。这违反了《广告法》关于非药品不得宣传疾病治疗功效的规定。
   - 使用“调理”一词试图规避监管，但结合具体疾病名称，实质构成医疗承诺。

2. **投资诈骗特征（高风险）**：
   - “零风险创业”和“月入十万不是梦”是典型的投资诈骗话术，承诺高额回报以吸引加盟费。
   - 缺乏任何可验证的盈利数据或成功案例，属于空头支票。

3. **价格不透明（中风险）**：
   - “市场价仅供参考，实际价格以门店为准”为后续加价或隐藏费用留下空间，常见于钓鱼式销售。

4. **伪科学概念**：
   - “火疗”本身缺乏现代医学证据支持，且操作不当可能导致烧伤。宣传中未提供任何科学依据或临床数据。

### 评分依据
- **风险等级9**：该产品同时包含虚假医疗宣称和投资诈骗要素，属于严重欺诈。
- 虚假疗效宣称（+4分）
- 投资诈骗话术（+3分）
- 价格不透明（+1分）
- 伪科学概念（+1分）

### 事实核查建议
- 查询国家药品监督管理局数据库，确认该产品是否有医疗器械或药品注册号。
- 搜索“火疗加盟 骗局”等关键词，查看是否有消费者投诉或警方通报。
- 要求提供加盟合同、盈利证明、产品检测报告等文件。

### 最终建议
**强烈不推荐购买或加盟**。该产品本质是利用老年人健康焦虑和创业梦想的骗局。建议向当地市场监管部门举报。
