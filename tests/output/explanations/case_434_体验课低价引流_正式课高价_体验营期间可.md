# Case Analysis

**Input:** 体验课低价引流-正式课高价。体验营期间可免费体验：针灸、拔罐、艾灸各一次。3天2夜国学养生体验营，仅需99元（含食宿）！原价2980元。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析

### 1. 营销模式风险
该商品采用典型的“低价引流+高价转化”模式：先以99元（含食宿）的体验课吸引消费者，体验内容包括针灸、拔罐、艾灸各一次，随后推销高价调理套餐（基础套餐19800元/年，尊享套餐39800元/年）。这种模式常见于保健品、养生机构，容易导致消费者在体验后因心理压力或信息不对称而冲动消费。

### 2. 价格不透明
标注“原价2980元”但实际体验价仅99元，价格差异巨大，缺乏合理成本解释。同时声明“市场价仅供参考，实际价格以门店为准”，规避了明码标价的法律责任，消费者在购买前无法获知最终价格，存在隐性收费风险。

### 3. 缺乏资质证明
商品提及针灸、拔罐、艾灸等中医诊疗服务，但未提供任何医疗机构执业许可证、医师资格证或相关资质证明。根据《医疗机构管理条例》，开展针灸等医疗行为必须取得相应资质，否则属于非法行医。

### 4. 疗效承诺缺失
调理方案费用高昂，但未说明具体服务内容、疗程、预期效果或科学依据。消费者无法评估其价值，容易受到夸大宣传误导。

### 5. 目标人群风险
此类营销常针对中老年人，利用其对健康的焦虑和对低价优惠的敏感，诱导其支付高额费用。一旦购买，退款困难，维权成本高。

### 结论
该商品存在明显的消费陷阱风险，建议消费者谨慎对待，优先选择有正规资质、价格透明、口碑良好的医疗机构。
