# Case Analysis

**Input:** 野生林下参礼盒。限量500份，售完即止！已售473份，仅剩27份！。但我们实事求是地说，人参不能替代药品治疗。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 5/10

**Verdict:** scam

---

## 详细分析

### 风险信号识别

1. **夸大宣传**：宣传中声称“生长年限15年以上”，但括号内注明“实际林下参5-8年”，存在明显矛盾。林下参的自然生长年限通常为5-8年，15年以上极为罕见且价格高昂，此宣传可能误导消费者。

2. **紧迫感营销**：使用“限量500份，已售473份，仅剩27份”等话术，制造稀缺性和紧迫感，促使消费者冲动购买。这种手法常见于高风险的促销活动中。

3. **免责声明**：广告中明确“人参不能替代药品治疗”，虽然看似诚实，但结合夸大宣传，可能是一种“让步免责”策略，即先承认不足，再通过其他夸大信息诱导购买。

4. **来源不明**：广告内容由品牌方提供，平台不承担责任，这降低了信息的可信度。消费者无法验证品牌方的资质和产品真实性。

### 风险等级评估

- **1-3分（安全）**：不符合，因为存在明显夸大和紧迫感营销。
- **4-6分（可疑）**：符合。产品本身可能为真实人参，但宣传内容存在夸大和误导，属于中等风险。
- **7-10分（诈骗）**：不适用，因为未涉及伪科学、虚假疗效或投资诈骗等严重欺诈行为。

综合判断，风险等级为**5分**，属于可疑范畴，建议消费者谨慎对待。

### 建议

- 消费者应要求品牌方提供生长年限的权威检测报告或产地证明。
- 对比市场价格，若价格异常低廉或过高，需警惕。
- 优先选择有良好口碑和透明信息的商家。

### 最终结论

该商品宣传存在夸大和误导性内容，但未达到诈骗级别。消费者在购买前应核实信息，避免冲动消费。
