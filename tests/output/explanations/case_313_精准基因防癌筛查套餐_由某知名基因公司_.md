# Case Analysis

**Input:** 精准基因防癌筛查套餐。由某知名基因公司（已停止该业务）提供技术支持。检测发现高风险基因者，可搭配购买本中心定制防癌套餐（另收费29800元/年）。检测结果仅供参考。

**Risk Level:** 2/10

**Verdict:** not_scam

---

## 详细分析

### 1. 伪科学包装
未发现量子、磁疗、排毒等典型伪科学术语，得分为0。

### 2. 虚假医疗声称
商品声称检测高风险基因者需搭配购买定制防癌套餐（29800元/年），暗示该套餐可预防或治疗癌症，属于疾病治疗暗示，得10分。

### 3. 虚假权威背书
商品声称由某知名基因公司提供技术支持，但注明已停止该业务，无法验证当前背书真实性，得5分。

### 4. 违法/传销特征
未发现多级返利、拉人头等传销特征，得分为0。

### 5. 诱导欺诈营销
商品利用基因检测结果制造健康焦虑，诱导购买高额套餐，属于紧迫感营销，得3分。

### 6. 数据欺诈
未发现引用不相关研究或动物实验数据，得分为0。

### 7. 可信信号
商品明确标注检测结果仅供参考，未直接声称治疗功效，得-2分。

### 综合评估
该商品存在中等风险，主要风险点在于利用基因检测结果制造健康焦虑，诱导购买高额防癌套餐，且技术支持方已停止业务，背书真实性存疑。建议消费者谨慎对待，避免因健康焦虑而购买高价无效产品。
