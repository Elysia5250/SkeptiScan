# Case Analysis

**Input:** 中老年养生会员卡。每月15日会员日，全场买二送一！仅限会员参加。本卡一经售出，概不退换。一次性充值满5000元升级钻石会员，额外赠送价值2980元的多功能理疗仪一台（出厂价约200元）。

**Risk Level:** 6/10

**Verdict:** scam

---

## 风险分析报告

### 总体评价
该商品为中老年养生会员卡，通过充值升级和赠送理疗仪的方式吸引消费者。经分析，其风险等级为**6分（可疑）**，属于需要高度警惕的促销活动。

### 风险信号识别
1. **赠品价值严重虚标**：声称赠送价值2980元的多功能理疗仪，但出厂价仅200元，虚标幅度超过10倍。这是典型的营销陷阱，利用消费者对“高价值赠品”的贪便宜心理。
2. **预付卡不退换**：条款明确“一经售出，概不退换”，违反《单用途商业预付卡管理办法》中关于消费者退卡权利的规定，属于霸王条款。
3. **大额充值诱导**：一次性充值5000元升级钻石会员，门槛较高，且未说明会员权益的具体内容，存在商家跑路或服务质量下降的风险。
4. **目标人群敏感**：针对中老年人，利用其对健康的关注和节俭心理，容易导致非理性消费。

### 事实核查建议
- 查询该理疗仪是否具有医疗器械注册证（如CFDA认证），若无则为普通电子产品，价值极低。
- 通过商务部单用途预付卡备案平台查询该商家是否备案。
- 在12315平台或黑猫投诉等渠道搜索该商家是否有投诉记录。

### 购买建议
- **不建议**进行大额充值，尤其是5000元档位。
- 如确需购买，先以最低金额尝试，并确认会员权益是否明确。
- 保留所有付款凭证、合同或宣传单，以便维权。
- 警惕“买二送一”等促销，可能诱导购买不需要的商品。

### 对老年人的特别提醒
老年人应警惕“免费赠送”“高价值赠品”等话术，不要被小恩小惠迷惑。预付卡一旦充值，退款困难，建议与子女商量后再决定。
