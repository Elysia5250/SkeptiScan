# Case Analysis

**Input:** 南极磷虾油胶囊。不替代药品治疗。上海某医院（研究合作协议，非临床验证）的初步实验表明，36例高血脂志愿者服用3个月后，总胆固醇平均下降1.2mmol/L。适宜人群：血脂偏高者。

**Risk Level:** 5/10

**Verdict:** scam

---

## 详细分析

### 1. 产品定位与声明
产品为南极磷虾油胶囊，明确标注“不替代药品治疗”，属于保健食品或普通食品范畴。但宣传中提及“上海某医院（研究合作协议，非临床验证）的初步实验”，试图借助医院权威性提升可信度，同时用括号注明“非临床验证”规避法律风险，属于典型的模糊宣传策略。

### 2. 数据可信度评估
- **样本量小**：仅36例高血脂志愿者，缺乏代表性，且未说明是否随机分组、有无对照组。
- **效果夸大**：声称总胆固醇平均下降1.2mmol/L，但未提供标准差、P值等统计指标，无法判断是否具有统计学意义。
- **缺乏长期安全性数据**：3个月观察期较短，未报告不良反应。
- **研究性质模糊**：“研究合作协议”不等于临床验证，可能只是初步探索，结果不可作为功效证据。

### 3. 风险信号识别
- **伪科学风险**：磷虾油虽含Omega-3，但降血脂效果因人而异，且该产品未提供国家认可的保健功能评价报告。
- **虚假权威暗示**：利用“上海某医院”字样，但未公开具体机构名称，消费者难以核实。
- **数据欺诈风险**：引用未公开的初步实验数据，可能选择性报告有利结果。

### 4. 合规性分析
- 未提及保健食品批准文号（如国食健字），若为普通食品则不得宣传保健功能。
- 即使有蓝帽子，降血脂功能需经国家审批，且宣传用语需严格符合批准内容。

### 5. 综合风险等级
风险等级5（中等偏上），主要因为数据不完整、权威性存疑，但产品未直接承诺治愈疾病，且标注了不替代药品，故未达到高风险。

### 6. 购买建议
- 建议消费者选择有明确保健食品批号、知名品牌的产品。
- 降血脂应优先遵医嘱用药，保健品仅作为辅助。
- 对任何声称“医院研究”但未提供完整报告的产品保持警惕。
