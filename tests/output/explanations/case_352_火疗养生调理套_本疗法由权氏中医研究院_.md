# Case Analysis

**Input:** 火疗养生调理套。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。我们不做治疗，只做调理。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 1. 伪科学概念
- **排毒**：声称火疗能打开毛孔排毒通道，但人体排毒主要依靠肝脏和肾脏，皮肤排毒缺乏科学依据。
- **激活自愈力**：灵芝孢子粉被宣传为能激活自愈力，但无具体机制和临床证据。

### 2. 虚假权威背书
- **权氏中医研究院（已注销）**：使用已注销机构名称增加可信度，但注销意味着该机构已不存在或存在合规问题，属于典型的虚假权威策略。
- **三十年研发**：无法验证，且无具体专利或论文支持。

### 3. 规避监管措辞
- **不做治疗，只做调理**：试图规避医疗器械或药品监管，但火疗和口服产品仍可能涉及非法行医或虚假宣传。

### 4. 缺乏科学验证
- 火疗本身有烧伤风险，且无证据表明能排毒。
- 灵芝孢子粉作为普通食品或保健品，不能声称治疗或调理疾病。

### 5. 风险等级评估
- 综合伪科学、虚假权威、规避监管、缺乏证据等因素，风险等级为8（高），属于严重可疑产品。

### 6. 建议
- 消费者应避免购买，尤其是老年人易受此类宣传误导。
- 监管部门应调查其是否涉及非法医疗或虚假广告。
