# Case Analysis

**Input:** 火疗养生调理套。我们不做治疗，只做调理。零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持。一个疗程调理：高血压、糖尿病、风湿骨痛、妇科问题全面改善。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 1. 产品性质与宣传手法
该商品名为“火疗养生调理套”，声称“不做治疗，只做调理”，但随后列出“一个疗程调理：高血压、糖尿病、风湿骨痛、妇科问题全面改善”，这实质上是暗示具有治疗效果，属于典型的打擦边球行为。火疗本身缺乏科学依据，且直接接触火焰存在烧伤风险。

### 2. 加盟模式风险
“零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持。”这种承诺高额回报、零风险的加盟模式，符合传销和非法集资的特征。正规加盟项目不会保证具体收入，更不会使用“零风险”这种绝对化用语。

### 3. 价格模糊
“市场价仅供参考，实际价格以门店为准”为后续随意定价、价格欺诈留下空间。

### 4. 风险等级评定
综合以上因素，该产品涉及伪科学（火疗）、虚假医疗宣传、疑似传销加盟、价格不透明，风险极高，因此评为9分。

### 5. 建议
消费者应远离此类产品，老年人尤其要警惕。如需调理身体，请咨询正规医疗机构。
