# Case Analysis

**Input:** 中老年养生会员卡。每月15日会员日，全场买二送一！仅限会员参加。本卡一经售出，概不退换。一次性充值满5000元升级钻石会员，额外赠送价值2980元的多功能理疗仪一台（出厂价约200元）。

**Risk Level:** 5/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品为“中老年养生会员卡”，主要面向中老年群体，通过会员日优惠（买二送一）和充值升级（满5000元送理疗仪）吸引消费。

### 风险信号识别
1. **价格虚标与赠品陷阱**：宣传中称赠送价值2980元的多功能理疗仪，但出厂价仅约200元，差价高达14倍。这属于典型的“高价标价、低价赠品”营销手段，可能误导消费者认为赠品价值很高，从而诱导大额充值。
2. **霸王条款**：明确标注“本卡一经售出，概不退换”，违反《消费者权益保护法》中关于预付卡退换的合理规定，限制了消费者的基本权利。
3. **诱导大额消费**：充值5000元升级钻石会员的门槛较高，且赠品价值被夸大，容易让消费者冲动消费。
4. **目标群体敏感**：针对中老年人群，利用其对健康养生的关注和促销活动的吸引力，可能缺乏足够的风险意识。

### 风险等级评估
综合以上因素，该商品存在明显的营销夸大和条款不合理问题，但尚未发现明确的虚假疗效或诈骗证据。因此，风险等级定为**5分**（可疑），建议消费者谨慎对待。

### 购买建议
- 不要被“会员日”和“赠品”冲昏头脑，先核实理疗仪的真实价格和功能。
- 查询商家是否在工商部门注册，有无消费者投诉记录。
- 优先选择支持退换、条款透明的商家。
- 若决定购买，建议先小额充值体验，避免一次性大额投入。

### 对老年用户的特别提醒
老年人应警惕此类“充值送赠品”活动，切勿因赠品标价高而冲动消费。购买前务必与家人商量，并仔细阅读条款，特别是退换政策。
