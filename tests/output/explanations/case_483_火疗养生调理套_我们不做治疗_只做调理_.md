# Case Analysis

**Input:** 火疗养生调理套。我们不做治疗，只做调理。零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持。一个疗程调理：高血压、糖尿病、风湿骨痛、妇科问题全面改善。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 1. 产品与宣传内容
- 产品名称：火疗养生调理套
- 核心宣传点：
  - 声称“不做治疗，只做调理”，但紧接着列出“一个疗程调理高血压、糖尿病、风湿骨痛、妇科问题全面改善”，这属于典型的**概念偷换**——用“调理”包装治疗功效，规避监管。
  - 推广“零风险创业：加盟火疗馆，月入十万不是梦”，并强调“总部全方位支持”，暗示高额回报。
  - 价格模糊：“市场价仅供参考，实际价格以门店为准”，为后续加价或隐藏费用留空间。

### 2. 风险信号识别
- **虚假医疗功效（高风险）**：宣称一个疗程可改善高血压、糖尿病等慢性病，这些疾病需要长期正规医疗管理，火疗作为物理疗法无科学依据。
- **非法传销/MLM模式（高风险）**：“零风险创业”、“月入十万”是典型的高回报诱惑，结合加盟模式，极可能涉及拉人头、收取加盟费、层级返利等传销特征。
- **免责声明陷阱**：先声明“不做治疗”，再列出治疗性效果，试图在法律责任上打擦边球。
- **模糊定价**：不明确最终价格，可能诱导消费者到店后高价消费。

### 3. 评分理由
- 风险等级：9（极高风险）
  - 同时包含虚假医疗功效、非法传销、模糊定价三大危险信号。
  - 针对老年人等易受骗群体，危害性大。
  - 无任何可信资质（如医疗器械注册号、知名品牌背书）。

### 4. 建议
- **消费者**：切勿购买或尝试该产品，更不要参与加盟。
- **老年人**：警惕以“调理”为名、实则治疗慢性病的宣传，以及承诺高回报的创业项目。
- **监管**：建议向市场监管部门举报该产品涉嫌虚假宣传和非法传销。
