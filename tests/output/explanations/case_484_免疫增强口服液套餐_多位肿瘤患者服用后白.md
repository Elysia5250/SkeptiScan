# Case Analysis

**Input:** 免疫增强口服液套餐。多位肿瘤患者服用后白细胞计数显著回升，部分案例见内部资料汇编。购买满3000元成为VIP会员，享受团队业绩分红。中国科学院微生物研究所（合作关系，非官方认可）提供技术支持。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细风险分析

### 1. 传销/非法集资风险（高）
- 明确提到“购买满3000元成为VIP会员，享受团队业绩分红”，这是典型的传销或非法集资特征。
- 团队业绩分红机制鼓励发展下线，而非基于产品本身价值，涉嫌违法。

### 2. 虚假医疗功效（高）
- 声称“多位肿瘤患者服用后白细胞计数显著回升”，但未提供任何可验证的临床数据或案例详情。
- “内部资料汇编”无法公开核实，属于典型的虚假宣传手段。

### 3. 虚假权威背书（中）
- 提及“中国科学院微生物研究所（合作关系，非官方认可）”，括号内说明“非官方认可”暗示实际关系薄弱，可能只是蹭名头。
- 没有具体合作项目或授权文件，消费者无法验证。

### 4. 价格欺诈风险（中）
- “市场价仅供参考，实际价格以门店为准”为模糊定价策略，可能随意加价或设置隐藏费用。

### 5. 综合评分
- 风险等级：9/10（极高风险）
- 主要风险点：传销模式、虚假医疗功效、模糊权威背书
- 建议：立即远离，并向市场监管部门举报。
