# Case Analysis

**Input:** 精准基因防癌筛查套餐。我们不是医疗机构，不提供诊断。与北京某三甲医院（退休医生个人合作）提供报告解读服务。但据统计，90%的早期癌症患者因未及时发现而错过最佳治疗期。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 7/10

**Verdict:** scam

---

## 详细分析

### 1. 风险信号识别
- **伪科学/虚假医疗宣称**：基因防癌筛查本身有一定科学依据，但宣传中‘90%早期癌症患者错过最佳治疗期’的数据来源不明，属于制造恐慌的营销手段。同时，明确表示‘不提供诊断’，却暗示筛查能发现早期癌症，存在矛盾。
- **模糊权威背书**：提及‘北京某三甲医院’但实际是‘退休医生个人合作’，无法验证合作真实性，且退休医生个人行为不代表医院官方，容易误导消费者。
- **制造紧迫感**：使用统计数据强调‘错过最佳治疗期’，诱导消费者立即下单，属于高压销售策略。

### 2. 营销模式分析
- **免责声明**：声明‘不是医疗机构，不提供诊断’和‘广告内容由品牌方提供’，试图规避法律责任，但宣传内容本身具有误导性。
- **模糊合作**：仅提及‘退休医生个人合作’，未提供医生姓名、资质或医院授权文件，可信度低。

### 3. 风险等级评估
综合以上分析，该商品存在虚假紧迫感、模糊权威背书和潜在医疗误导，风险等级为7（高风险）。建议消费者避免购买，选择正规医疗机构进行癌症筛查。
