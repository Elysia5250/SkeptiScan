# Case Analysis

**Input:** 野生林下参礼盒。不满意7天无理由退换，运费我们承担。买一盒送一盒，再送枸杞一袋！每天一根参，换你十年少。前100名下单再送价值299元的养生壶一台。

**Risk Level:** 5/10

**Verdict:** scam

---

该商品宣传语中“每天一根参，换你十年少”属于典型的夸大抗衰老功效，缺乏科学依据，容易误导消费者。虽然商家提供了7天无理由退换和赠品促销，但这些营销手段常见于高毛利或低质量产品，用以吸引冲动消费。野生林下参本身价格不透明，普通消费者难以辨别真伪，且未提供任何权威认证或检测信息。综合来看，该商品存在一定虚假宣传风险，建议消费者保持警惕，理性评估后再决定是否购买。
