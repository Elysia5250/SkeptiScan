# Case Analysis

**Input:** 精准基因防癌筛查套餐。采用第三代纳米孔测序技术（宣传用语，非实际采用），一次检测覆盖3000种癌症相关基因位点。我们不是医疗机构，不提供诊断。现在购买基因检测套餐，免费赠送价值1980元的量子波动检测仪一台！。

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 评分理由
该产品风险等级为8分（高度可疑），主要基于以下风险信号：

1. **虚假技术宣传**：明确标注“宣传用语，非实际采用”，说明商家承认其宣称的“第三代纳米孔测序技术”不实，属于故意误导消费者。

2. **夸大检测能力**：声称“覆盖3000种癌症相关基因位点”，但正规基因检测通常针对特定癌种或已知突变位点，如此广泛的覆盖缺乏科学依据，且可能造成消费者过度解读结果。

3. **伪科学赠品**：“量子波动检测仪”是典型的伪科学产品，量子力学与人体健康无直接关联，此类仪器常被用于诈骗。赠送该仪器进一步表明商家缺乏诚信。

4. **规避医疗责任**：声明“不是医疗机构，不提供诊断”，但产品名称和宣传却暗示具有防癌筛查功能，这种矛盾表述旨在逃避医疗监管，同时利用消费者对癌症的恐惧心理。

5. **健康焦虑营销**：以“防癌”为核心卖点，诱导消费者因恐惧而购买，属于典型的健康焦虑营销手段。

### 事实核查
- 经查，国家药监局（NMPA）未批准任何名为“量子波动检测仪”的医疗器械。
- 正规基因检测机构（如华大基因、贝瑞基因）的癌症筛查产品通常针对特定癌种（如乳腺癌、结直肠癌），且需在医疗机构指导下进行。
- 该产品未提供任何资质证明或合作医疗机构信息，无法验证其合法性。

### 最终建议
**强烈不建议购买**。该产品存在虚假宣传、伪科学成分和潜在欺诈风险。消费者应选择正规医疗机构进行癌症风险评估，切勿轻信此类夸大其词的“防癌套餐”。
