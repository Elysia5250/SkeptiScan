# Case Analysis

**Input:** 体验课低价引流-正式课高价。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。郑重声明：体验营无强制消费。报名体验营即送500元抵用券。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 产品/促销概述
该商品以“体验课低价引流”为切入点，后续推销高价“调理方案”，包括基础套餐19800元/年和尊享套餐39800元/年。同时，声明“体验营无强制消费”，并赠送500元抵用券以吸引报名。

### 风险评分：7/10（高风险）

### 主要风险信号

1.  **钓鱼式营销（Bait-and-switch pricing）**：这是最核心的风险。通过低价体验课吸引用户，然后推销价格高达数万元的调理方案。这种模式常见于保健品、健康管理、美容养生等领域，用户一旦进入体验营，可能面临持续的心理压力和推销话术，最终被迫或冲动消费。

2.  **高价套餐缺乏透明度**：基础套餐19800元/年、尊享套餐39800元/年，但未说明具体包含哪些服务、产品、疗程或资质。如此高的价格，用户无法判断其价值，极易被夸大宣传所误导。

3.  **“无强制消费”声明的双重性**：虽然声明本身是合规的，但在低价引流+高价套餐的背景下，这种声明可能是一种“让步免责声明”（concession disclaimer），旨在让用户放松警惕，实际上后续的推销可能非常强势。

4.  **赠送抵用券的诱导性**：500元抵用券看似优惠，但通常有使用门槛（如满一定金额、仅限特定套餐、不可与其他优惠叠加等），目的是降低用户报名体验课的心理门槛，从而进入后续的销售漏斗。

### 营销技巧分析
- **低价引流**：利用用户贪便宜心理，以极低价格（甚至免费）吸引报名。
- **高价转化**：在体验营中通过“专家”讲座、一对一咨询、案例分享等方式，逐步推销高价套餐。
- **紧迫感营造**：可能配合“限时优惠”、“名额有限”等话术，促使用户当场决策。
- **心理暗示**：通过赠送抵用券、声明无强制消费，营造“诚信”形象，降低用户防御。

### 事实核查建议
- **查证机构背景**：通过国家企业信用信息公示系统查询该机构的工商注册信息、经营范围、是否有行政处罚或消费投诉。
- **搜索用户评价**：在社交媒体、消费者维权平台（如黑猫投诉、12315）搜索该机构或类似套餐的反馈。
- **核实抵用券条款**：要求对方提供500元抵用券的完整使用规则，包括有效期、适用产品、是否可退款等。
- **了解调理方案的科学依据**：如果调理方案涉及健康或医疗效果，必须要求提供权威的临床研究证据、资质证书，并咨询专业医生。

### 购买建议
- **强烈不建议**在未充分了解机构资质、套餐内容、退费政策的情况下报名体验课或支付任何费用。
- 如果确实感兴趣，建议先以书面形式（如邮件、聊天记录）要求对方提供套餐的详细说明、服务合同样本、退费条款，并保留所有沟通记录。
- 对于任何声称能“调理”慢性病、改善健康的高价方案，务必先咨询正规医院的医生，避免被虚假宣传所骗。
