# Case Analysis

**Input:** 精准基因防癌筛查套餐。由某知名基因公司（已停止该业务）提供技术支持。检测发现高风险基因者，可搭配购买本中心定制防癌套餐（另收费29800元/年）。检测结果仅供参考。

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为“精准基因防癌筛查套餐”，声称由某知名基因公司提供技术支持，但该公司已停止该业务。检测后，若发现高风险基因，会推荐购买本中心定制的防癌套餐，年费高达29800元。检测结果仅作参考。

### 风险信号识别

1. **虚假背书（风险信号3）**：声称由某知名基因公司提供技术支持，但该公司已停止该业务。这属于利用已停业公司的名义增加可信度，属于虚假背书。

2. **钓鱼式营销（风险信号8）**：先以低价或免费检测吸引用户，然后推荐购买高额后续服务（29800元/年）。这是典型的“免费试用+后续收费”套路。

3. **免责声明（风险信号7）**：检测结果“仅供参考”，这属于让步免责声明，暗示检测结果可能不准确，但同时又引导用户购买后续服务，存在矛盾。

4. **健康焦虑营销（风险信号4）**：利用用户对癌症的恐惧，通过“高风险基因”的检测结果诱导用户购买高价防癌套餐。

5. **概念偷换（风险信号6）**：基因检测本身是科学，但将其与“防癌套餐”直接挂钩，缺乏科学依据。防癌套餐的具体内容未说明，可能包含无效或有害的保健品。

### 评分依据
- **风险等级8**：属于高风险诈骗。
  - 虚假背书（+2分）
  - 钓鱼式营销（+2分）
  - 健康焦虑营销（+2分）
  - 免责声明（+1分）
  - 概念偷换（+1分）

### 建议
- **不购买**：该产品存在明显的欺诈风险。
- **核实信息**：如需基因检测，请选择国家卫健委认证的医疗机构。
- **警惕后续收费**：任何检测后推荐的高价服务都应高度警惕。

### 最终结论
该产品利用已停业公司的名义进行虚假背书，通过基因检测诱导用户购买高额防癌套餐，属于典型的钓鱼式诈骗。建议消费者远离此类产品，并举报相关宣传材料。
