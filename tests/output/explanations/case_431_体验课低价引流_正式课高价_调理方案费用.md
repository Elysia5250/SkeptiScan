# Case Analysis

**Input:** 体验课低价引流-正式课高价。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。郑重声明：体验营无强制消费。报名体验营即送500元抵用券。

**Risk Level:** 7/10

**Verdict:** scam

---

## 详细分析

### 1. 营销模式分析
该商品采用典型的“低价引流-高价转化”模式，先以体验课吸引用户，再推销价格高昂的调理方案（19800元/年或39800元/年）。这种模式常见于健康、养生、教育等领域，容易导致消费者在体验后因心理压力或话术诱导而冲动消费。

### 2. 风险信号识别
- **价格异常**：年费高达数万元，远超普通健康服务市场价，缺乏透明定价依据。
- **抵用券策略**：赠送500元抵用券看似优惠，实为降低消费者决策门槛，增加转化率。
- **声明无强制消费**：此类声明往往是为了规避法律风险，但实际销售过程中可能通过情感施压、限时优惠等方式变相强制。

### 3. 潜在危害
- 消费者可能因体验课效果不明确而支付高额费用，最终发现服务与宣传不符。
- 若调理方案涉及健康或医疗内容，缺乏资质可能导致健康风险。
- 高额费用可能造成经济负担，尤其是对老年人或信息不对称群体。

### 4. 建议
- 消费者应要求提供详细的调理方案说明、科学依据和成功案例（需可验证）。
- 查询该公司是否具备相关行业资质，如健康管理、医疗咨询等许可。
- 搜索网络评价，特别是是否有消费者投诉或法律纠纷。
- 避免在体验课现场冲动付款，给自己至少24小时冷静期。
