# Case Analysis

**Input:** 野生林下参礼盒。限量500份，售完即止！已售473份，仅剩27份！。买一盒送一盒，再送枸杞一袋！每天一根参，换你十年少。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 详细分析

### 1. 产品真实性存疑
- 宣称“生长年限15年以上”，但括号内注明“实际林下参5-8年”，存在明显矛盾，涉嫌虚假标注年限。
- 产地“长白山野生林下参”缺乏具体证明文件，如地理标志或检测报告。

### 2. 营销手法高风险
- **制造紧迫感**：限量500份、已售473份、仅剩27份，是典型的“饥饿营销”手段，诱导消费者冲动下单。
- **买赠促销**：买一盒送一盒再送枸杞，看似优惠，但可能掩盖产品实际价值或质量问题。
- **夸大功效**：“每天一根参，换你十年少”暗示抗衰老、延年益寿，属于虚假健康宣传，违反广告法。

### 3. 价格不透明
- “市场价仅供参考，实际价格以门店为准”模糊了定价依据，消费者无法判断真实价值，易被高价低质产品欺骗。

### 4. 风险等级评估
- 综合以上问题，风险等级为7（高风险），主要风险点包括虚假宣传、价格欺诈和营销误导。
- 建议消费者避免购买，尤其是老年人群体，因其更容易被夸大功效和紧迫感营销所影响。
