# Case Analysis

**Input:** 南极磷虾油胶囊。本品为保健食品，保健功能：辅助降血脂。个体数据存在差异，我们如实陈述研究结果，不做疗效保证。中国科学院南海海洋研究所（联合实验室合作项目）参与研发。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 5/10

**Verdict:** scam

## 风险分析报告

### 产品概述
南极磷虾油胶囊，宣称保健功能为辅助降血脂，并提及中国科学院南海海洋研究所参与研发。产品明确声明'不做疗效保证'，但暗示研究结果支持其功效。

### 风险信号识别
1. **免责声明陷阱**：'不做疗效保证'看似合规，实则是一种常见的营销话术——先声明不保证疗效，再通过'研究结果'暗示有效，让消费者自行联想。这种'概念替换'手法在保健品中屡见不鲜。
2. **模糊权威背书**：'中国科学院南海海洋研究所（联合实验室合作项目）参与研发'，未说明具体合作形式、是否提供数据支持或仅挂名。消费者容易误以为有国家级科研机构背书，但实际可能只是名义参与。
3. **缺乏具体证据**：未提供任何临床试验数据、用户案例或第三方检测报告，仅以'研究结果'笼统带过，信息不透明。

### 风险等级评估
综合以上分析，该产品存在中等风险（5/10）。主要风险点在于营销话术的误导性，而非直接虚假宣传。产品本身有保健食品批文，但'不做疗效保证'的表述和模糊背书增加了消费者的判断难度。

### 购买建议
- 对于一般消费者：建议优先选择有明确临床证据、品牌信誉好的降血脂产品，并咨询医生。
- 对于老年人：警惕'不做疗效保证'背后的暗示，不要将其理解为'有效但不敢说'，保健食品不能替代药物治疗。
- 购买前务必核实产品批准文号，并通过官方渠道查询备案信息。
