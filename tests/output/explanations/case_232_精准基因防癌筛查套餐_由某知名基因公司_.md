# Case Analysis

**Risk Level:** 2/10

**Verdict:** not_scam

## 详细分析

### 1. 伪科学包装（0分）
未发现量子、磁疗、排毒等典型伪科学术语。

### 2. 虚假医疗声称（10分）
- **明确疾病治疗声称（+10分）**：商品声称检测发现高风险基因者需搭配购买防癌套餐，暗示该套餐可预防或治疗癌症，属于明确的疾病治疗暗示。

### 3. 虚假权威背书（5分）
- **不可验证的专家/机构背书（+5分）**：声称由某知名基因公司提供技术支持，但该公司已停止该业务，无法验证当前技术支持的可靠性，存在误导嫌疑。

### 4. 违法/传销特征（0分）
未发现多级返利、拉人头等传销特征。

### 5. 诱导欺诈营销（3分）
- **紧迫/稀缺压力（+3分）**：暗示高风险者需立即购买防癌套餐，否则可能错过治疗机会，制造焦虑。

### 6. 数据欺诈（0分）
未发现引用不相关研究或动物实验数据。

### 7. 可信信号（0分）
未发现药品注册号、知名品牌正常营销或纯信息描述。

### 综合评估
该商品利用基因检测结果制造健康焦虑，诱导消费者购买高价防癌套餐，且技术支持方已停止业务，存在虚假宣传和诱导消费风险。建议消费者保持警惕，避免冲动消费。
