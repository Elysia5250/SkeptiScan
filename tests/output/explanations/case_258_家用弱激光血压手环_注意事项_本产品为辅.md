# Case Analysis

**Input:** 家用弱激光血压手环。注意事项：本产品为辅助保健用具，并非医疗器械，不能替代降压药物治疗。建议每日佩戴4-6小时，持续使用3个月后由医生评估血压控制情况。采用腕式激光照射技术，激光波长650nm，输出功率<1mW（符合Class I激光安全标准）。个体效果存在差异，请理性看待。

**Risk Level:** 1/10

**Verdict:** not_scam

## 详细分析

### 1. 伪科学包装（+5分）
产品声称采用“弱激光照射技术”辅助控制血压，但激光照射用于降压缺乏主流医学认可的科学依据。虽然产品声明非医疗器械，但“辅助保健”的表述仍隐含健康功效，属于伪科学包装中的“其他伪科学术语”。

### 2. 虚假医疗声称（0分）
产品明确声明“并非医疗器械，不能替代降压药物治疗”，未直接或间接声称治疗疾病，因此未触发此项。

### 3. 虚假权威背书（0分）
未提及任何专家、媒体或认证背书。

### 4. 违法/传销特征（0分）
无相关特征。

### 5. 诱导欺诈营销（0分）
无紧迫性、免费陷阱或针对特定人群的营销手段。

### 6. 数据欺诈（0分）
未引用任何研究数据。

### 7. 可信信号（-2分）
产品声明非医疗器械、不能替代药物，并强调个体差异，属于正常描述性语言，未夸大功效。

### 综合评估
该产品风险较低，主要风险在于伪科学包装（弱激光降压缺乏科学依据）。但产品声明清晰，未进行虚假医疗宣传，整体可信度较高。建议消费者理性看待，咨询医生后再决定是否使用。
