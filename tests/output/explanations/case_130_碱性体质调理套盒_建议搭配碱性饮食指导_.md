# Case Analysis

**Input:** 碱性体质调理套盒。建议搭配碱性饮食指导（另购）效果更佳。限时赠送pH试纸一盒（100条），免费检测自身体质！。本产品依据「酸性体质是万病之源」理论，采用天然矿物质提取液，pH值8.5-9.0，每日三次每次10ml可快速纠正酸性体质。

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品宣称通过饮用碱性液体（pH 8.5-9.0）来“纠正酸性体质”，并搭配碱性饮食指导（另购）和赠送pH试纸。其核心理论是“酸性体质是万病之源”。

### 风险信号识别

1. **伪科学理论（高风险）**：
   - “酸性体质是万病之源”是已被医学界广泛否定的伪科学概念。人体血液pH值严格维持在7.35-7.45之间，饮食或碱性水无法显著改变这一平衡。该理论常被用于推销碱性水、保健品等，属于典型的健康焦虑营销。
   - 产品声称“快速纠正酸性体质”，暗示用户存在“酸性体质”问题，但医学上并无此诊断标准。

2. **虚假或夸大功效（高风险）**：
   - 产品未提供任何临床试验数据或权威机构认证，仅凭“天然矿物质提取液”和pH值宣传功效。
   - 声称“每日三次每次10ml可快速纠正”，但无任何科学依据支持。

3. **营销陷阱**：
   - **限时赠送pH试纸**：制造紧迫感，诱导用户立即购买。pH试纸本身廉价，但作为“免费检测”工具，进一步强化用户对“酸性体质”的担忧。
   - **建议搭配碱性饮食指导（另购）**：捆绑销售，增加后续消费。
   - **免费检测自身体质**：暗示用户需要干预，利用健康焦虑。

4. **缺乏合法资质**：
   - 产品未标注医疗器械或药品批号，极可能属于普通食品或饮料，却宣传治疗性功效，违反《广告法》和《食品安全法》。

### 风险等级评定
- **8分（高风险）**：基于伪科学理论，虚假功效宣传，利用健康焦虑营销，且无合法资质。符合“7-8分”标准中的“使用伪科学概念、疾病治疗承诺”。

### 事实核查建议
- 搜索“酸性体质 辟谣”可找到大量权威医学机构（如中国科协、丁香医生）的辟谣文章。
- 查询国家药监局或市场监管局网站，确认该产品是否有注册信息（大概率无）。
- 搜索品牌名称+“虚假宣传”查看是否有处罚记录。

### 最终建议
**强烈不推荐购买**。该产品是典型的伪科学健康骗局，不仅浪费金钱，还可能延误真正疾病的治疗。请勿相信任何声称能“纠正酸性体质”的产品。
