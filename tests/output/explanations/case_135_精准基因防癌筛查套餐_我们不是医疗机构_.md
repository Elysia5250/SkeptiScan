# Case Analysis

**Input:** 精准基因防癌筛查套餐。我们不是医疗机构，不提供诊断。与北京某三甲医院（退休医生个人合作）提供报告解读服务。但据统计，90%的早期癌症患者因未及时发现而错过最佳治疗期。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 风险评分：8/10（高度可疑，接近诈骗）

### 主要风险信号

1. **虚假/模糊的权威背书**：
   - 声称“与北京某三甲医院（退休医生个人合作）”，但未提供医院名称、医生姓名、执业编号。这种模糊表述常见于伪科学产品，利用“三甲医院”的声誉但无法核实。
   - 退休医生个人合作不等于医院认可，且无法确认该医生是否具备基因筛查解读资质。

2. **健康焦虑营销**：
   - 引用“90%的早期癌症患者因未及时发现而错过最佳治疗期”这一数据，但未提供任何来源。此类无出处数据旨在制造恐慌，促使消费者冲动购买。

3. **让步免责声明**：
   - 明确声明“我们不是医疗机构，不提供诊断”，但随后又暗示筛查能发现癌症风险。这是一种典型的“免责+暗示”手法，既规避法律风险，又让消费者误以为产品有医疗价值。
   - 真正的医疗筛查产品会明确其用途和局限性，不会用这种矛盾表述。

4. **概念偷换**：
   - “精准基因防癌筛查”听起来像高科技，但未说明具体检测哪些基因、准确率如何、是否经过临床验证。许多商业基因检测仅提供风险概率，并非诊断，但广告暗示能“防癌”。

5. **平台免责声明**：
   - “本广告内容由品牌方提供，不代表本平台立场”进一步表明平台不背书，消费者需自行承担风险。

### 事实核查建议

- 查询国家药监局（NMPA）数据库，看该产品是否作为医疗器械注册。
- 搜索“北京某三甲医院 退休医生 基因筛查”看是否有公开报道或投诉。
- 检查该产品是否有同行评审的临床研究支持其筛查准确性。

### 结论

该产品利用癌症恐惧心理，通过模糊权威背书和免责声明，诱导消费者购买未经证实的基因筛查服务。风险极高，建议避免购买。正规癌症筛查应通过医院或认证体检中心进行。
