# Case Analysis

**Input:** 免疫增强口服液套餐。多位肿瘤患者服用后白细胞计数显著回升，部分案例见内部资料汇编。本产品采用独家「仿野生」灵芝孢子粉+虫草多糖复合配方，通过调节树突状细胞活性增强机体免疫功能。中国科学院微生物研究所（合作关系，非官方认可）提供技术支持。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 4/10

**Verdict:** not_scam

---

## 详细分析

### 1. 伪科学包装（5分）
- **触发项**：产品使用“仿野生”灵芝孢子粉等术语，该术语在科学上无明确定义，属于营销话术。

### 2. 虚假医疗声称（15分）
- **触发项**：明确声称“多位肿瘤患者服用后白细胞计数显著回升”，暗示对肿瘤治疗有效，属于疾病治疗声称。
- **触发项**：虽未直接说“替代化疗”，但“免疫增强”和“白细胞回升”暗示可辅助或替代医疗，属于隐性医疗替代。

### 3. 虚假权威背书（10分）
- **触发项**：提及“中国科学院微生物研究所”合作，但明确标注“非官方认可”，这种模糊表述可能误导消费者认为有官方背书。
- **触发项**：未提供任何国家药品或医疗器械注册号，也未提供专利号或认证信息。

### 4. 数据欺诈（5分）
- **触发项**：声称“部分案例见内部资料汇编”，但未提供公开可查的研究数据，属于引用非公开、不可验证的研究作为证据。

### 5. 其他风险
- 产品未提供任何可信信号（如药品注册号、知名品牌等），因此无负分项。
- 未发现传销或诱导欺诈营销特征。

### 结论
该产品存在较高的风险，主要涉及虚假医疗声称和权威背书误导。建议消费者避免购买，并优先选择有国家认证的正规保健品或药品。
