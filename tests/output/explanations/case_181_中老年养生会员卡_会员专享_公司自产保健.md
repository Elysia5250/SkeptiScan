# Case Analysis

**Input:** 中老年养生会员卡。会员专享：公司自产保健品全场8折、优先预约国内专家（退休副主任医师视频问诊5分钟）、定期旅游。本卡一经售出，概不退换。每月15日会员日，全场买二送一！仅限会员参加。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品为“中老年养生会员卡”，主要面向中老年消费者，承诺提供保健品折扣、退休副主任医师视频问诊、定期旅游等会员权益。

### 风险信号识别

1. **模糊的专家服务**：
   - 宣称“优先预约国内专家（退休副主任医师视频问诊5分钟）”，但未提供任何专家姓名、原单位、职称证明。
   - “退休副主任医师”可能为低级别或虚构身份，且5分钟问诊时间极短，难以提供有效医疗建议。
   - 此类承诺常见于保健品销售套路，利用老年人对权威的信任诱导消费。

2. **自产保健品折扣**：
   - “公司自产保健品”暗示产品来源单一，可能缺乏第三方质量监管。
   - 保健品非药品，但常被夸大功效，此处未说明具体产品，存在后续推销风险。

3. **霸王条款**：
   - “本卡一经售出，概不退换”违反《消费者权益保护法》中关于预付卡退换的规定，属于不公平格式条款。
   - 消费者一旦购买，即使服务不满意也无法退款，风险极高。

4. **价格不透明**：
   - “市场价仅供参考，实际价格以门店为准”为典型的价格欺诈手法，商家可随意调整价格，使折扣失去意义。

5. **促销诱导**：
   - “每月会员日，全场买二送一”鼓励囤货，可能导致消费者购买过量保健品，造成浪费或健康风险。
   - “仅限会员参加”制造稀缺感，促使冲动决策。

### 风险评估
- **风险等级：6/10**（可疑）
- 该产品不属于明显诈骗，但存在多个高风险信号：模糊的专家资质、霸王条款、价格不透明、促销诱导。
- 主要风险在于：消费者可能支付高额会员费后，获得的专家服务名不副实，保健品质量无保障，且无法退款。

### 建议
- 购买前务必要求商家提供专家名单、资质证明，并核实公司营业执照和保健品生产许可证。
- 仔细阅读合同，拒绝“概不退换”条款，要求明确退换政策。
- 不要被折扣和旅游承诺冲昏头脑，优先选择信誉良好的大型连锁药店或医疗机构。
- 老年人应在家属陪同下决策，避免冲动消费。
