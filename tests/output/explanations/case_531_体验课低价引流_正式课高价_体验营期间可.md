# Case Analysis

**Input:** 体验课低价引流-正式课高价。体验营期间可免费体验：针灸、拔罐、艾灸各一次。但名额有限，仅限60岁以上人士报名。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析

### 1. 营销模式分析
该商品采用典型的“低价引流-高价转化”模式，通过免费体验针灸、拔罐、艾灸等传统疗法吸引目标人群，然后推销高价调理套餐。这种模式本身不一定违法，但结合针对60岁以上老年人的定向营销，风险显著增加。

### 2. 目标人群风险
明确限制“仅限60岁以上人士报名”，表明商家有意针对老年人群体。老年人通常对健康问题更关注，且可能缺乏足够的辨别能力，容易受到营销话术影响。

### 3. 价格不透明
“市场价仅供参考，实际价格以门店为准”这一表述暗示价格可能随意调整，存在隐藏费用或临时加价的风险。高价套餐（19800元/年、39800元/年）缺乏明确的服务内容和效果说明，容易导致消费者支付后得不到相应价值。

### 4. 缺乏科学依据
调理方案未提供任何科学依据或临床数据，仅以“针灸、拔罐、艾灸”等传统疗法为卖点。虽然这些疗法有一定历史，但作为高价套餐的核心内容，需要更严谨的疗效证据。

### 5. 综合评分
综合以上因素，该商品存在明显的营销陷阱和针对弱势群体的风险，风险等级为7分（高风险）。建议消费者尤其是老年人群体保持警惕，避免冲动消费。
