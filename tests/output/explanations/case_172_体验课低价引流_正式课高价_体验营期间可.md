# Case Analysis

**Input:** 体验课低价引流-正式课高价。体验营期间可免费体验：针灸、拔罐、艾灸各一次。但名额有限，仅限60岁以上人士报名。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：7/10（高度可疑）

### 主要风险信号

1. **后续收费（钓鱼）**：以“免费体验”为诱饵，吸引老年人参与，随后推销高价套餐（基础套餐19800元/年，尊享套餐39800元/年）。这是典型的“低价引流-高价收费”模式，消费者在体验后可能因心理压力或健康焦虑而被迫购买高价服务。

2. **针对特定人群**：明确限制“仅限60岁以上人士报名”，利用老年人对健康的关注和易受营销影响的特点，增加转化率。

3. **信息不透明**：未说明调理方案的具体内容、科学依据或资质证明。仅给出价格，缺乏服务细节，消费者无法判断其价值。

4. **制造稀缺性**：“名额有限”营造紧迫感，促使消费者快速决策，减少理性思考时间。

### 营销技巧分析

- **低价引流**：免费体验是常见的获客手段，但后续收费远超市场合理范围。
- **年龄限制**：看似“专属福利”，实则是精准筛选易受骗群体。
- **价格模糊**：“市场价仅供参考，实际价格以门店为准”为后续加价留有余地。

### 建议

- 消费者应要求机构提供详细的调理方案、资质证明及费用明细。
- 查询该机构是否在卫生健康部门备案，以及是否有消费者投诉记录。
- 建议老年人在子女或专业人士陪同下进行决策，避免冲动消费。

### 结论

该推广模式具有明显的消费陷阱特征，风险较高。建议消费者保持警惕，避免因免费体验而陷入高价消费的圈套。
