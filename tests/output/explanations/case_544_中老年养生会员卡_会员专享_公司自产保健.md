# Case Analysis

**Input:** 中老年养生会员卡。会员专享：公司自产保健品全场8折、优先预约国内专家（退休副主任医师视频问诊5分钟）、定期旅游。本卡一经售出，概不退换。每月15日会员日，全场买二送一！仅限会员参加。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 商品概述
该商品为“中老年养生会员卡”，主要面向老年人销售，宣称提供保健品折扣、专家视频问诊和旅游活动等会员权益。

### 风险信号识别

1. **概不退换条款（高风险）**：
   - 明确标注“一经售出，概不退换”，违反《消费者权益保护法》关于七日无理由退货的规定，属于霸王条款。
   - 老年人一旦购买，即使发现产品问题或服务不符，也无法退款，权益严重受损。

2. **价格不透明（中高风险）**：
   - “市场价仅供参考，实际价格以门店为准”意味着商家可以随意定价，存在虚标原价再打折的嫌疑。
   - 保健品本身利润空间大，8折可能并不优惠，甚至比市场价更高。

3. **专家资质存疑（中风险）**：
   - “退休副主任医师视频问诊5分钟”中，专家身份无法核实，退休医师可能已无执业资格，且5分钟问诊时间过短，难以提供有效医疗建议。
   - 可能只是普通医生或健康顾问冒充专家。

4. **促销诱导（中风险）**：
   - “买二送一”鼓励囤货，老年人可能因优惠而购买大量保健品，造成浪费或过期。
   - 会员日限时活动制造紧迫感，促使冲动消费。

5. **旅游活动（低风险但需警惕）**：
   - 定期旅游可能为“会销”形式，在旅游过程中推销高价保健品或医疗器械，属于常见营销手段。

### 综合评分
- **风险等级：7/10**（高风险）
- 主要风险点：霸王条款、价格欺诈、专家资质不明、诱导消费。
- 该商品不具备合法医疗器械或药品资质，且营销模式类似“会销”，对老年人具有较大欺骗性。

### 购买建议
**强烈不建议购买**。建议老年人通过正规渠道（如医院、大型药店）购买保健品，并注意查看产品批准文号（国食健字或蓝帽子）。如遇类似会员卡推销，可向当地市场监管部门举报。
