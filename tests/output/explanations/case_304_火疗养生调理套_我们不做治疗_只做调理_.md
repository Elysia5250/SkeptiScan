# Case Analysis

**Input:** 火疗养生调理套。我们不做治疗，只做调理。零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持。一个疗程调理：高血压、糖尿病、风湿骨痛、妇科问题全面改善。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 4/10

**Verdict:** not_scam

## 详细分析

### 1. 伪科学包装（5分）
- 火疗属于传统养生概念，但将其与疾病调理直接挂钩，缺乏现代科学验证。文案中未提供任何科学原理或临床数据，属于伪科学包装。

### 2. 虚假医疗声称（15分）
- 明确列出“高血压、糖尿病、风湿骨痛、妇科问题”等具体疾病，并声称“一个疗程调理……全面改善”，这属于明确的疾病治疗暗示。虽然文案开头说“不做治疗，只做调理”，但后续内容直接与疾病挂钩，构成虚假医疗声称。

### 3. 非法/传销特征（8分）
- “零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持”是典型的传销话术，以高额回报为诱饵，鼓励发展下线加盟，符合多级返利特征。

### 4. 其他风险
- 价格模糊：“市场价仅供参考，实际价格以门店为准”为后续加价或欺诈留下空间。
- 未提供任何权威认证、专利或专家背书，可信度极低。

### 结论
该产品综合风险极高，建议消费者远离，并向市场监管部门举报。
