# Case Analysis

**Input:** 火疗养生调理套。已有超过10万人受益。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。传承千年中医火疗秘法，以特制药酒配合火焰燃烧技术，从足三里到大椎穴全线疏通。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 1. 风险信号识别
- **伪科学概念**：商品使用“排毒”这一伪科学术语，声称火疗能打开毛孔排毒通道，但医学上不存在通过皮肤排毒的科学机制。灵芝孢子粉“激活自愈力”也是模糊的健康承诺，缺乏临床证据。
- **虚假医疗暗示**：宣传“从足三里到大椎穴全线疏通”，暗示能治疗经络相关疾病，属于未经批准的医疗宣称。
- **夸大受益人数**：“已有超过10万人受益”无法验证，是典型的从众效应营销手段。
- **价格不透明**：标注“市场价仅供参考，实际价格以门店为准”，可能涉及价格欺诈或诱导消费。

### 2. 营销手段分析
- **权威包装**：借用“千年中医火疗秘法”等传统医学词汇，但未提供具体出处或认证。
- **组合销售**：将火疗和灵芝孢子粉捆绑销售，增加客单价，且两者均缺乏有效证据。
- **模糊定价**：通过“以门店为准”规避价格监管，可能在不同门店随意定价。

### 3. 风险等级评估（8/10）
- 该商品涉及多项高风险信号：伪科学、虚假医疗宣称、夸大宣传、价格不透明。
- 缺乏任何权威认证（如国药准字、国械注准、保健食品标志），且火疗本身存在烫伤等安全隐患。
- 针对老年人等易受骗群体，风险极高。

### 4. 建议
- 消费者应避免购买此类产品，尤其是老年人。
- 如需养生调理，应选择正规医疗机构或具有国家批准文号的保健食品。
- 监管部门应关注此类虚假宣传，防止消费者上当受骗。
