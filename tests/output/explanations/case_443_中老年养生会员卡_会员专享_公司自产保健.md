# Case Analysis

**Input:** 中老年养生会员卡。会员专享：公司自产保健品全场8折、优先预约国内专家（退休副主任医师视频问诊5分钟）、定期旅游。本卡一经售出，概不退换。每月15日会员日，全场买二送一！仅限会员参加。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

## 详细分析

### 1. 商品概述
该商品为“中老年养生会员卡”，主要面向中老年群体，宣称提供保健品折扣、专家视频问诊和定期旅游等会员权益。

### 2. 风险信号识别
- **虚假医疗/健康承诺**：声称“优先预约国内专家（退休副主任医师视频问诊5分钟）”，但未明确专家资质和问诊内容，可能存在夸大或虚假宣传。
- **营销陷阱**：使用“全场8折”、“买二送一”等促销手段，但“市场价仅供参考，实际价格以门店为准”导致价格不透明，易产生误导。
- **霸王条款**：“本卡一经售出，概不退换”违反消费者权益保护法，属于不公平格式条款。
- **潜在传销/非法集资**：虽未明确提及返利，但会员卡模式可能涉及预付费风险，需警惕公司跑路或资金链断裂。

### 3. 风险等级评估
综合以上风险，该商品存在价格不透明、霸王条款、虚假专家承诺等问题，风险等级为6（中等偏高），建议消费者谨慎对待。

### 4. 购买建议
- 购买前要求商家提供营业执照、保健品批号等资质证明。
- 核实专家问诊服务的真实性，可要求提供专家姓名和执业证书编号。
- 避免一次性大额充值，选择可退换或按次付费的产品。
- 保留所有付款凭证和宣传材料，以便维权。
