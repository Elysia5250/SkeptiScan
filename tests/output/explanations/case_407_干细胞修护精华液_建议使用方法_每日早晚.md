# Case Analysis

**Input:** 干细胞修护精华液。建议使用方法：每日早晚各一次，涂抹于清洁后的面部皮肤。实际效果因个人肤质而异。实验室数据显示（体外细胞实验，非人体），该成分可延长成纤维细胞传代次数约30%。

**Risk Level:** 5/10

**Verdict:** scam

---

### 详细分析

**1. 产品宣称与证据链**
- 产品名称为“干细胞修护精华液”，直接关联“干细胞”这一生物医学前沿概念，容易让消费者联想到再生、修复等强大功效。
- 提供的唯一功效证据是“实验室数据显示（体外细胞实验，非人体），该成分可延长成纤维细胞传代次数约30%”。体外细胞实验与人体皮肤实际吸收、代谢环境差异巨大，不能直接等同于人体抗衰老效果。

**2. 风险信号识别**
- **伪科学嫌疑**：干细胞护肤品在化妆品领域存在争议，因为活细胞难以在常温护肤品中保持活性，且化妆品法规通常不允许直接添加活细胞。产品未说明干细胞来源（植物？动物？人源？）及活性保存技术，存在夸大宣传风险。
- **虚假医疗暗示**：虽未直接宣称治疗疾病，但“修护”“延长传代次数”等表述易被解读为抗衰老、修复皮肤损伤，可能误导消费者认为具有医疗级效果。
- **数据欺诈风险**：仅提供体外实验数据，且未披露实验具体条件、重复性、统计学显著性等，属于典型的“实验室数据误导”。

**3. 营销手法**
- 利用“干细胞”这一高科技词汇制造差异化，吸引追求前沿护肤的消费者。
- 以“实验室数据”作为权威背书，但刻意省略“非人体”这一关键限制，属于选择性信息披露。

**4. 综合风险评估**
- 风险等级5（中等风险）：产品未发现直接违法宣称（如治疗疾病），但存在明显的证据不足和概念炒作，消费者容易被误导。
- 建议消费者在购买前查询国家药监局化妆品备案信息，确认产品是否合规。对于宣称“干细胞”的护肤品，应保持高度警惕，优先选择有明确人体功效测试报告的品牌。
