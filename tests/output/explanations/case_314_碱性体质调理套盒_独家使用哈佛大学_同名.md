# Case Analysis

**Input:** 碱性体质调理套盒。独家使用哈佛大学（同名同姓研究员）论文数据作为科研依据。建议搭配碱性饮食指导（另购）效果更佳。《人体酸碱平衡论》创始人亲传弟子团队研发（创始人已去世）。本广告内容由品牌方提供，不代表本平台立场。

**Verdict:** not_scam

## 详细分析

### 1. 伪科学包装（10分）
- **Detox / meridian / microcirculation pseudo-claims (+5)**：产品核心概念‘碱性体质调理’直接源于已被科学界广泛否定的‘酸碱体质理论’。该理论创始人Robert O. Young已因欺诈被判刑，其理论无任何科学依据。
- **Other pseudoscience terms not covered above (+5)**：‘碱性体质’本身是伪科学术语，无生物学或医学基础。

### 2. 虚假权威背书（10分）
- **Unverifiable 'academician'/'expert team' endorsement (+5)**：声称由‘《人体酸碱平衡论》创始人亲传弟子团队研发’，但创始人已去世，团队身份无法通过公开渠道验证，属于虚假背书。
- **Unverifiable 'CCTV'/'state media' recommendation (+5)**：虽未直接提及媒体，但‘哈佛大学（同名同姓研究员）论文数据’暗示权威机构背书，实际利用同名同姓误导消费者，构成虚假权威背书。

### 3. 数据欺诈（5分）
- **Citing non-applicable research as own evidence (+5)**：引用哈佛大学同名同姓研究员的论文数据，但该论文并非针对本产品，属于数据欺诈。

### 4. 诱导欺诈营销（3分）
- **Bait-and-switch (free → paid) (+3)**：建议搭配‘碱性饮食指导（另购）’，暗示需额外付费才能达到效果，属于诱导消费。

### 5. 可信信号（0分）
- 无任何可信信号（如药品注册号、知名品牌等）。

### 总结
该产品综合风险较高，主要涉及伪科学包装、虚假权威背书、数据欺诈和诱导营销。建议消费者避免购买，并警惕类似‘碱性体质’等伪科学概念。
