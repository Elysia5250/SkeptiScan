# Case Analysis

**Input:** 体验课低价引流-正式课高价。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。郑重声明：体验营无强制消费。报名体验营即送500元抵用券。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 1. 营销模式分析
该商品采用典型的“低价引流-高价转化”模式，通过体验课吸引用户，再推销高价正式调理方案。这种模式本身不违法，但容易导致消费者冲动消费，尤其是对价格敏感或缺乏经验的用户。

### 2. 价格风险
基础套餐19800元/年，尊享套餐39800元/年，价格较高，且未说明具体服务内容。高价可能对应虚高的价值，用户需警惕是否物有所值。

### 3. 承诺与免责声明
“郑重声明：体验营无强制消费”看似保障用户权益，但实际可能通过心理营销、限时优惠等手段诱导消费，而非强制。这种声明可能掩盖真实的营销意图。

### 4. 优惠陷阱
“报名即送500元抵用券”是常见的营销手段，抵用券可能有限制条件（如仅限高价套餐、不可退款等），用户需仔细阅读条款。

### 5. 综合风险评估
该商品存在较高的营销风险，尤其是针对老年人或缺乏辨别能力的消费者。建议用户保持警惕，避免因低价引流而陷入高价消费陷阱。
