# Case Analysis

**Input:** 火疗养生调理套。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。我们不做治疗，只做调理。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。

**Risk Level:** 4/10

**Verdict:** not_scam

---

## 详细分析

### 1. 伪科学包装（15分）
- **排毒声称**：产品明确提到‘火疗打开毛孔排毒通道’，‘排毒’是典型的伪科学概念，人体有自身代谢系统，无需通过火疗‘排毒’。
- **自愈力激活**：‘灵芝孢子粉激活自愈力’中的‘自愈力’属于模糊的伪科学术语，无科学定义。
- **道医养生**：‘融合道医养生精华’使用传统文化包装，但无具体科学依据。

### 2. 虚假医疗声称（10分）
- **间接治疗暗示**：虽然声称‘不做治疗，只做调理’，但‘排毒’‘激活自愈力’等表述暗示对疾病有调理作用，可能诱导消费者放弃正规医疗。
- **未明确疾病名称**：但整体语境指向慢性病或亚健康人群，属于隐性医疗声称。

### 3. 虚假权威背书（5分）
- **已注销机构**：‘权氏中医研究院（已注销）’的背书无法验证，且机构已注销，权威性为零。

### 4. 其他维度
- 未发现传销、诱导营销或数据欺诈证据。
- 无可信信号（无注册号、非知名品牌）。

### 综合建议
该产品风险较高，主要集中于伪科学和虚假医疗声称。消费者应警惕此类‘调理’话术，避免因延误治疗造成健康损害。
