# Case Analysis

**Input:** 精准基因防癌筛查套餐。采用第三代纳米孔测序技术（宣传用语，非实际采用），一次检测覆盖3000种癌症相关基因位点。我们不是医疗机构，不提供诊断。现在购买基因检测套餐，免费赠送价值1980元的量子波动检测仪一台！。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 1. 产品概述
该产品宣称提供“精准基因防癌筛查套餐”，使用“第三代纳米孔测序技术”，覆盖3000种癌症相关基因位点，并附赠“量子波动检测仪”。

### 2. 风险信号识别
- **伪科学概念**：量子波动检测仪是典型的伪科学产品，量子力学与波动检测无任何关联，此类赠品常被用于欺诈。
- **虚假技术宣传**：文字中明确标注“宣传用语，非实际采用”，表明商家承认技术宣传不实，属于故意误导。
- **夸大检测范围**：声称覆盖3000种癌症相关基因位点，但未提供任何权威机构认证或临床验证，普通基因检测通常针对特定基因，如此广泛的覆盖缺乏科学依据。
- **免责声明陷阱**：声称“不是医疗机构，不提供诊断”，试图规避法律责任，但实际销售的是医疗相关服务，存在法律风险。
- **免费赠品诱导**：赠送高价值量子仪器，利用消费者贪便宜心理，掩盖产品本身的虚假性。

### 3. 风险等级评估
综合以上信号，该产品涉及伪科学、虚假宣传、欺诈性营销，风险等级为9（极高风险），属于典型骗局。

### 4. 购买建议
绝对不要购买。基因检测应选择正规医院或有资质的第三方检测机构（如华大基因、金域医学等），并确认产品具有国家药监局批准的医疗器械注册证。任何附带量子、纳米等伪科学赠品的检测服务均不可信。
