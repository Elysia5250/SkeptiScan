# Case Analysis

**Input:** 精准基因防癌筛查套餐。采用第三代纳米孔测序技术（宣传用语，非实际采用），一次检测覆盖3000种癌症相关基因位点。我们不是医疗机构，不提供诊断。现在购买基因检测套餐，免费赠送价值1980元的量子波动检测仪一台！。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 风险评分：8/10（高度可疑，接近诈骗）

### 主要风险信号

1. **虚假技术宣传**：声称采用“第三代纳米孔测序技术”，但明确标注为“宣传用语，非实际采用”。这是典型的偷换概念，利用前沿科技名词误导消费者，实际可能使用低成本、低准确度的检测方法。

2. **夸大检测范围**：“覆盖3000种癌症相关基因位点”缺乏科学依据。目前权威的癌症基因检测通常针对特定癌种或已知致病突变，而非泛泛覆盖数千个位点。这种说法容易让消费者误以为能全面预防癌症。

3. **伪科学赠品**：赠送“量子波动检测仪”是典型的伪科学营销。量子波动检测仪已被多次揭露为无任何医疗或检测功能的骗局产品，其宣称的“量子波动”概念在医学上毫无根据。

4. **责任规避声明**：明确表示“不是医疗机构，不提供诊断”，这属于让步免责声明。一方面暗示检测有健康价值，另一方面又否认医疗责任，使消费者在出现误诊或延误治疗时无法追责。

5. **健康焦虑营销**：利用“防癌”这一敏感话题，诱导消费者因恐惧而购买，尤其是对健康高度关注的老年人。

### 事实核查建议

- 查询该检测是否由具备资质的医学检验所或医院提供，是否获得NMPA批准。
- 搜索“量子波动检测仪”相关新闻，可发现大量揭露其为骗局的报道。
- 联系当地卫健委或市场监管部门核实该产品是否合法。

### 最终建议

**不要购买**。该产品综合了虚假技术、伪科学赠品和规避责任声明，属于高风险消费陷阱。如需进行基因检测，应选择正规三甲医院或国家认证的基因检测机构（如华大基因、贝瑞基因等），并咨询医生建议。切勿因免费赠品而落入圈套。
