# Case Analysis

**Input:** 野生林下参礼盒。限量500份，售完即止！已售473份，仅剩27份！。买一盒送一盒，再送枸杞一袋！每天一根参，换你十年少。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 5/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品为“野生林下参礼盒”，宣传来自长白山，生长年限15年以上，并搭配限量销售、买赠促销等营销手段。

### 风险信号识别

1. **虚假/模糊年限标注**：宣传“生长年限15年以上”，但括号内注明“实际林下参5-8年”，存在明显矛盾。林下参的生长年限通常由参农或商家自行标注，缺乏第三方权威认证，消费者难以核实。

2. **夸大健康功效**：广告语“每天一根参，换你十年少”暗示抗衰老、延年益寿，属于典型的健康功效夸大。人参作为食品或保健食品，不能宣传治疗或抗衰老效果，此类表述违反《广告法》。

3. **紧迫感营销**：“限量500份，已售473份，仅剩27份”制造稀缺感，诱导冲动消费。实际库存可能为虚构，常见于促销套路。

4. **模糊定价**：“市场价仅供参考，实际价格以门店为准”为商家预留了价格调整空间，可能实际售价远高于宣传价。

5. **买赠促销**：买一盒送一盒再送枸杞，属于常见促销手段，本身不违法，但结合其他风险信号，可能用于掩盖产品真实价值。

### 风险等级评估
综合以上分析，该商品存在虚假宣传、夸大功效、营销套路等风险，但未涉及医疗欺诈或投资骗局，因此风险等级为**5分（可疑）**。

### 购买建议
- 不建议轻信“15年”年限，实际可能为5-8年。
- 不要因“限量”而冲动下单，可先搜索商家口碑。
- 人参属于药食同源，并非人人适合，尤其老年人、高血压患者等需谨慎。
- 建议选择有正规检测报告、明确产地和年限证明的商家。

### 总结
该商品营销手法较为典型，存在虚假年限和夸大功效问题，但尚未达到诈骗级别。消费者应保持警惕，理性购买。
