# Case Analysis

**Input:** 野生林下参礼盒。限量500份，售完即止！已售473份，仅剩27份！。买一盒送一盒，再送枸杞一袋！每天一根参，换你十年少。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 详细分析

### 1. 产品概述
该商品为“野生林下参礼盒”，宣称产自长白山，生长年限15年以上，但实际林下参通常为5-8年，存在年限虚假标注。

### 2. 风险信号识别
- **虚假宣传**：声称“每天一根参，换你十年少”，暗示抗衰老功效，但人参作为普通食品或保健食品，不能宣传治疗或抗衰老效果，违反广告法。
- **营销欺诈**：使用“限量500份，已售473份，仅剩27份”制造紧迫感，但库存数据无法验证，常见于虚假促销。
- **价格误导**：标注“市场价仅供参考，实际价格以门店为准”，可能虚高原价再打折，诱导消费者。
- **赠品诱惑**：买一送一再加赠品，分散消费者对产品本身的注意力。

### 3. 风险等级评估
综合以上，风险等级为7（高风险），属于虚假宣传和营销欺诈。

### 4. 购买建议
不建议购买。消费者应警惕此类夸大宣传和紧迫感营销，选择正规渠道购买有明确产地和年限证明的人参产品。
