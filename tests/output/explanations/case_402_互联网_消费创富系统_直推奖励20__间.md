# Case Analysis

**Input:** 互联网+消费创富系统。直推奖励20%，间推奖励10%，三代奖励5%。月入过万不是梦：推荐10人，团队裂变至100人，月被动收入超5万元！已有来自全国32个省市超过50万人参与，累计分配利润超过20亿元。平台资质：已取得ICP备案（正在办理中）、与某国有银行（仅签署意向协议）达成资金存管合作。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 1. 核心风险：多级分销传销
- 宣传中明确列出“直推奖励20%，间推奖励10%，三代奖励5%”，这是典型的多级分销（MLM）特征，且层级超过三级，涉嫌违法传销。
- 承诺“月入过万”“月被动收入超5万元”等暴富话术，利用人性贪婪诱导参与。

### 2. 虚假或夸大宣传
- “已有来自全国32个省市超过50万人参与，累计分配利润超过20亿元”——无任何第三方审计或公开数据支持，极可能是编造的数字。
- “平台资质：已取得ICP备案（正在办理中）”——ICP备案要么已取得要么未取得，“正在办理”是常见拖延话术，实际可能根本没有提交申请。
- “与某国有银行（仅签署意向协议）达成资金存管合作”——意向协议不具备法律约束力，且未披露具体银行名称，无法核实。

### 3. 营销手段分析
- 利用“互联网+”“消费创富”等概念包装，掩盖传销本质。
- 强调“团队裂变”“被动收入”，制造轻松赚钱的幻觉。
- 模糊处理资质和合作信息，降低用户警惕。

### 4. 对老年人的特别风险
- 老年人更容易被“月入过万”“被动收入”等话术吸引，且对ICP备案、银行存管等概念缺乏辨别能力。
- 一旦投入资金，可能血本无归，且难以追回。

### 5. 综合判断
- 风险等级：9（极高风险）。该平台几乎具备所有传销和资金盘特征，建议立即远离并向市场监管部门举报。
