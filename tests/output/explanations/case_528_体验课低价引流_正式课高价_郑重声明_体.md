# Case Analysis

**Input:** 体验课低价引流-正式课高价。郑重声明：体验营无强制消费。体验营期间可免费体验：针灸、拔罐、艾灸各一次。报名体验营即送500元抵用券。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

## 风险分析

### 1. 营销模式识别
该商品采用典型的“低价引流-高价转化”模式。通过免费体验（针灸、拔罐、艾灸）和500元抵用券吸引消费者到店，但未明确正式课程的价格和内容。这种模式常见于美容、养生、培训等行业，容易导致消费者在体验后被诱导购买高价服务。

### 2. 风险信号
- **价格不透明**：声明“市场价仅供参考，实际价格以门店为准”，为后续高价收费留下空间。
- **抵用券陷阱**：500元抵用券可能附带苛刻使用条件（如满额使用、仅限特定项目），实际价值存疑。
- **强制消费声明**：虽然声称“无强制消费”，但体验过程中可能通过话术或环境压力诱导消费。

### 3. 风险等级评估
综合以上因素，风险等级为6（中等偏高）。虽然不涉及伪科学或虚假医疗，但营销手段具有欺骗性，容易导致消费者产生额外支出。

### 4. 建议
- 消费者应提前通过电话或在线渠道确认体验课的具体内容、时长、是否有额外费用。
- 对抵用券的使用规则进行书面确认。
- 避免在体验过程中冲动购买，可要求提供正式合同和价格清单。
