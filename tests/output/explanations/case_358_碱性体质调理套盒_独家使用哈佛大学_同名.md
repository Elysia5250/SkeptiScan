# Case Analysis

**Input:** 碱性体质调理套盒。独家使用哈佛大学（同名同姓研究员）论文数据作为科研依据。建议搭配碱性饮食指导（另购）效果更佳。本产品依据「酸性体质是万病之源」理论，采用天然矿物质提取液，pH值8.5-9.0，每日三次每次10ml可快速纠正酸性体质。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 8/10

**Verdict:** scam

---

该产品宣传存在多项高风险信号：首先，声称独家使用哈佛大学论文数据，但注明“同名同姓研究员”，暗示可能利用名字混淆视听，缺乏真实科研背书。其次，“酸性体质是万病之源”是已被主流医学否定的伪科学理论，产品据此宣称能“快速纠正酸性体质”，属于虚假医疗功效宣传。此外，建议搭配另购的碱性饮食指导，属于捆绑销售和诱导消费。价格信息模糊，以“市场价仅供参考，实际价格以门店为准”规避明码标价义务。整体来看，产品缺乏任何正规药品或保健食品批准文号，风险极高，尤其容易误导老年人。
