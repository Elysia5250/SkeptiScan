# Case Analysis

**Input:** 免费领智能血糖仪。活动说明：血糖仪免费送，耗材有偿提供。免费领！原价598元的智能语音血糖仪（市场同类产品实际价格50-80元），现在0元领取！只需支付耗材费299元（含50条试纸+采血针），后续试纸需从我司购买（单价6.9元/条，市场价2元/条以下）。立即拨打电话400-XXX-XXXX免费领取！说朋友介绍再送维生素C一瓶！。个体效果存在差异，请理性看待。

**Risk Level:** 7/10

**Verdict:** scam

---

该活动以“免费领智能血糖仪”为噱头，实际要求支付299元耗材费（含50条试纸和采血针），而市场同类血糖仪价格仅为50-80元，试纸市场价低于2元/条，活动后续试纸单价6.9元/条，明显偏高。活动未提供医疗器械注册证号，无法验证产品合法性。虚高原价（598元）制造优惠假象，朋友介绍奖励机制可能涉及传销风险。整体属于典型的高价耗材锁定模式，对消费者尤其是老年人具有较大欺骗性。
