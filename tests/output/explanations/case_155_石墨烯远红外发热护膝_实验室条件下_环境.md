# Case Analysis

**Input:** 石墨烯远红外发热护膝。实验室条件下（环境温度25℃，无风环境），测试显示可提高局部皮肤温度3-5℃。石墨烯发热技术也应用于航天、军事等高精尖领域（本产品所用材料非同等技术等级）。据用户反馈统计（在线问卷，样本量128人），约68%的使用者表示膝盖不适感有所缓解。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 5/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为石墨烯远红外发热护膝，主要宣传点包括：实验室条件下的升温效果、石墨烯技术的背景、以及用户反馈的缓解效果。

### 风险信号识别

1. **概念偷换（高风险）**：
   - 宣传中提到“石墨烯发热技术也应用于航天、军事等高精尖领域”，但立即声明“本产品所用材料非同等技术等级”。这是一种典型的偷换概念手法：利用高端技术的光环提升产品形象，同时用免责声明规避责任。消费者容易忽略后半句，误以为产品与航天级技术相同。

2. **虚假/误导性数据**：
   - “68%的使用者表示膝盖不适感有所缓解”基于128人的在线问卷，样本量小且无对照组，无法排除安慰剂效应或选择偏差。这种数据不能作为科学证据，属于营销话术。

3. **实验室条件与实际效果脱节**：
   - 实验室条件（25℃、无风）下的升温效果（3-5℃）在实际使用中可能大打折扣，因为人体活动、衣物、环境温度等都会影响。宣传未说明实际使用效果，容易误导消费者。

4. **免责声明**：
   - “本广告内容由品牌方提供，不代表本平台立场”是典型的平台免责声明，但消费者可能仍会信任广告内容。品牌方未提供任何第三方验证。

### 评分依据
- **风险等级5**：产品本身可能无害（护膝），但宣传存在夸大和误导，属于“可疑”范畴。未发现明确的虚假医疗承诺（如“治疗关节炎”），但概念偷换和用户反馈数据不严谨，足以引起警惕。
- 未达到7分（伪科学）是因为没有使用“量子”“磁疗”等明显伪科学术语，也没有直接宣称治疗疾病。

### 建议
- 消费者应理性看待“石墨烯”“远红外”等概念，实际效果可能有限。
- 建议选择有医疗器械注册证（如“械字号”）的产品，或咨询医生。
- 对于老年人，需特别提醒不要被“高科技”和“用户好评”迷惑，避免花冤枉钱。
