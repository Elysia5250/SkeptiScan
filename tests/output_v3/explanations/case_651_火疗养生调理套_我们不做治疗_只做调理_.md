# Case Analysis

**Input:** 火疗养生调理套。我们不做治疗，只做调理。零风险创业：加盟火疗馆，月入十万不是梦！总部全方位支持。一个疗程调理：高血压、糖尿病、风湿骨痛、妇科问题全面改善。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 9/10

**Verdict:** scam

## 风险分析报告

### 评分依据
该产品宣传内容存在多项严重风险信号，综合评定风险等级为9（高度欺诈）。

### 主要风险信号
1. **虚假疗效承诺**：明确宣称一个疗程可改善高血压、糖尿病、风湿骨痛、妇科问题。这些均为慢性病或复杂疾病，火疗作为非医疗手段，不可能产生如此广泛的治疗效果。根据《广告法》，非药品不得宣传疾病治疗功效。
2. **创业骗局特征**：'零风险创业'、'月入十万不是梦'是典型的加盟骗局话术，利用高回报吸引投资者，实际可能涉及传销或资金盘。
3. **规避监管话术**：'不做治疗，只做调理'试图规避医疗监管，但后续又列出疾病名称，实质仍是暗示治疗功效，属于打擦边球的欺诈行为。
4. **健康焦虑营销**：同时列出多种常见慢性病和妇科问题，利用消费者对健康的焦虑，诱导其购买或加盟。

### 事实核查
- 火疗本身属于中医外治法的一种，但未经严格临床验证，且操作不当有烧伤风险。国家中医药管理局曾多次提醒公众警惕火疗骗局。
- 高血压、糖尿病等疾病需要长期规范治疗，任何声称'一个疗程改善'的产品均不可信。
- 加盟类项目承诺固定高回报的，绝大多数是骗局，商务部备案的特许经营企业极少做出此类承诺。

### 最终建议
**不要购买该产品，不要加盟该项目。** 消费者应通过正规医疗机构治疗疾病，投资者应选择合法合规的创业项目。如已受骗，请立即向当地市场监管部门或公安机关举报。
