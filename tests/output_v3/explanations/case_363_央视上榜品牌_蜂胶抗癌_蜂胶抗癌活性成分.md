# Case Analysis

**Input:** 央视上榜品牌【蜂胶抗癌】蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，限量100套赶快下单！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：9/10（高度疑似诈骗）

### 主要风险信号

1. **虚假医疗功效宣称**：
   - 声称“蜂胶抗癌活性成分是化疗药的10倍”——这是极其危险的虚假比较，没有任何科学依据。化疗药物是经过严格临床试验的处方药，而蜂胶作为食品/保健品，其抗癌作用在人体中从未被证实。
   - “抑制肿瘤扩散”、“多名患者康复”——这些是明确的疾病治疗承诺，属于违法宣传。根据《广告法》，非药品不得宣传疾病治疗功效。

2. **伪权威背书**：
   - “央视上榜品牌”——这是一个常见的虚假营销话术。央视从未设立过“上榜品牌”认证，许多商家通过付费或伪造方式使用此标签。消费者无法验证其真实性。

3. **紧迫感营销**：
   - “限量100套赶快下单”——利用稀缺性制造焦虑，迫使消费者在不理性状态下做出购买决定。

### 科学事实核查

- 蜂胶是蜜蜂从植物中采集的树脂混合物，含有多种生物活性成分（如黄酮类、酚酸类）。体外实验和动物实验确实显示某些成分有抗肿瘤潜力，但**没有任何高质量的人体临床试验证明蜂胶可以治疗癌症**。
- 美国国家癌症研究所（NCI）和世界卫生组织（WHO）均未将蜂胶列为抗癌药物。
- 中国国家药品监督管理局（NMPA）批准的蜂胶产品均为“保健食品”或“普通食品”，功能仅限于“增强免疫力”等，绝无“抗癌”功效。

### 法律风险

- 违反《中华人民共和国广告法》第17条：除医疗、药品、医疗器械广告外，禁止其他任何广告涉及疾病治疗功能。
- 违反《食品安全法》第73条：食品广告不得含有虚假内容，不得涉及疾病预防、治疗功能。

### 最终建议

**绝对不要购买**。该产品是典型的“保健品诈骗”，利用癌症患者的求生欲望和健康焦虑进行欺诈。如果患者听信宣传而放弃正规治疗，后果不堪设想。请立即向当地市场监管部门举报（12315）。
