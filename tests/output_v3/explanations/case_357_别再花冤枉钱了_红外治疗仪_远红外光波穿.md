# Case Analysis

**Input:** 别再花冤枉钱了！红外治疗仪，远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，最后一天！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 评分理由
该产品宣传语包含多个高风险信号，综合评定风险等级为8（高风险）。

### 风险信号识别
1. **虚假医疗功效宣称**：声称“治好颈椎病”，属于对疾病治疗效果的绝对化承诺，违反《广告法》第十七条（除医疗、药品、医疗器械广告外，禁止其他任何广告涉及疾病治疗功能）。
2. **伪科学概念**：“远红外光波穿透皮下15cm”缺乏科学依据。根据生物医学工程研究，远红外线（波长3-1000μm）在人体组织中的穿透深度通常仅为数毫米至1-2厘米，15cm的穿透深度在物理学上不可能。
3. **制造紧迫感**：“最后一天”是典型的促销话术，用于诱导消费者冲动购买。
4. **暗示替代医疗**：“不吃药不打针”暗示该产品可替代正规医疗，可能误导患者放弃有效治疗。

### 来源核查
- 国家药监局（NMPA）数据库：未查询到名为“红外治疗仪”的通用产品注册信息（需具体品牌型号进一步核实）。
- 科学文献：远红外线的生物效应主要是热效应，其穿透深度受波长和组织类型影响，但均远小于15cm。
- 消费者投诉：类似产品在电商平台常因虚假宣传被投诉。

### 结论与建议
该产品宣传内容高度可疑，属于典型的“伪医疗器械”营销。建议消费者：
- 不要购买，避免经济损失。
- 颈椎病应到正规医院骨科或康复科就诊。
- 如已购买，可向当地市场监督管理局投诉举报。
