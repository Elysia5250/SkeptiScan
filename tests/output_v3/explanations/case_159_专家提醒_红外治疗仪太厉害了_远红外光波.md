# Case Analysis

**Input:** 专家提醒：红外治疗仪太厉害了！远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，买一送一！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：8（高风险，接近诈骗）

### 主要风险信号

1. **虚假疗效承诺**：
   - 宣称“治好颈椎病”，属于对慢性疾病的根治性承诺，违反《广告法》中关于非药品不得宣传疾病治疗功效的规定。
   - “消炎止痛”是明确的医疗术语，普通家用电器或保健器械不得使用。

2. **伪科学包装**：
   - “远红外光波穿透皮下15cm”严重夸大。根据生物医学工程研究，远红外线（波长4-14μm）在人体组织中的穿透深度通常仅为0.5-5mm，15cm完全违背物理常识。
   - 远红外理疗有一定辅助作用，但绝不能替代医学治疗。

3. **营销手段**：
   - 使用“太厉害”、“不吃药不打针”等煽动性语言，诱导消费者放弃正规医疗。
   - “买一送一”制造限时优惠假象，降低消费者警惕性。

### 事实核查

- 未能在国家药监局数据库中查到任何名为“红外治疗仪”且宣称可治疗颈椎病的合法医疗器械。
- 医学共识：颈椎病需根据类型（神经根型、脊髓型等）进行药物、物理或手术治疗，不存在单一仪器可“治好”所有类型。

### 最终建议

该产品极可能为三无或虚假宣传产品，存在严重健康风险（延误治疗）和经济损失风险。请勿购买，并建议向市场监管部门举报。
