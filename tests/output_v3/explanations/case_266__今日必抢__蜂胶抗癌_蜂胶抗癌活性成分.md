# Case Analysis

**Input:** 🔥今日必抢🔥 蜂胶抗癌，蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，无效退款！

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 风险评分：9/10（明显欺诈）

### 风险信号识别

1. **虚假疗效宣称**：
   - 声称“蜂胶抗癌活性成分是化疗药的10倍”——这是极其夸张且无科学依据的对比。化疗药物经过严格的临床试验和审批，而蜂胶作为保健品，从未被证明具有抗癌疗效。
   - “抑制肿瘤扩散”——直接宣称对肿瘤有治疗作用，违反《广告法》和《食品安全法》关于保健品不得宣传疾病治疗功效的规定。
   - “多名患者康复”——使用未经证实的个案暗示疗效，属于典型的虚假宣传。

2. **营销手段**：
   - “今日必抢”制造紧迫感，诱导冲动消费。
   - “无效退款”看似保障，实则难以兑现，且是欺诈性产品的常见话术。

3. **缺乏权威背书**：
   - 未提供任何临床试验数据、药品批准文号或权威机构认证。

### 事实核查

- 根据国家市场监督管理总局规定，蜂胶属于保健食品，只能声称“增强免疫力”等保健功能，不得宣称疾病预防或治疗。
- 在PubMed等医学数据库中，没有高质量临床研究支持蜂胶能抗癌或优于化疗。
- 中国食品药品检定研究院未批准任何蜂胶抗癌药物。

### 结论

该产品属于典型的“保健品冒充药品”欺诈，利用消费者对癌症的恐惧和对天然产品的盲目信任。强烈建议不要购买，并向市场监管部门举报。
