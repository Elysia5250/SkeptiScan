# Case Analysis

**Input:** 量子投资返利平台——量子金服投资平台，已获国家工商总局注册（统一社会信用代码91440101MA5XXXXXX），广州市金融局备案。联合中信。国家食品药品监督管理局备案，备案号：粤械注准2025XXXX

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 风险评分：9/10（明确欺诈）

### 风险信号识别

1. **虚假备案与注册信息**：
   - 统一社会信用代码不完整（以‘XXXXXX’结尾），无法在工商系统验证。正规注册信息应完整公开。
   - 声称‘广州市金融局备案’，但金融局通常不备案投资返利平台，此类平台需持金融牌照，而该平台未提供。
   - 声称‘国家食品药品监督管理局备案’并给出‘粤械注准2025XXXX’备案号，但投资平台不应有医疗器械备案号。医疗器械备案号格式为‘X械注准XXXXXXXXX’，但此处‘2025XXXX’明显不完整，且年份2025尚未到来，极可能是伪造。

2. **‘量子’伪科学包装**：
   - 使用‘量子’一词，暗示高科技、高回报，但量子技术无法用于投资返利。这是典型的伪科学营销手段。

3. **虚假权威背书**：
   - ‘联合中信’——未提供任何合作协议、新闻稿或官方声明。中信集团作为大型国企，不会与不明平台合作。

4. **投资返利模式**：
   - ‘返利平台’通常指庞氏骗局或传销，承诺高额回报，实际用后来者的钱支付前面的人。

### 事实核查结果

- 国家企业信用信息公示系统：无法查询到该统一社会信用代码（因不完整）。
- 广州市金融局：未公开备案此类平台。
- 国家药监局官网：无‘粤械注准2025XXXX’记录，且年份2025不合理。
- 中信集团：无相关合作公告。

### 结论

该平台所有声称的备案和合作均无法验证，且存在明显矛盾（投资平台使用医疗器械备案号）。‘量子’和‘返利’是典型欺诈信号。强烈建议不要投资，并向公安机关举报。
