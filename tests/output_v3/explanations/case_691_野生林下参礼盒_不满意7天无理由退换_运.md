# Case Analysis

**Input:** 野生林下参礼盒。不满意7天无理由退换，运费我们承担。买一盒送一盒，再送枸杞一袋！每天一根参，换你十年少。前100名下单再送价值299元的养生壶一台。

**Risk Level:** 4/10

**Verdict:** not_scam

---

## 风险分析报告

### 商品概述
该商品为“野生林下参礼盒”，属于农产品或保健品范畴。宣传文案中包含了大量促销信息（买一送一、赠枸杞、赠养生壶）以及一句健康暗示语：“每天一根参，换你十年少”。

### 风险信号识别
1. **虚假/夸大功效暗示**："每天一根参，换你十年少" 属于典型的抗衰老暗示，但人参（包括林下参）并无科学证据支持其具有“十年少”的效果。这种表述容易误导消费者，尤其是老年人，认为长期食用可以逆转衰老。
2. **过度促销手段**：多级赠品（买一送一、再送枸杞、前100名送养生壶）制造了“限时优惠”和“稀缺性”的假象，属于常见的营销套路，但本身不违法。
3. **退换货承诺**："7天无理由退换，运费我们承担" 是合法承诺，但可能用于降低消费者警惕性。

### 风险等级评估
- **1-3分（安全）**：如果产品是正规品牌、真实林下参，且仅使用正常促销语言，则风险低。但此处使用了健康暗示语，超出普通营销范畴。
- **4-6分（可疑）**：该商品未直接宣称治疗疾病，但“十年少”属于未经证实的功效暗示，且促销力度过大，存在夸大宣传嫌疑。因此评为4分。
- **7-10分（诈骗）**：未达到，因为没有伪科学概念（如量子、磁疗）或虚假权威背书。

### 建议
- 消费者应理性看待人参的保健作用，不要期待“返老还童”。
- 购买前核实商家资质，查看产品是否有正规检测报告。
- 老年人尤其要警惕此类“养生”话术，避免过度消费。

### 结论
该商品属于**可疑**级别，建议谨慎购买，不要被赠品和夸张宣传冲昏头脑。
