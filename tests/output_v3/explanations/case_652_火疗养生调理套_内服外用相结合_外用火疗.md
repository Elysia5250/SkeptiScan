# Case Analysis

**Input:** 火疗养生调理套。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。已有超过10万人受益。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。个体效果存在差异，请理性看待。

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 评分依据
该产品风险等级为**8分（高风险）**，属于明显欺诈倾向的伪科学产品。

### 风险信号识别
1. **伪科学包装**：
   - '火疗打开毛孔排毒通道'：排毒概念在医学上无依据，毛孔不负责排毒，火疗可能造成烫伤。
   - '激活自愈力'：模糊的伪科学术语，无具体机制或证据。
   - '道医养生精华'：利用传统文化包装，缺乏现代科学验证。

2. **虚假权威背书**：
   - '权氏中医研究院（已注销）'：使用已注销机构增加可信度，是常见欺诈手段。
   - '三十年研发'：无法验证，且与已注销机构矛盾。

3. **夸大宣传**：
   - '已有超过10万人受益'：无法核实，属于典型的从众效应营销。
   - '内服外用相结合'：暗示综合疗法，但无任何临床数据支持。

4. **健康风险**：
   - 火疗本身存在烧伤、感染风险，尤其对老年人或皮肤敏感者。
   - 灵芝孢子粉作为普通食品，不能声称治疗或激活自愈力。

### 事实核查结果
- 未找到权氏中医研究院的合法注册信息，其已注销状态表明可能已因违规被取缔。
- 灵芝孢子粉在正规保健食品中仅可标注'增强免疫力'，不能声称'激活自愈力'。
- 火疗不属于国家认可的中医诊疗项目，多个地区已禁止开展。

### 最终建议
该产品属于典型的伪科学保健品欺诈，利用老年人对中医和排毒的信任进行虚假宣传。建议消费者远离此类产品，如有健康问题应咨询正规医疗机构。监管部门应关注此类已注销机构继续开展宣传的行为。
