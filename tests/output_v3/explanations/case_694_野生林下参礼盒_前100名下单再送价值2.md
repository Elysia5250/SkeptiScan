# Case Analysis

**Input:** 野生林下参礼盒。前100名下单再送价值299元的养生壶一台。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。产地直销，无中间商赚差价！只要198元！原价1280元的林下参礼盒，中秋特惠仅需198元。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

### 风险分析报告

**商品名称**：野生林下参礼盒

**风险等级**：6（可疑）

**主要风险信号**：

1. **虚假年限宣传**：商品宣称“生长年限15年以上”，但括号内注明“实际林下参5-8年”，存在明显的年限虚标。林下参的生长年限直接影响其药用价值和价格，15年参和5年参价格相差数倍。这种前后矛盾的说法属于虚假宣传，误导消费者。

2. **价格欺诈嫌疑**：原价1280元，现价198元，折扣高达84%。这种大幅降价通常是先虚标原价再打折的促销手段。根据《价格法》，原价应为商品在本次促销活动前7日内有交易记录的最低价格，但此处原价很可能从未实际销售过。

3. **限时赠品营销**：“前100名下单送价值299元的养生壶”制造稀缺感和紧迫感，促使消费者冲动下单。赠品价值通常被夸大，实际成本可能很低。

4. **产地直销话术**：“产地直销，无中间商赚差价”是常见营销话术，无法验证。实际上，即使产地直销，商家仍需盈利，价格未必比市场低。

**事实核查建议**：
- 查询长白山地区林下参的市场行情：15年林下参市场价通常在500-1000元/支，而5-8年参价格在100-300元/支。198元买15年参几乎不可能。
- 检查商家是否具备药品或食品经营许可证，以及参产品的检验报告。
- 核实养生壶赠品的品牌和市场价格，避免被虚高价值误导。

**购买建议**：
- 不建议购买此商品，尤其是用于送礼或保健目的。年限虚标可能导致实际效果远低于预期。
- 如需购买林下参，请选择知名品牌或大型药店，并要求提供年限检测报告。
- 老年人应警惕此类“原价上千、现价几百”的促销，避免因贪图便宜而购买到劣质产品。

**总结**：该商品存在虚假宣传和价格欺诈嫌疑，风险等级为6，属于可疑商品。消费者应保持警惕，避免冲动消费。
