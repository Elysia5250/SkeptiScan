# Case Analysis

**Input:** 火疗养生调理套。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。我们不做治疗，只做调理。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 风险评分：8/10（高度可疑，接近诈骗）

### 主要风险信号

1. **虚假权威背书**：产品声称由“权氏中医研究院（已注销）”创始人研发。机构已注销意味着其合法性或存续状态有问题，使用已注销机构名称作为背书，属于典型的虚假权威营销。

2. **伪科学概念**：
   - **火疗排毒**：火疗是一种民间疗法，无科学证据支持其能“打开毛孔排毒通道”。人体排毒主要通过肝脏、肾脏完成，皮肤排毒概念是伪科学。
   - **灵芝孢子粉激活自愈力**：灵芝孢子粉虽有一定免疫调节作用，但“激活自愈力”是夸大宣传，缺乏针对该产品的临床证据。

3. **规避监管话术**：声称“不做治疗，只做调理”，试图规避医疗器械或药品监管，但实际宣传内容（排毒、激活自愈力）已涉及健康疗效暗示。

4. **营销策略**：强调“三十年研发”和“道医养生精华”，利用传统医学的模糊概念增加神秘感，诱导消费者。

### 事实核查结果

- 未找到权氏中医研究院的公开注册信息或权威媒体报道，机构已注销进一步削弱其可信度。
- 火疗在正规医学中未被认可为排毒疗法，且存在烫伤、感染等风险。
- 灵芝孢子粉作为保健食品，不能声称治疗或激活自愈力，违反广告法。

### 结论

该产品属于高风险伪科学产品，利用已注销机构背书和伪科学概念进行营销，可能对消费者健康造成危害（如火疗烫伤），且存在虚假宣传。建议消费者远离，并向市场监管部门举报。
