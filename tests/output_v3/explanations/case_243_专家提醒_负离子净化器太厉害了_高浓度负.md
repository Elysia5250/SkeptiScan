# Case Analysis

**Input:** 专家提醒：负离子净化器太厉害了！高浓度负离子直达肺部，杀死癌细胞，根治哮喘支气管炎，买一送一！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 总体评分：9/10（高度疑似诈骗）

### 风险信号识别

1. **虚假疗效宣称**：明确声称“杀死癌细胞”、“根治哮喘支气管炎”，这是典型的未经批准的医疗宣称。根据中国《广告法》和《医疗器械监督管理条例》，非药品或非医疗器械产品不得宣传疾病治疗功效。

2. **伪科学包装**：使用“负离子”这一伪科学概念，声称能“直达肺部”并治疗癌症和呼吸系统疾病。负离子在医学上并无治疗癌症或根治哮喘的证据，此类宣传常见于伪健康产品。

3. **虚假权威背书**：开头“专家提醒”未指明具体专家或机构，属于捏造权威以增加可信度。

4. **健康焦虑营销**：通过“太厉害了”等夸张表述，诱导消费者因恐惧疾病而购买。

5. **促销陷阱**：“买一送一”是常见的促销手段，用于降低消费者警惕性，加速决策。

### 事实核查

- 世界卫生组织（WHO）及中国国家卫生健康委员会均未认可负离子作为癌症或哮喘的治疗手段。
- 中国国家药品监督管理局（NMPA）数据库中未查到任何负离子净化器具有治疗癌症或哮喘的注册信息。
- 类似“负离子治疗”已被多个科普平台（如果壳网、丁香医生）列为伪科学。

### 结论

该产品宣传内容严重违反中国广告法规，属于典型的虚假医疗广告，极有可能为诈骗产品。消费者应避免购买，并建议向市场监管部门举报。
