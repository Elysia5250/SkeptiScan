# Case Analysis

**Input:** 野生林下参礼盒。限量500份，售完即止！已售473份，仅剩27份！。买一盒送一盒，再送枸杞一袋！每天一根参，换你十年少。长白山野生林下参，生长年限15年以上（实际林下参5-8年）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

## 风险分析报告

### 商品概述
该商品为“野生林下参礼盒”，宣传来自长白山，生长年限15年以上，但实际林下参通常为5-8年。采用限量500份、已售473份、仅剩27份的紧迫感营销，并搭配买一送一及赠品促销。

### 风险信号识别

1. **虚假/夸大功效宣传**：
   - “每天一根参，换你十年少”属于典型的抗衰老功效暗示，但林下参作为普通农产品（非药品或保健食品），不得宣传任何医疗或保健功效。该口号违反《广告法》关于非药品不得宣传疗效的规定。

2. **年限虚标**：
   - 宣传“生长年限15年以上”，但括号内注明“实际林下参5-8年”，属于故意误导消费者。年限直接影响产品价值，虚标行为涉嫌欺诈。

3. **限量营销手法**：
   - “限量500份，已售473份，仅剩27份”是常见的制造稀缺性手段，但无法验证库存真实性。此类手法常被用于诱导消费者快速下单，避免理性比较。

4. **价格模糊**：
   - “市场价仅供参考，实际价格以门店为准”为商家预留了价格调整空间，可能使消费者在购买后发现价格与宣传不符。

### 风险等级评估

综合以上因素，该商品存在明显的虚假宣传（年限虚标）、夸大功效（抗衰老暗示）以及营销诱导（限量、赠品）。但尚未发现明确的医疗欺诈或投资骗局，因此风险等级定为**6级（可疑）**。

### 建议
- 消费者应核实商家资质，要求提供年限检测报告。
- 不要轻信“换你十年少”等无科学依据的承诺。
- 如购买，保留好聊天记录、宣传截图和发票，以便维权。
- 老年人尤其需警惕，人参并非人人适宜，过量服用可能带来健康风险。
