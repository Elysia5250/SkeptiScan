# Case Analysis

**Input:** 南极磷虾油胶囊。中国科学院南海海洋研究所（联合实验室合作项目）参与研发。适宜人群：血脂偏高者。本品为保健食品，保健功能：辅助降血脂。市场价仅供参考，实际价格以门店为准。

**Verdict:** not_scam

---

## 风险分析报告

### 产品概述
该产品为南极磷虾油胶囊，声称由中国科学院南海海洋研究所（联合实验室合作项目）参与研发，适宜人群为血脂偏高者，保健功能为辅助降血脂。产品标注为保健食品。

### 风险评分：2/10（安全）

### 评分依据
1. **产品性质**：该产品明确标注为保健食品，功能声称“辅助降血脂”属于国家允许的保健食品功能范围，未涉及疾病治疗或虚假疗效承诺。
2. **科研背书**：提及中国科学院南海海洋研究所参与研发，虽未提供具体合作细节，但科研机构名称本身不构成虚假宣传，需进一步核实。
3. **无危险信号**：未发现伪科学术语（如量子、磁疗）、虚假权威认证（如“CCTV推荐”）、健康焦虑营销或投资诈骗特征。
4. **市场价说明**：标注“市场价仅供参考，实际价格以门店为准”属于正常商业说明，无欺诈意图。

### 潜在风险点（轻微）
- **科研合作真实性**：消费者无法直接验证“联合实验室合作项目”是否真实存在，可能存在夸大宣传。建议通过中国科学院官网或产品包装上的批准文号查询。
- **保健食品局限性**：保健食品不能替代药物，对于需要药物治疗的高血脂患者，仅靠本品可能无效。

### 建议
- 购买前确认产品包装上有“蓝帽子”标志和批准文号（格式如：国食健注G/J+年份+编号）。
- 通过国家市场监督管理总局特殊食品信息查询平台核实批准信息。
- 对于高血脂患者，应遵医嘱治疗，保健食品仅作为辅助。

### 结论
该产品属于正规保健食品，风险极低，但消费者仍需保持理性，核实科研合作信息，避免被营销话术误导。
