# Case Analysis

**Input:** 【消费返利理财】每日返利1%稳赚不赔，推荐好友赚佣金，拉满收益翻倍，限时优惠仅需198元！

**Risk Level:** 10/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：10（明确诈骗）

### 风险信号识别

1. **虚假高回报承诺**：宣传“每日返利1%”，年化收益率高达365%，远超正常投资回报率，属于典型的庞氏骗局特征。
2. **稳赚不赔宣传**：任何投资都有风险，承诺“稳赚不赔”违反金融常识，是诈骗的常见话术。
3. **拉人头返佣机制**：鼓励推荐好友赚佣金，这是传销和庞氏骗局的典型结构，依赖新用户资金支付老用户收益。
4. **限时优惠制造紧迫感**：用“限时优惠”诱导用户快速决策，避免理性思考。

### 事实核查

- 此类“消费返利”模式在中国已被多次定性为非法集资或传销，例如“善心汇”、“云联惠”等案件。
- 没有任何合法金融机构会提供每日1%的固定返利。
- 该宣传未提供任何公司名称、资质证明或合同条款，无法核实其合法性。

### 最终建议

**绝对不要购买或参与**。该宣传内容符合庞氏骗局的所有特征，参与者最终将面临本金损失和法律风险。建议立即向当地金融监管部门或公安机关举报。
