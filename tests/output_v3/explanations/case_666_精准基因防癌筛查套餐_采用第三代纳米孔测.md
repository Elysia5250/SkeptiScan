# Case Analysis

**Input:** 精准基因防癌筛查套餐。采用第三代纳米孔测序技术（宣传用语，非实际采用），一次检测覆盖3000种癌症相关基因位点。我们不是医疗机构，不提供诊断。现在购买基因检测套餐，免费赠送价值1980元的量子波动检测仪一台！。

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：8（高度可疑，接近诈骗）

### 主要风险信号

1. **虚假技术宣传**：明确标注“宣传用语，非实际采用”，说明商家承认夸大技术，属于故意误导消费者。

2. **夸大检测范围**：声称覆盖3000种癌症相关基因位点，远超正规基因检测的实际能力（通常针对特定癌种或遗传风险），缺乏科学依据。

3. **伪科学赠品**：赠送“量子波动检测仪”是典型的伪科学产品，量子波动在医学检测中无任何有效应用，此类赠品常被用于吸引对科技不了解的消费者。

4. **规避医疗责任**：声明“不提供诊断”，但宣传内容却暗示防癌筛查功能，属于打擦边球，逃避医疗监管。

### 营销手法分析

- 使用“第三代纳米孔测序”等前沿术语，制造技术领先假象。
- 通过免费赠送高价值伪科学产品，诱导消费者忽略产品本身的可疑性。
- 利用消费者对癌症的恐惧心理，以“防癌筛查”为卖点，但实际检测价值存疑。

### 事实核查建议

- 查询国家药监局数据库，确认该公司是否具备基因检测的医疗器械注册证。
- 搜索“量子波动检测仪”是否被官方或权威机构认定为有效医疗设备。
- 对比正规医院或权威基因检测机构（如华大基因）的同类服务，其通常针对特定基因位点，而非笼统的“3000种”。

### 最终建议

该产品综合了虚假宣传、伪科学和营销陷阱，风险极高。消费者应选择正规医疗机构进行基因检测，切勿因赠品或夸大宣传而购买。对于老年人群体，尤其要警惕“量子”“防癌”等词汇的误导，避免财产损失和健康风险。
