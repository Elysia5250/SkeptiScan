# Case Analysis

**Input:** 精准基因防癌筛查套餐。我们不是医疗机构，不提供诊断。与北京某三甲医院（退休医生个人合作）提供报告解读服务。但据统计，90%的早期癌症患者因未及时发现而错过最佳治疗期。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 7/10

**Verdict:** scam

---

### 风险分析报告

**产品名称**：精准基因防癌筛查套餐

**风险等级**：7（高风险，接近诈骗）

**主要风险信号**：

1. **虚假/夸大功效暗示**：宣传“精准基因防癌筛查”，但未提供任何科学证据表明该筛查能有效发现早期癌症。基因筛查在癌症预防中仅作为辅助手段，不能替代临床诊断。

2. **模糊权威背书**：声称“与北京某三甲医院（退休医生个人合作）”，但未提供具体医院名称、医生姓名或合作证明。退休医生个人合作不等于医院官方认可，存在误导消费者以为有医院背书的嫌疑。

3. **健康焦虑营销**：引用“90%的早期癌症患者因未及时发现而错过最佳治疗期”这一未注明来源的统计数据，刻意制造恐慌情绪，诱导消费者购买。

4. **免责声明矛盾**：明确声明“不是医疗机构，不提供诊断”，但宣传内容却暗示能筛查癌症，这属于典型的“打擦边球”行为，试图规避医疗监管。

**事实核查建议**：
- 查询国家药监局（NMPA）数据库，确认该基因筛查产品是否获得医疗器械注册证。
- 联系北京相关三甲医院，核实是否有退休医生与该品牌合作。
- 搜索“90%早期癌症患者错过治疗期”的来源，大概率是捏造或断章取义。

**最终建议**：该产品风险极高，不建议购买。消费者应选择正规医疗机构进行癌症筛查，如低剂量螺旋CT（肺癌）、胃肠镜（消化道癌）等，而非依赖此类无资质、无保障的基因筛查套餐。
