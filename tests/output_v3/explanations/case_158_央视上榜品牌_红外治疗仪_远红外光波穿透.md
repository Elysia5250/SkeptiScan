# Case Analysis

**Input:** 央视上榜品牌【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限量100套赶快下单！

**Risk Level:** 8/10

**Verdict:** scam

## 风险分析报告

### 1. 风险等级：8（高度可疑，接近诈骗）

### 2. 风险信号识别

- **虚假疗效宣称**：产品明确声称“治好颈椎病”，这是典型的疾病治疗承诺。根据《广告法》和《医疗器械监督管理条例》，非医疗器械或未取得相应注册证的普通产品不得宣称治疗疾病。远红外理疗类产品通常只能作为辅助缓解，不能替代治疗。

- **伪科学术语**：“远红外光波穿透皮下15cm”缺乏科学依据。远红外线的穿透深度通常只有几毫米到几厘米，15cm的穿透深度在人体组织内不可能实现，属于明显的伪科学夸大。

- **虚假权威背书**：“央视上榜品牌”是常见的营销话术，但未提供具体节目名称、播出时间或可查证的官方链接。许多商家通过购买广告或虚假宣传来冒充“央视推荐”。

- **健康焦虑营销**：强调“不吃药不打针”暗示传统医疗的负面，诱导消费者放弃正规治疗。同时“限量100套”制造稀缺性，促使冲动消费。

### 3. 事实核查建议

- 搜索“央视上榜品牌 红外治疗仪”通常会发现大量类似产品，但央视从未正式认证过任何“上榜品牌”名单。
- 查询国家药品监督管理局（NMPA）数据库，看该产品是否具有医疗器械注册证。若无，则属于违法销售。
- 咨询骨科或康复科医生：远红外理疗对颈椎病有一定缓解作用，但绝不可能“治好”。

### 4. 最终建议

**强烈不建议购买**。该产品利用老年人对慢性病的焦虑和对央视的信任，通过虚假疗效和伪科学包装进行营销。购买此类产品不仅浪费金钱，还可能延误正规治疗。请通过正规医院或药店购买有明确医疗器械注册号的理疗设备，并遵医嘱使用。
