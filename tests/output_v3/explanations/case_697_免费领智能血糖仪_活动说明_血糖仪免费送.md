# Case Analysis

**Input:** 免费领智能血糖仪。活动说明：血糖仪免费送，耗材有偿提供。我们靠耗材赚钱。免费领！原价598元的智能语音血糖仪（市场同类产品实际价格50-80元），现在0元领取！只需支付耗材费299元（含50条试纸+采血针），后续试纸需从我司购买（单价6.9元/条，市场价2元/条以下）。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 6/10

**Verdict:** scam

---

## 详细分析

### 风险评分：6/10（可疑）

该活动属于典型的“免费赠送+高价耗材”模式，虽然不直接涉及虚假医疗功效或伪科学，但存在明显的营销陷阱和价格虚高问题。

### 风险信号识别

1. **虚高原价**：活动声称原价598元，但市场同类产品实际价格仅50-80元，虚高约10倍，属于典型的“锚定效应”营销技巧，让消费者误以为获得巨大优惠。

2. **高价耗材锁定**：首次耗材费299元（含50条试纸+采血针），后续试纸单价6.9元/条，而市场价通常低于2元/条。这意味着消费者一旦开始使用，后续将被迫支付远高于市场价的价格，长期成本显著增加。

3. **免费赠送的误导性**：虽然血糖仪本身免费，但消费者实际支付了299元耗材费，相当于以高于市场价的价格购买了耗材，并附赠了血糖仪。这种模式并不违法，但容易让消费者忽略实际成本。

4. **缺乏资质信息**：活动未提供公司名称、医疗器械注册证号等关键信息，消费者无法核实产品是否合规。

### 事实核查

- 市场同类智能语音血糖仪价格：经查询主流电商平台（如京东、天猫），类似功能的产品价格多在50-80元之间，部分品牌甚至更低。598元的原价明显虚高。
- 试纸市场价格：通用型血糖试纸单价通常在1-3元/条，品牌试纸如罗氏、强生等约3-5元/条。6.9元/条的价格显著高于市场平均水平。
- 商业模式：这种“免费硬件+高价耗材”模式在打印机、净水器等领域常见，但用于医疗器械时，需警惕消费者因价格敏感而减少检测频率，影响健康管理。

### 最终建议

该活动不构成直接欺诈，但存在明显的价格误导和长期消费绑定风险。消费者应：
- 自行比价，确认血糖仪和试纸的市场价格。
- 选择正规渠道购买，避免被锁定在高价耗材上。
- 对于老年人，建议子女协助评估，避免因“免费”而冲动消费。

**评分理由**：未发现虚假医疗功效或伪科学，但虚高原价和高价耗材锁定属于高风险营销手段，因此评为6分（可疑）。
