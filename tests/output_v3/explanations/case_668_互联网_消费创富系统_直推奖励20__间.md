# Case Analysis

**Input:** 互联网+消费创富系统。直推奖励20%，间推奖励10%，三代奖励5%。月入过万不是梦：推荐10人，团队裂变至100人，月被动收入超5万元！已有来自全国32个省市超过50万人参与，累计分配利润超过20亿元。平台资质：已取得ICP备案（正在办理中）、与某国有银行（仅签署意向协议）达成资金存管合作。本广告内容由品牌方提供，不代表本平台立场。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细风险分析

### 风险等级：9（高度疑似传销诈骗）

### 风险信号识别

1. **多级推荐奖励模式**：文案明确列出直推奖励20%、间推奖励10%、三代奖励5%，这是典型的传销金字塔结构。参与者主要通过发展下线获取收益，而非通过销售真实产品或服务。

2. **夸大收益承诺**：“月入过万不是梦”、“月被动收入超5万元”等表述，利用人们对快速致富的渴望进行诱导。此类承诺在正规投资或商业项目中极为罕见，通常是诈骗的常见话术。

3. **虚假或模糊的资质背书**：
   - “已取得ICP备案（正在办理中）”自相矛盾——已取得就不应“正在办理”。ICP备案是网站运营的基本要求，但并非金融或传销活动的合法凭证。
   - “与某国有银行（仅签署意向协议）达成资金存管合作”——意向协议不具备法律效力，且故意模糊银行名称，无法核实。

4. **数据造假**：声称“50万人参与”、“累计分配利润20亿元”，但未提供任何可验证的第三方审计或监管数据。此类数字通常是为了制造项目火爆的假象，吸引更多人加入。

5. **免责声明**：“本广告内容由品牌方提供，不代表本平台立场”试图将责任推给品牌方，但发布平台若未尽审核义务，仍需承担相应法律责任。

### 法律与监管视角

根据中国《禁止传销条例》，传销行为的主要特征包括：拉人头、收取入门费、团队计酬（多级返利）。本案中的推荐奖励模式完全符合“团队计酬”特征，且未提及任何实质性商品或服务，属于典型的“拉人头”传销。

此外，此类项目往往没有真实的盈利来源，资金链完全依赖新加入者的投入，一旦新增人数放缓，平台可能随时跑路，参与者将血本无归。

### 最终建议

**不要参与，不要投资，立即远离。** 如果已经投入资金，请尽快收集证据（转账记录、聊天记录、宣传材料等）并向当地公安机关经侦部门或市场监管部门举报。同时，提醒身边的亲友提高警惕，避免上当受骗。
