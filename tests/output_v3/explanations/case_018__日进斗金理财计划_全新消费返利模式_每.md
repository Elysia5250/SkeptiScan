# Case Analysis

**Input:** 【日进斗金理财计划】全新消费返利模式，每天返利1%，投1万每天赚100，稳赚不赔！推荐好友额外奖励10%佣金。限时加入，名额仅剩最后20个！

**Risk Level:** 10/10

**Verdict:** scam

---

## 详细风险分析

### 风险评分：10/10（明确诈骗）

### 主要风险信号

1. **虚假收益承诺**：
   - 文案声称“每天返利1%”，即年化收益率高达365%，远超任何合法金融产品的正常回报率。正规理财产品的年化收益率通常在3%-8%之间，超过10%即需高度警惕。
   - “稳赚不赔”是典型的绝对化承诺，任何投资都存在风险，包括本金损失。合法金融机构严禁使用此类表述。

2. **庞氏骗局特征**：
   - 推荐好友奖励10%佣金，这是典型的拉人头模式，新投资者的资金被用于支付老投资者的返利，而非来自真实盈利。
   - 高额日返利不可持续，一旦新增资金放缓，平台立即崩盘，后加入者将血本无归。

3. **制造紧迫感**：
   - “限时加入，名额仅剩最后20个”是常见的营销伎俩，目的是阻止用户理性思考，诱导冲动决策。

4. **缺乏合法资质**：
   - 文案未提及任何金融牌照、监管机构或备案信息。在中国，任何面向公众的理财计划必须经过银保监会或证监会批准，否则即为非法集资。

### 事实核查结果

- 经搜索，未发现任何名为“日进斗金理财计划”的合法金融产品。
- 类似“每天返利1%”的骗局已被多地警方和媒体多次曝光，如“消费返利”“刷单返利”等模式。
- 该文案未提供公司名称、地址、联系方式等可验证信息，进一步证实其欺诈性质。

### 最终建议

**立即远离，切勿投资！** 该计划是典型的庞氏骗局，参与者的本金将面临100%损失风险。建议向当地公安机关经侦部门或金融监督管理局举报，避免更多人上当。
