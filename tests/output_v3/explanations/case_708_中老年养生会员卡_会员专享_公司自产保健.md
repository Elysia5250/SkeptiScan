# Case Analysis

**Input:** 中老年养生会员卡。会员专享：公司自产保健品全场8折、优先预约国内专家（退休副主任医师视频问诊5分钟）、定期旅游。本卡一经售出，概不退换。每月15日会员日，全场买二送一！仅限会员参加。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 5/10

**Verdict:** not_scam

---

## 风险分析报告

### 风险等级：5（可疑）

### 主要风险信号

1. **模糊的专家问诊承诺**：宣称“优先预约国内专家（退休副主任医师视频问诊5分钟）”，但未提供任何具体专家姓名、执业机构或资质证明。退休副主任医师的身份无法核实，且“优先预约”可能只是营销话术，实际服务可能缩水或无法兑现。

2. **不可退款条款**：“本卡一经售出，概不退换”属于典型的霸王条款，违反《消费者权益保护法》中关于经营者不得以格式条款排除消费者权利的规定。这增加了消费者的风险，一旦购买后对服务不满意，将无法追回款项。

3. **模糊的旅游服务**：“定期旅游”未说明频率、目的地、费用包含范围等细节，可能只是吸引消费者的噱头，实际组织方或服务质量无法保证。

4. **价格误导**：“市场价仅供参考，实际价格以门店为准”为常见规避价格欺诈的手段，消费者无法判断折扣的真实性。

### 营销手法分析

- **限时优惠**：“每月15日会员日，全场买二送一”利用时间压力促使冲动消费。
- **稀缺性**：“仅限会员参加”制造专属感，诱导非理性购买。
- **价格锚定**：通过“市场价仅供参考”模糊定价，让消费者误以为折扣力度大。

### 事实核查建议

- 通过国家企业信用信息公示系统查询该公司是否注册，以及是否有保健品销售资质。
- 要求商家提供退休副主任医师的姓名和执业证书编号，并通过卫健委官网核实。
- 要求提供旅游服务的具体合同条款，包括组织方、行程、费用等。
- 检查会员卡合同是否违反《消费者权益保护法》第26条关于格式条款的规定。

### 最终建议

该会员卡存在多个可疑点，但尚未达到明显欺诈的程度（如无虚假疾病治疗承诺）。建议消费者谨慎购买，优先选择有明确退款政策、资质公开的商家。若商家无法提供专家资质证明和旅游服务细则，应避免购买。老年人尤其要注意，不要被“专家问诊”和“旅游”等模糊承诺所吸引，以免造成经济损失。
