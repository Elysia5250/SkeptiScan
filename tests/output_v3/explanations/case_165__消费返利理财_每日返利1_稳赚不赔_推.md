# Case Analysis

**Input:** 【消费返利理财】每日返利1%稳赚不赔，推荐好友赚佣金，拉满收益翻倍，限时优惠仅需198元！

**Risk Level:** 10/10

**Verdict:** scam

---

## 详细风险分析

### 风险评分：10/10（明确诈骗）

### 风险信号识别

1. **虚假收益承诺**：宣传“每日返利1%稳赚不赔”。年化收益率高达365%，远超正常投资回报率，且“稳赚不赔”是典型的诈骗话术，任何投资都有风险。

2. **传销式推广**：“推荐好友赚佣金，拉满收益翻倍”表明该项目依赖拉人头而非实际盈利，符合庞氏骗局或传销特征。

3. **限时优惠压力**：“限时优惠仅需198元”制造紧迫感，诱导用户快速决策，避免理性思考。

### 事实核查

- 此类“消费返利”项目在历史上多次被认定为非法集资或诈骗，例如“云联惠”“善林金融”等案例。
- 没有任何合法金融机构会承诺每日固定高返利。
- 中国银保监会、公安部等多次发布风险提示，警惕“高额返利”骗局。

### 结论

该宣传内容完全符合诈骗特征，属于典型的庞氏骗局或非法集资。建议立即远离，并向当地公安机关或金融监管部门举报。
