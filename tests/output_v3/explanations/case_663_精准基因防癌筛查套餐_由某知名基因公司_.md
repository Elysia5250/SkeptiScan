# Case Analysis

**Input:** 精准基因防癌筛查套餐。由某知名基因公司（已停止该业务）提供技术支持。检测发现高风险基因者，可搭配购买本中心定制防癌套餐（另收费29800元/年）。检测结果仅供参考。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 1. 产品概述
该商品为“精准基因防癌筛查套餐”，声称由某知名基因公司提供技术支持（但该公司已停止该业务），检测后若发现高风险基因，可搭配购买每年29800元的定制防癌套餐。

### 2. 风险信号识别
- **虚假/夸大功效**：基因检测对癌症风险的预测并非100%准确，宣传“精准防癌”容易误导消费者认为检测结果等同于诊断。
- **健康焦虑营销**：通过暗示“高风险”来诱导消费者购买高额套餐，利用恐惧心理。
- **模糊权威背书**：提及知名公司但已停止业务，可能利用其剩余声誉，实际技术支持存疑。
- **捆绑销售**：检测后立即推荐高价套餐，形成消费闭环，缺乏透明选择。

### 3. 评分依据
- **风险等级7**：属于较高风险，但未达到完全欺诈（如无产品、虚假公司）。主要问题在于夸大基因检测的临床意义，并利用健康焦虑推销高价服务。
- 该检测结果“仅供参考”的声明降低了部分法律风险，但整体营销模式仍具有欺骗性。

### 4. 事实核查建议
- 查询该基因公司是否确实已停止相关业务，以及其技术是否经过认证。
- 核实该检测是否获得国家药品监督管理局（NMPA）批准用于癌症风险评估。
- 对比正规医院（如三甲医院）的基因检测价格（通常数百至数千元）和科学依据。

### 5. 最终建议
**不推荐购买**。基因检测用于癌症筛查应基于医生建议，而非商业促销。消费者应警惕任何将检测与高价套餐捆绑的销售模式，尤其是针对老年人。

### 6. 老年人特别提醒
老年人更容易被“防癌”“精准”等词汇吸引，且可能因健康焦虑而冲动消费。请务必与家人商量，或咨询正规医疗机构后再做决定。
