# Case Analysis

**Input:** 蜂胶抗癌黄金胶囊——蜂胶王浆胶囊，国家食品药品监督管理局批准（批准文号：国食健字G2024XXXX），中国蜂产品协会推荐品牌。中国农业大学蜂。国药准字Z2025XXXX，GMP认证，三甲医院临床验证有效率91.3%

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细风险分析

### 1. 批准文号严重矛盾
- 产品同时标注“国食健字G2024XXXX”（保健食品）和“国药准字Z2025XXXX”（药品），这是不可能的。一个产品只能有一个批准文号，且保健食品不能宣称治疗疾病，药品不能以保健食品形式销售。这种混用是典型的伪造文号手法。
- 国药准字Z2025XXXX中的“Z”代表中药，但蜂胶通常作为保健食品原料，极少作为药品批准。该文号很可能不存在或已被注销。

### 2. 虚假疗效宣称
- 宣称“蜂胶抗癌”且“三甲医院临床验证有效率91.3%”，这是极其严重的虚假宣传。蜂胶在科学上未被证实能治疗癌症，任何声称有如此高有效率的保健品或药品都违反广告法。
- 未提供任何具体医院名称、研究论文或数据来源，属于凭空捏造。

### 3. 权威背书可疑
- “中国蜂产品协会推荐品牌”和“中国农业大学蜂”字样，但未提供任何官方推荐文件或合作证明。此类模糊引用常见于欺诈产品。
- “GMP认证”是药品生产的基本要求，但单独列出并不能证明产品真实有效。

### 4. 风险评分理由
- 综合以上因素，该产品存在伪造批准文号、虚假疗效、滥用权威背书等多项严重欺诈信号，风险等级为9（接近最高级10）。
- 消费者若购买，不仅浪费金钱，还可能延误正规治疗，危害健康。

### 5. 最终建议
- **不要购买**。立即向当地市场监管部门或12315举报该产品。
- 对于任何宣称能治疗癌症、糖尿病等重大疾病的保健品或药品，务必通过官方渠道核实批准文号，并咨询专业医生。
