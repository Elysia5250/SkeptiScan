# Case Analysis

**Input:** 【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限时优惠仅需198元！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为一款名为“红外治疗仪”的电子设备，宣传语强调“远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病”，并标注“限时优惠仅需198元”。

### 风险信号识别
1. **虚假疗效宣称**：声称“治好颈椎病”属于对慢性疾病的根治性承诺，违反《广告法》关于医疗器械广告不得含有“断言或者保证”功效的规定。颈椎病为退行性病变，目前医学上无法“根治”，此类表述具有明显欺骗性。
2. **伪科学数据**：“穿透皮下15cm”严重违背物理学常识。远红外线的穿透深度通常仅为皮下几毫米至1-2厘米，15cm远超人体软组织厚度，属于无依据的夸大。
3. **低价陷阱**：正规医用红外治疗仪价格通常在数百至数千元，198元远低于合理成本，极可能为劣质或普通发热贴片冒充。
4. **营销话术**：“限时优惠”制造紧迫感，诱导冲动消费。

### 风险等级评估
综合以上信号，该产品存在明确的虚假疗效、伪科学包装和低价欺诈特征，风险等级为**8分**（高度可疑，接近诈骗）。

### 建议
- 消费者切勿购买，尤其是老年人易受“不吃药不打针”等话术误导。
- 如已购买，可向当地市场监督管理局或12315投诉。
- 颈椎病治疗应遵循医嘱，采用正规物理治疗、药物或手术方案。
