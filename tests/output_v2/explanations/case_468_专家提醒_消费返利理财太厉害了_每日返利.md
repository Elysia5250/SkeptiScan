# Case Analysis

**Input:** 专家提醒：消费返利理财太厉害了！每日返利1%稳赚不赔，推荐好友赚佣金，拉满收益翻倍，买一送一！

**Risk Level:** 10/10

**Verdict:** scam

---

## 详细分析

### 风险等级：10（最高风险）

### 风险信号识别

1. **虚假收益承诺**：宣传“每日返利1%稳赚不赔”，年化收益率高达365%，远超正常投资回报率，属于典型的“高息诱饵”。
2. **传销式推广**：鼓励“推荐好友赚佣金”，并声称“拉满收益翻倍”，这是庞氏骗局或非法传销的典型特征，通过发展下线来维持资金链。
3. **绝对化用语**：使用“稳赚不赔”等绝对化承诺，违反广告法，且不符合任何投资规律。
4. **缺乏合法资质**：未提及任何金融牌照或监管机构，属于非法集资或诈骗。

### 事实核查

- 任何承诺“稳赚不赔”且日回报率超过1%的理财项目，几乎都是骗局。
- 中国银保监会多次警示此类“消费返利”模式为非法集资。
- 类似案例（如“云联惠”“善林金融”）均已被警方查处。

### 最终建议

**立即远离**，不要投入任何资金，也不要推荐给亲友。如已参与，请尽快向当地公安机关报案。
