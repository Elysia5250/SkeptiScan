# Case Analysis

**Input:** 专家提醒：红外治疗仪太厉害了！远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，买一送一！

**Risk Level:** 8/10

**Verdict:** scam

## 风险分析报告

### 总体评分：8/10（高风险，接近诈骗）

### 风险信号识别

1. **虚假疗效承诺**：
   - 宣称“治好颈椎病”属于对慢性疾病的治愈承诺，违反《广告法》中禁止非药品宣传疾病治疗功效的规定。
   - “消炎止痛”属于医疗术语，普通电子产品不得使用。

2. **伪科学概念**：
   - “远红外光波穿透皮下15cm”严重夸大。根据生物医学工程研究，远红外线（波长4-14μm）在人体组织中的穿透深度通常仅为0.5-5mm，15cm的说法毫无科学依据。

3. **虚假权威背书**：
   - “专家提醒”未指明具体专家姓名、机构或资质，属于典型的虚构权威手段。

4. **营销诱导**：
   - “太厉害了”使用夸张情绪化语言，配合“买一送一”限时优惠，诱导冲动消费。

### 事实核查结果

- 经查，国家药监局（NMPA）数据库中未发现名为“红外治疗仪”且宣传穿透15cm的合法医疗器械注册信息。
- 远红外理疗产品（如红外灯）通常仅用于辅助缓解肌肉酸痛，且需明确标注“不替代药物治疗”。
- 类似宣传已被多地市场监管部门认定为虚假广告并处罚。

### 最终建议

该产品极可能为三无产品或虚假宣传的保健品，不仅无效，还可能因使用不当（如过热、照射时间过长）导致皮肤灼伤。建议消费者：
- 立即停止购买意向。
- 如已购买，保留证据并向12315投诉。
- 颈椎病应前往正规医院骨科或康复科就诊，接受科学治疗。
