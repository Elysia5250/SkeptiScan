# Case Analysis

**Input:** 【红外治疗仪】30粒装，专家强烈推荐，央视合作品牌，品质有保障！立即抢购！

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品宣传为“红外治疗仪”，规格为“30粒装”，但未明确说明是医疗器械、保健品还是药品。宣传语包含“专家强烈推荐”、“央视合作品牌”、“品质有保障”等营销话术，并带有“立即抢购”的紧迫感提示。

### 风险信号识别
1. **虚假权威背书**：
   - “专家强烈推荐”未指明具体专家姓名、职称或所属机构，无法验证。
   - “央视合作品牌”未提供任何可查证的报道、广告或授权文件，常见于虚假宣传。
2. **模糊产品属性**：
   - “30粒装”暗示可能是口服产品，但“红外治疗仪”通常指物理治疗设备，两者矛盾，可能故意混淆概念。
3. **营销紧迫感**：
   - “立即抢购”是典型促销手段，诱导冲动消费，掩盖产品真实信息。

### 风险等级评定
根据评分标准，该产品存在以下问题：
- 使用“专家推荐”、“央视合作”等无法验证的权威背书（风险信号3）。
- 产品描述模糊，可能涉及虚假功效暗示（风险信号1）。
- 未提供任何科学依据或资质证明。

综合判断，风险等级为 **7（高风险）**，属于有明显欺诈嫌疑的产品，建议消费者避免购买。

### 建议行动
- 消费者应要求商家提供医疗器械注册证号、生产许可证号及央视合作证明。
- 通过国家药品监督管理局官网查询产品备案信息。
- 保留购买凭证，如发现虚假宣传可向12315投诉。

### 结论
该产品宣传内容高度可疑，极有可能是利用老年人对权威和健康的信任进行欺诈，强烈建议不要购买。
