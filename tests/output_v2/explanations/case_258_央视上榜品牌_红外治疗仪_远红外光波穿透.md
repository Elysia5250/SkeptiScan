# Case Analysis

**Input:** 央视上榜品牌【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限量100套赶快下单！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 总体评分：9分（高度疑似诈骗）

### 风险信号识别

1. **虚假疗效宣称**：
   - 声称“远红外光波穿透皮下15cm”——人体皮下组织厚度远小于15cm，且远红外光波穿透深度通常仅数毫米至1-2厘米，15cm是严重夸大，无科学依据。
   - “消炎止痛，不吃药不打针治好颈椎病”——颈椎病属于慢性退行性疾病，目前没有任何家用仪器能“治好”，该说法涉嫌虚假医疗广告，违反《广告法》。

2. **伪权威背书**：
   - “央视上榜品牌”是常见营销话术，通常无法在央视官方渠道查证。真正的央视合作会有具体节目名称和播出记录，而此处未提供任何可核实信息。

3. **营销伎俩**：
   - “限量100套赶快下单”——制造稀缺性和紧迫感，诱导冲动消费。
   - 利用消费者对“央视”的信任，模糊品牌真实性。

### 科学事实核查

- 远红外线疗法在物理治疗中确有辅助作用，但仅限于缓解肌肉紧张、促进局部血液循环，**不能替代药物治疗或手术**，更不可能“治好”颈椎病。
- 国家药品监督管理局（NMPA）对“红外治疗仪”有严格分类管理，二类及以上医疗器械需注册审批。但该产品未提供任何注册号，且宣传语明显超出许可范围。

### 结论与建议

该产品属于典型的“伪医疗器械”诈骗，利用老年人对慢性病的焦虑和对权威媒体的信任，通过夸大疗效、虚构背书、制造紧迫感来诱导购买。**强烈建议不要购买**，并提醒身边老年人警惕类似宣传。如已购买，可向当地市场监督管理局或12315投诉举报。
