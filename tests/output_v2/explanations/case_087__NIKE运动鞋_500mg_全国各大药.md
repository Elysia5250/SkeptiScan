# Case Analysis

**Input:** 【NIKE运动鞋】500mg，全国各大药房及电商平台均有销售。官方售价198元。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 风险信号识别

1. **品类与描述矛盾**：商品名称为“NIKE运动鞋”，但描述中出现了“500mg”这一药品或保健品常用的剂量单位。运动鞋不可能以毫克计量，这明显是错误或故意混淆的信息。

2. **销售渠道异常**：声称“全国各大药房及电商平台均有销售”。运动鞋通常通过体育用品店、品牌专卖店或电商平台销售，药房并非其常规销售渠道。将药房列为销售点，可能是为了借用药房的专业形象来误导消费者，尤其是老年人。

3. **价格异常**：官方售价198元。NIKE品牌运动鞋的正品价格通常在300元以上，198元远低于市场均价，极可能是假冒产品。

### 风险等级评估

- **品类矛盾**：+3分（严重误导）
- **虚假销售渠道**：+2分（利用药房信任）
- **价格异常**：+2分（低价陷阱）
- **品牌滥用**：+1分（冒用知名品牌）

总分8分，属于高风险（Scam级别）。

### 结论

该商品描述存在多处明显矛盾，极可能是诈骗或假冒产品。消费者应避免购买，并通过正规渠道核实NIKE产品信息。
