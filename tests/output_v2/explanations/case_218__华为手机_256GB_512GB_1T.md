# Case Analysis

**Input:** 【华为手机】256GB|512GB|1TB，正规品牌，各大电商平台及实体店有售。售价198元。

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 商品信息
- **声称品牌**：华为
- **存储容量**：256GB/512GB/1TB
- **售价**：198元

### 风险信号识别
1. **价格异常**：华为品牌手机，即使是入门级型号，售价也远高于198元。例如，华为畅享系列最低售价也在1000元以上，而高端旗舰机型（如Mate系列、P系列）售价在4000元至10000元不等。198元的价格完全不符合市场规律，极有可能是假冒产品、翻新机或诈骗。
2. **品牌滥用**：利用“华为”这一知名品牌吸引消费者，但未提供任何具体型号、配置或官方认证信息，属于典型的“傍名牌”欺诈手段。
3. **虚假容量宣传**：声称提供256GB、512GB甚至1TB存储，但如此低的价格根本无法支撑这些高容量存储芯片的成本，明显是虚假宣传。

### 风险等级评定
- **风险等级：9/10**（高度疑似诈骗）
- 该商品几乎可以确定是假冒伪劣产品或诈骗信息，购买后极有可能收到劣质手机、空包裹或根本无法收到商品。

### 购买建议
- **绝对不要购买**。请通过华为官方商城、京东自营、天猫官方旗舰店等正规渠道购买手机。
- 如果看到类似“超低价名牌手机”的广告，请立即举报。

### 针对老年用户的特别提醒
- 请勿相信任何“名牌手机仅售几百元”的广告，这通常是针对老年人的诈骗手段。购买手机请到正规实体店或大型电商平台，并索要发票。
