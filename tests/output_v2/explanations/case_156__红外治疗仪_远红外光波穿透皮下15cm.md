# Case Analysis

**Input:** 【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限时优惠仅需198元！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为“红外治疗仪”，宣传语强调“远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病”，并采用“限时优惠仅需198元”的促销手段。

### 风险信号识别

1. **虚假疗效承诺（高风险）**：
   - “治好颈椎病”属于绝对化疗效承诺，违反《广告法》第十七条（除医疗、药品、医疗器械广告外，禁止其他任何广告涉及疾病治疗功能）。
   - “不吃药不打针”暗示替代正规医疗，可能误导消费者放弃必要治疗。

2. **伪科学概念（高风险）**：
   - “远红外光波穿透皮下15cm”严重夸大。科学上，远红外线（波长8-14μm）在人体组织中的穿透深度通常仅为数毫米至1-2厘米，15cm的穿透深度无任何科学依据。
   - “消炎止痛”属于治疗性宣称，非医疗器械不得使用。

3. **营销陷阱（中风险）**：
   - “限时优惠”制造紧迫感，诱导冲动消费。
   - 198元的价格远低于正规医用红外治疗仪（通常数千元），暗示产品可能为劣质或普通发热垫。

4. **缺乏权威背书**：
   - 未提及任何医疗器械注册证号、生产厂家或临床研究，无法验证其合法性。

### 风险等级判定
综合以上因素，该产品存在明确的虚假医疗宣传、伪科学概念和营销欺诈特征，风险等级为 **8（高度可疑）**，接近诈骗。

### 建议
- **不要购买**：该产品很可能为三无产品，不仅无效，还可能因劣质发热元件导致烫伤。
- **正规就医**：颈椎病需经医生诊断后采取物理治疗、药物或手术等科学方法。
- **举报**：可向当地市场监督管理局或12315平台举报该虚假广告。
