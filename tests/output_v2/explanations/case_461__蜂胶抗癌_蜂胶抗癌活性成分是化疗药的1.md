# Case Analysis

**Input:** 【蜂胶抗癌】蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，限时优惠仅需198元！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 评分理由
该商品宣传内容包含多项严重风险信号，综合评分9分（属于明显欺诈）。

### 风险信号识别
1. **虚假医疗功效**：声称蜂胶抗癌活性成分是化疗药的10倍，并宣称能抑制肿瘤扩散、多名患者康复。这些是典型的未经科学验证的疾病治疗承诺，违反《广告法》和《药品管理法》。
2. **伪科学包装**：蜂胶作为普通保健品，其抗癌功效缺乏大规模临床试验支持，更不可能与化疗药物直接比较。
3. **健康焦虑营销**：利用癌症患者及家属的恐惧心理，暗示产品能替代正规治疗。
4. **限时优惠**：制造紧迫感，诱导冲动消费。

### 事实核查
- 蜂胶未被任何国家权威机构（如中国国家药品监督管理局、美国FDA）批准为抗癌药物。
- 化疗药物经过严格临床试验，蜂胶的活性成分浓度和效果无法与之相提并论。
- 所谓的“多名患者康复”无具体病例、医院或医生信息，无法核实。

### 最终建议
该商品属于典型的保健品虚假宣传，涉嫌欺诈。消费者应避免购买，并建议向市场监管部门举报。对于癌症患者，务必遵循正规医疗方案，切勿因轻信此类宣传而延误治疗。
