# Case Analysis

**Input:** 专家提醒：蜂胶抗癌太厉害了！蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，买一送一！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：9（高度欺诈）

### 风险信号识别

1. **虚假疗效宣称**：
   - 明确声称蜂胶“抗癌活性成分是化疗药的10倍”，这是典型的未经科学验证的夸大宣传。化疗药物经过严格的临床试验，而蜂胶作为保健品，从未被批准用于癌症治疗。
   - “抑制肿瘤扩散，多名患者康复”属于不实案例，缺乏可验证的医学证据。

2. **伪权威背书**：
   - 使用“专家提醒”字样，但未提供具体专家姓名、机构或可查证的来源，属于常见的伪权威营销手段。

3. **健康焦虑营销**：
   - 通过暗示蜂胶能“抗癌”“抑制肿瘤”，利用患者对癌症的恐惧心理，诱导购买。

4. **促销陷阱**：
   - “买一送一”是典型的促销手段，用于降低消费者警惕性，加速决策。

### 事实核查

- 根据国家市场监督管理总局规定，普通食品和保健品不得宣传疾病预防或治疗功能。蜂胶属于保健食品，其批准功能通常为“增强免疫力”等，绝无抗癌功效。
- 在权威医学数据库（如PubMed）中，蜂胶的体外或动物实验研究显示某些成分可能具有抗肿瘤活性，但远未达到人体临床验证阶段，更不可能“是化疗药的10倍”。
- 化疗药物剂量和疗效经过严格临床试验，直接对比是荒谬且不负责任的。

### 结论

该宣传内容属于典型的虚假医疗广告，涉嫌欺诈。消费者应避免购买，并向市场监管部门举报。对于癌症患者，请务必遵循正规医疗方案，切勿相信此类“神奇”保健品。
