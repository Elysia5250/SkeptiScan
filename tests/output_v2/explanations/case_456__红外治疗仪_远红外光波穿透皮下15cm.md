# Case Analysis

**Input:** 【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限时优惠仅需198元！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为一款名为“红外治疗仪”的设备，宣传语声称其“远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病”，并标注“限时优惠仅需198元”。

### 风险信号识别
1. **虚假疗效宣称**：明确声称“治好颈椎病”，属于对疾病治疗效果的绝对化承诺，违反了《广告法》中关于非药品不得宣传治疗疾病的规定。
2. **伪科学概念**：“远红外光波穿透皮下15cm”严重夸大。根据物理学和医学常识，远红外线的穿透深度通常只有几毫米到几厘米，15cm的说法毫无科学依据。
3. **健康焦虑营销**：暗示“不吃药不打针”即可治愈，利用患者对药物副作用的恐惧心理，诱导购买。
4. **价格异常**：正规医用红外治疗仪价格通常在数百至数千元，198元的售价远低于市场正常水平，极可能是劣质或无效产品。
5. **限时优惠**：制造紧迫感，促使消费者冲动下单。

### 风险等级评估
综合以上信号，该产品存在明显的虚假宣传和欺诈嫌疑，风险等级评定为**8分**（高风险）。

### 建议
- **消费者**：切勿购买。颈椎病需专业诊断和治疗，建议前往正规医院骨科或康复科就诊。
- **监管**：建议市场监管部门对该产品进行调查，核实其是否具备医疗器械注册证，并依法查处虚假广告。
- **老年人**：尤其容易受此类宣传影响，家人应加强提醒，避免上当受骗。
