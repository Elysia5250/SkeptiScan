# Case Analysis

**Input:** 专家提醒：蜂胶抗癌太厉害了！蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，买一送一！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：9（高度疑似诈骗）

### 主要风险信号

1. **虚假疗效宣称**：
   - 声称“蜂胶抗癌活性成分是化疗药的10倍”——这是典型的伪科学对比，化疗药物经过严格临床试验，而蜂胶作为保健品并无此类数据支持。
   - “抑制肿瘤扩散”和“多名患者康复”属于未经证实的医疗断言，违反《广告法》和《食品安全法》。

2. **伪权威包装**：
   - 使用“专家提醒”模糊表述，未提供具体专家姓名、机构或可查证的来源，属于常见的权威暗示手法。

3. **健康焦虑营销**：
   - 直接针对癌症患者或家属的恐惧心理，暗示“不买就会错过治疗机会”，极具欺骗性。

4. **促销诱导**：
   - “买一送一”进一步降低消费者警惕，促使冲动购买。

### 事实核查结果

- 蜂胶在传统医学中确实有抗菌、抗炎作用，但**没有任何高质量临床研究**证明其能治疗或抑制癌症。
- 中国国家药品监督管理局（NMPA）未批准任何蜂胶产品作为抗癌药物。
- 类似“抗癌神药”宣传已被多地市场监管部门列为典型虚假广告案例。

### 最终建议

**强烈不建议购买**。该产品涉嫌欺诈，不仅浪费金钱，更可能延误患者接受正规医疗。请立即向当地市场监管部门举报（12315）。
