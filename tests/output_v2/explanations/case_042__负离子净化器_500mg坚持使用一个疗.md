# Case Analysis

**Input:** 【负离子净化器】500mg坚持使用一个疗程，效果显著，无效全额退款！原价39.9限时特惠仅需198元！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品为“负离子净化器”，宣传文案中出现了“500mg”、“一个疗程”、“效果显著”、“无效全额退款”等表述，并使用了“原价39.9限时特惠仅需198元”的价格对比策略。

### 风险信号识别

1. **虚假疗效暗示**：使用“疗程”这一医疗术语，暗示产品具有治疗疾病的效果。负离子净化器属于普通家用电器，不具备治疗疾病的功能，此类宣传违反《广告法》中关于非医疗器械不得宣传疗效的规定。

2. **价格矛盾**：宣称原价39.9元，但限时特惠价却为198元，价格逻辑混乱，明显是虚假原价。这种手法常见于欺诈性销售，利用消费者对“限时优惠”的冲动心理。

3. **无效退款承诺**：虽然“无效全额退款”看似保障消费者权益，但实际执行中往往设置苛刻条件或根本不予兑现，是典型的营销陷阱。

4. **伪科学概念**：“负离子”本身是真实存在的物理概念，但许多廉价产品夸大其健康功效，如声称能净化空气、改善睡眠、治疗疾病等，缺乏科学依据。

### 评分依据
- 该产品未直接宣称治疗具体疾病（如癌症、高血压），但使用了“疗程”、“效果显著”等暗示性医疗用语，风险等级为8分。
- 价格矛盾明显，属于欺诈性定价。
- 负离子类产品常被用于针对老年人群体的保健品骗局。

### 建议
- 消费者应避免购买此类宣传疗效的家电产品。
- 如需购买空气净化器，应选择知名品牌，并查看其是否具有正规检测报告。
- 若已购买，可向当地市场监管部门举报虚假宣传。
