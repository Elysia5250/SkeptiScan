# Case Analysis

**Input:** 央视上榜品牌【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限量100套赶快下单！

**Risk Level:** 8/10

**Verdict:** scam

---

### 风险分析报告

**产品名称**：红外治疗仪（宣传为“央视上榜品牌”）

**风险等级**：8（高风险，接近诈骗）

#### 主要风险信号

1. **虚假/模糊权威背书**：
   - “央视上榜品牌”是一个常见的虚假宣传话术。央视从未颁发过此类认证，许多商家通过购买小栏目广告或伪造证书来误导消费者。
   - 未提供任何具体节目名称、播出时间或可查证的链接，属于典型的“权威光环”营销。

2. **伪科学疗效宣称**：
   - “远红外光波穿透皮下15cm”：远红外线的穿透深度通常只有几毫米到1-2厘米，15cm的说法完全违背物理学常识，属于伪科学。
   - “消炎止痛”、“治好颈椎病”：红外治疗仪作为二类医疗器械，其注册适用范围通常仅为“辅助缓解疼痛、促进局部血液循环”，绝不允许宣称“治疗”或“根治”疾病。该宣传直接违反《广告法》和《医疗器械监督管理条例》。

3. **绝对化承诺与健康焦虑营销**：
   - “不吃药不打针治好颈椎病”暗示患者可以放弃正规医疗，极易延误病情。
   - “限量100套”制造紧迫感，诱导冲动消费，尤其针对老年人和慢性病患者。

#### 事实核查建议

- 登录**国家药品监督管理局官网**（www.nmpa.gov.cn），在“医疗器械”栏目中搜索产品名称或生产企业，查看是否有合法的注册证号（格式如：国械注准202X3XXXXXX）。若无，则属于非法产品。
- 搜索“央视上榜品牌 骗局”，可发现大量类似案例已被媒体曝光。
- 咨询正规医院康复科医生，了解远红外治疗的真正效果和适用范围。

#### 最终建议

**强烈不建议购买**。该产品利用老年人对权威媒体的信任和对疾病的恐惧，进行虚假宣传。真正的医疗器械不会用“治好”这类绝对化用语，也不会虚构穿透深度。请消费者（尤其是老年人）务必通过正规渠道就医，切勿轻信此类广告。
