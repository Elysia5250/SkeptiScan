# Case Analysis

**Input:** 🔥今日必抢🔥 红外治疗仪，远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，无效退款！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品为“红外治疗仪”，宣传语强调“远红外光波穿透皮下15cm”“消炎止痛”“不吃药不打针治好颈椎病”，并附有“无效退款”承诺。

### 风险信号识别
1. **虚假疗效承诺**：宣称“治好颈椎病”属于对慢性疾病的根治性承诺，违反《广告法》中关于非药品不得宣传疾病治疗功效的规定。
2. **伪科学表述**：“远红外光波穿透皮下15cm”严重夸大——科学文献显示，远红外线在人体组织中的穿透深度通常仅为数毫米至1厘米左右，15cm的说法毫无依据。
3. **健康焦虑营销**：利用“不吃药不打针”暗示传统治疗痛苦，诱导消费者放弃正规医疗。
4. **无效退款陷阱**：此类承诺在虚假宣传中常见，实际退款条件苛刻或根本无法联系商家。

### 风险等级评定
综合以上信号，该产品风险等级为**8分**（高风险），属于明显欺诈倾向。其核心问题在于：
- 非医疗器械却宣称治疗疾病；
- 使用未经科学验证的穿透深度数据；
- 利用消费者对颈椎病的焦虑进行营销。

### 建议
- **不要购买**：该产品极可能为三无或贴牌产品，不仅无效，还可能延误病情。
- **正规就医**：颈椎病需经医生诊断后采取物理治疗、药物或手术等科学方法。
- **举报**：可向市场监管部门或12315平台举报该虚假宣传。

### 结论
该商品属于典型的“伪科学+虚假疗效”骗局，尤其容易误导老年人和慢性病患者。请务必提高警惕，切勿轻信。
