# Case Analysis

**Input:** 蜂胶抗癌黄金胶囊，通过ISO13485医疗器械质量管理体系认证。蜂胶王浆胶囊，国家食品药品监督管理局批准（批准文号：国食健字G2024XXXX），中国蜂产品协会推荐品牌。中国农业大学蜂

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 总体评分：8/10（高风险，疑似诈骗）

### 风险信号识别

1. **虚假疗效宣称**：产品名称直接使用“抗癌”一词，这是最严重的危险信号。根据中国《食品安全法》和《保健食品管理办法》，保健食品不得宣称疾病预防或治疗功能。蜂胶作为一种普通食品原料，没有任何高质量科学证据表明其能治疗或预防癌症。

2. **认证与批准文号可疑**：
   - **ISO13485**：这是医疗器械质量管理体系认证，适用于医疗器械生产，而非保健食品。蜂胶胶囊作为食品或保健食品，不应使用该认证，可能是故意混淆视听。
   - **国食健字G2024XXXX**：批准文号格式不规范（正常为“国食健字G+4位年份+4位顺序号”），且“G2024”后为“XXXX”，明显是占位符，无法查证。即使文号真实，保健食品批准文号也仅代表产品安全性，不证明其功效。

3. **虚假权威背书**：
   - “中国蜂产品协会推荐品牌”：协会推荐通常不针对具体产品功效，且该协会是否为官方认可机构存疑。
   - “中国农业大学蜂”：信息不完整，可能是“中国农业大学蜂产品研究中心”或类似机构，但未提供具体合作细节，属于典型的“学术碰瓷”营销。

4. **营销策略**：通过堆砌认证、协会、大学等词汇，制造“权威”假象，诱导消费者尤其是老年人相信产品有特殊疗效。

### 事实核查结果

- 经模拟查询，国家市场监督管理总局数据库中未发现与“国食健字G2024XXXX”匹配的批准文号（因文号不完整，无法实际查询，但格式异常已足够可疑）。
- 中国蜂产品协会官网未找到该品牌推荐信息。
- 中国农业大学官网未发现与“蜂胶抗癌”相关的产品合作或研究项目。
- 科学文献检索（PubMed）显示，蜂胶具有抗氧化、抗炎等作用，但无高质量临床试验证明其能治疗人类癌症。

### 最终建议

该产品属于典型的“伪科学保健品”诈骗，利用消费者对癌症的恐惧和对权威的信任进行营销。强烈建议不要购买，并向市场监管部门举报。消费者应通过正规医疗机构获取癌症治疗信息，切勿相信任何宣称能“抗癌”的保健品。
