# Case Analysis

**Input:** 【蜂胶抗癌】蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，限时优惠仅需198元！

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：9（高度欺诈）

### 主要风险信号

1. **虚假医疗功效宣称**：
   - 声称“蜂胶抗癌活性成分是化疗药的10倍”是典型的伪科学对比，没有任何科学依据。蜂胶作为保健品，其抗癌作用从未在严格临床试验中得到证实，更不可能与标准化疗药物进行量化比较。
   - “抑制肿瘤扩散”和“多名患者康复”属于明确的疾病治疗承诺，违反了《广告法》和《食品安全法》中关于普通食品和保健品不得宣传疾病治疗功效的规定。

2. **利用健康焦虑营销**：
   - 通过暗示蜂胶能替代化疗，诱导癌症患者放弃正规治疗，转而购买无效产品，可能延误病情，造成严重后果。
   - “限时优惠仅需198元”是典型的促销话术，制造紧迫感，促使消费者冲动下单。

3. **缺乏权威来源支持**：
   - 宣传中未提供任何可验证的临床研究、专利号或权威机构背书。所谓的“多名患者康复”无法核实，很可能是虚构案例。

### 事实核查结果

- 经查，蜂胶在《中国药典》中仅作为保健品原料，未被批准用于治疗任何疾病。
- 世界卫生组织（WHO）和美国国家癌症研究所（NCI）均未将蜂胶列为抗癌药物。
- 中国国家药品监督管理局（NMPA）数据库中无蜂胶抗癌相关的药品批准文号。

### 最终建议

该产品属于典型的“保健品冒充药品”欺诈，风险极高。消费者切勿购买，已购买者应立即停止使用并联系商家退款。建议向当地市场监督管理局或12315平台举报，防止更多人受骗。
