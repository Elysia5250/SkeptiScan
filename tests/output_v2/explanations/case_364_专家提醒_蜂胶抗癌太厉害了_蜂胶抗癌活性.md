# Case Analysis

**Input:** 专家提醒：蜂胶抗癌太厉害了！蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，买一送一！

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 风险等级：9（高度欺诈）

该商品宣传内容包含多项明确的欺诈信号，属于典型的虚假医疗广告。

### 风险信号识别

1. **虚假疗效声称**：
   - “蜂胶抗癌活性成分是化疗药的10倍”：这是无法验证的夸张比较，化疗药物经过严格的临床试验和审批，而蜂胶作为食品或保健品，从未被批准用于癌症治疗。
   - “抑制肿瘤扩散”、“多名患者康复”：这些声称暗示蜂胶具有治疗癌症的效果，但缺乏任何科学证据或真实病例支持。

2. **伪权威包装**：
   - 使用“专家提醒”这一模糊表述，但未提供任何具体专家姓名、机构或可查证的来源，属于典型的伪权威营销手法。

3. **健康焦虑诱导**：
   - 直接针对癌症患者及其家属的恐惧心理，暗示不购买就会错过“治愈”机会，属于不道德的健康焦虑营销。

4. **促销手段**：
   - “买一送一”是常见的促销策略，用于降低消费者的警惕性，加速冲动购买。

### 事实核查

- 根据中国《食品安全法》和《广告法》，普通食品和保健品不得宣传疾病预防或治疗功能。蜂胶作为保健食品，其批准功能通常仅限于“增强免疫力”等，绝无抗癌功效。
- 在权威医学数据库（如PubMed）中，蜂胶的体外或动物实验研究确实存在，但没有任何高质量的人体临床试验证明蜂胶可以治疗癌症。将体外实验或动物实验的结果直接等同于人体疗效，是典型的伪科学误导。
- 国家药品监督管理局（NMPA）未批准任何蜂胶产品作为抗癌药物。

### 最终建议

该商品宣传内容严重违反中国广告法，属于欺诈行为。消费者不应购买，并应向当地市场监督管理局或12315平台举报。癌症患者应坚持正规医院的治疗方案，切勿因虚假宣传而放弃或延误治疗。
