# Case Analysis

**Input:** 专家提醒：消费返利理财太厉害了！每日返利1%稳赚不赔，推荐好友赚佣金，拉满收益翻倍，买一送一！

**Risk Level:** 10/10

**Verdict:** scam

---

## 详细分析

### 风险等级：10（明显诈骗）

### 风险信号识别

1. **虚假收益承诺**：宣传“每日返利1%稳赚不赔”，年化收益率高达365%，远超任何合法金融产品的正常回报率。这是典型的庞氏骗局特征，用后加入者的本金支付前加入者的利息，最终必然崩盘。

2. **传销式裂变机制**：“推荐好友赚佣金，拉满收益翻倍”是典型的传销或非法集资手法，鼓励发展下线，形成金字塔结构。

3. **绝对化用语**：“稳赚不赔”是金融监管明令禁止的广告用语，任何投资都有风险，不存在稳赚不赔。

4. **缺乏合法资质**：文字中未提及任何金融牌照或监管机构，属于非法金融活动。

### 事实核查

- 根据中国银保监会和公安部警示，任何承诺“稳赚不赔”“日返利”的项目几乎都是非法集资或诈骗。
- 类似“消费返利”骗局已被多地警方曝光，如“云联惠”“善林金融”等案例。

### 结论

该宣传内容完全符合庞氏骗局和非法集资的特征，风险极高。建议立即远离，并向当地公安机关或金融监管部门举报。
