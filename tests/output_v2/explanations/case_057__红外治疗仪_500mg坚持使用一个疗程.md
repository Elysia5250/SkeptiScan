# Case Analysis

**Input:** 【红外治疗仪】500mg坚持使用一个疗程，效果显著，无效全额退款！原价39.9限时特惠仅需198元！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 总体评价
该产品宣传内容存在多处严重风险信号，风险等级评定为 **8分（高度可疑）**，极有可能为虚假宣传或欺诈产品。

### 风险信号分析

1. **虚假原价与价格欺诈**：宣传中写明“原价39.9元”，但“限时特惠仅需198元”，特惠价反而比原价高出近5倍，这是典型的虚假促销手段，违反《价格法》相关规定。

2. **夸大治疗功效**：声称“坚持使用一个疗程，效果显著”，但未明确说明治疗何种疾病，也未提供任何临床证据。红外治疗仪属于医疗器械，必须经过严格的临床试验和审批才能宣称疗效，普通产品不得使用此类医疗用语。

3. **无效退款承诺**：虽然“无效全额退款”看似对消费者有利，但实际执行中往往设置苛刻条件或直接失联，是许多骗局中常用的套路，目的是降低消费者警惕性。

4. **价格异常**：原价39.9元的产品，即使打折后也不应高达198元，且红外治疗仪作为医疗器械，成本通常较高，39.9元的原价本身就不合理，暗示产品可能为劣质或普通发热贴片冒充。

### 事实核查建议
- 建议消费者通过国家药品监督管理局官网查询该产品是否具有医疗器械注册证号。
- 搜索该品牌及生产厂家的工商信息，查看是否有行政处罚或消费投诉记录。
- 在电商平台查看真实用户评价，注意区分刷单评价。

### 最终建议
**强烈不建议购买**。该产品宣传内容充满矛盾与虚假信息，极可能为三无产品或普通发热贴片冒充的治疗仪，不仅无法达到宣传效果，还可能因使用不当造成皮肤烫伤等风险。消费者应选择正规药店或医院推荐的、有明确医疗器械注册证的产品。
