# Case Analysis

**Input:** 蜂胶抗癌黄金胶囊——蜂胶王浆胶囊，国家食品药品监督管理局批准（批准文号：国食健字G2024XXXX），中国蜂产品协会推荐品牌。中国农业大学蜂。国药准字Z2025XXXX，GMP认证，三甲医院临床验证有效率91.3%

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细风险分析

### 1. 虚假功效宣称
产品名称直接包含“抗癌”二字，这是保健品或普通食品绝对禁止的宣称。根据《食品安全法》和《广告法》，非药品不得宣传疾病预防或治疗功能。蜂胶作为普通保健食品原料，从未被任何权威医学机构证实具有抗癌功效。

### 2. 批准文号混乱
- 同时标注“国食健字G2024XXXX”（保健食品批准文号）和“国药准字Z2025XXXX”（药品批准文号），这是极其可疑的信号。一个产品不可能同时拥有两种身份。
- 国药准字Z开头代表中药制剂，但蜂胶胶囊通常不属于中药范畴。且2025年的批准文号在2024年出现，时间逻辑矛盾。
- 建议立即通过国家药监局官网查询这两个文号，极大概率是伪造或盗用。

### 3. 虚假权威背书
- “中国蜂产品协会推荐品牌”——协会通常不会为具体产品背书，除非有商业合作，但宣传中未提供任何可查证的推荐文件。
- “中国农业大学蜂”——表述不完整，疑似冒用高校名称。
- “三甲医院临床验证有效率91.3%”——这是典型的伪科学数字。任何抗癌药物的有效率都需要经过严格的多中心随机双盲临床试验，91.3%的数字高得离谱，且未提供医院名称、试验编号、发表期刊等任何可查证信息。

### 4. 营销手法分析
- 使用多个权威机构名称堆砌，制造“官方认可”假象。
- 同时使用保健食品和药品批准文号，让消费者误以为产品既有保健功能又有药品疗效。
- 编造高有效率数字，利用消费者对“数字”的信任。
- 使用“黄金”“抗癌”等夸张词汇，吸引对健康焦虑的消费者。

### 5. 风险等级判定
综合以上分析，该产品存在多项严重欺诈特征：虚假功效宣称、伪造批准文号、冒用权威机构、编造临床数据。风险等级为9分（接近最高级10分），属于明确的诈骗产品。

### 6. 最终建议
- **绝对不要购买**。
- 如果已经购买，请保留证据并向当地市场监督管理局或12315投诉举报。
- 对于老年人群体，子女应加强防骗教育，提醒其任何声称能抗癌的保健品都是骗局。
- 购买保健品前，务必通过国家市场监督管理总局特殊食品信息查询平台核实批准文号。
