# Case Analysis

**Input:** 【蜂胶抗癌】30粒装，专家强烈推荐，央视合作品牌，品质有保障！立即抢购！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 评分理由
该商品宣传语包含多个高风险信号，综合评定风险等级为8（高度可疑）。

### 风险信号识别
1. **虚假疗效宣称**：明确声称“蜂胶抗癌”，这是典型的保健品或食品违法宣传。根据中国《广告法》和《食品安全法》，非药品不得宣传疾病预防或治疗功能。蜂胶作为普通食品或保健食品，没有任何科学证据表明其能治疗癌症。
2. **虚假权威背书**：使用“专家强烈推荐”和“央视合作品牌”等模糊表述，但未提供任何具体专家姓名、机构或央视合作证明。这类话术常见于欺诈性产品，利用消费者对权威的信任。
3. **紧迫感营销**：“立即抢购”制造稀缺感，诱导冲动消费，是典型的高压销售策略。

### 事实核查
- 蜂胶在传统医学中曾被用于伤口愈合等，但现代医学研究并未证实其具有抗癌功效。美国国家癌症研究所（NCI）和世界卫生组织（WHO）均未将蜂胶列为抗癌药物。
- 中国国家药品监督管理局（NMPA）批准的药品中，没有以“蜂胶”为主要成分的抗癌药。
- 所谓的“央视合作品牌”无法通过公开渠道核实，且央视从未为任何单一保健品品牌背书。

### 最终建议
该商品极有可能是利用老年人对癌症的恐惧心理进行欺诈。消费者应避免购买，并建议向当地市场监管部门举报。对于癌症患者，务必遵循正规医院的治疗方案，切勿相信任何声称能“抗癌”的保健品。
