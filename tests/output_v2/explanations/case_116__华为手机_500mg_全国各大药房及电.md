# Case Analysis

**Input:** 【华为手机】500mg，全国各大药房及电商平台均有销售。官方售价198元。

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 风险信号识别
- **价格异常**：华为手机官方售价通常在千元以上，198元远低于正常市场价，极不合理。
- **销售渠道矛盾**：手机通常通过电子产品商店或电商平台销售，而非药房。声称在药房销售，暗示产品可能为保健品或药品，但描述为“华为手机”，存在严重矛盾。
- **品牌滥用**：利用“华为”这一知名品牌吸引消费者，但未提供任何官方授权或验证信息。

### 风险评估
- **风险等级：9**（明显欺诈）
- 该商品极可能是假冒伪劣产品或诈骗，利用低价和知名品牌诱导消费者。

### 建议
- **不要购买**：避免财产损失和个人信息泄露。
- **举报**：向相关平台或监管部门举报该商品信息。
- **提醒家人**：尤其是老年人，容易受低价和品牌影响上当。
