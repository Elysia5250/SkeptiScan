# Case Analysis

**Input:** 【蜂胶抗癌】蜂胶抗癌活性成分是化疗药的10倍，抑制肿瘤扩散，多名患者康复，限时优惠仅需198元！

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细分析

### 风险评分：9/10（高度欺诈）

### 风险信号识别

1. **虚假医疗功效宣称**：文字明确声称蜂胶“抗癌活性成分是化疗药的10倍”、“抑制肿瘤扩散”，这属于未经科学验证的夸大疗效，违反了《广告法》和《食品安全法》中关于保健食品不得宣传疾病治疗功效的规定。

2. **伪科学包装**：蜂胶作为普通蜂产品，其抗癌活性从未在人体临床试验中得到证实。声称“是化疗药的10倍”缺乏任何可信的临床数据支持，属于典型的伪科学营销。

3. **虚假患者证言**：“多名患者康复”是常见的欺诈手段，无法核实具体病例，且可能侵犯患者隐私或纯属虚构。

4. **紧迫感营销**：“限时优惠仅需198元”制造购买压力，诱使消费者冲动决策。

### 事实核查

- 国家药品监督管理局（NMPA）未批准任何蜂胶产品作为抗癌药物。
- 美国国家癌症研究所（NCI）和世界卫生组织（WHO）均未将蜂胶列为抗癌疗法。
- 现有研究多为体外或动物实验，远未达到人体应用标准，且剂量与安全性未知。

### 结论

该商品宣传内容属于典型的“保健品冒充药品”欺诈，利用癌症患者的焦虑心理牟利。购买此类产品不仅浪费金钱，更可能延误正规治疗，危及生命。

### 建议

- 立即向当地市场监管部门或12315举报。
- 癌症患者应遵循正规医院的治疗方案，切勿轻信网络广告。
- 如需补充营养，应选择有信誉的品牌，并咨询医生或营养师。
