# Case Analysis

**Input:** 【红外治疗仪】远红外光波穿透皮下15cm，消炎止痛，不吃药不打针治好颈椎病，限时优惠仅需198元！

**Risk Level:** 8/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该商品为“红外治疗仪”，宣称通过远红外光波穿透皮下15cm，达到消炎止痛、治疗颈椎病的效果，售价仅198元，并采用“限时优惠”促销。

### 风险信号识别

1. **虚假疗效承诺**：明确宣称“治好颈椎病”，属于疾病治疗承诺。根据《广告法》，非医疗器械产品不得宣传治疗作用，即使有医疗器械注册证，也不能使用“治好”“根治”等绝对化用语。

2. **伪科学概念**：“远红外光波穿透皮下15cm”严重违背科学常识。医学研究表明，远红外线的穿透深度通常仅为数毫米至1厘米左右，15cm的穿透深度在人体组织中不可能实现，属于典型的伪科学夸大。

3. **低价陷阱**：正规医用红外治疗仪价格通常在数千元至上万元，198元的价格远低于成本，极可能为劣质或假冒产品。

4. **营销手段**：使用“限时优惠”制造紧迫感，诱导消费者冲动购买，是常见营销骗局手法。

### 风险等级评定
综合以上因素，该产品存在多项高风险信号：虚假疗效、伪科学、低价异常、营销欺骗。根据1-10风险量表，评定为**8级**，属于高风险（接近诈骗）。

### 建议
- **不要购买**：该产品极可能为无效甚至有害的假冒产品。
- **正规就医**：颈椎病需经专业医生诊断，采用物理治疗、药物或手术等科学方法。
- **举报**：如发现该产品在电商平台销售，可向平台或市场监管部门举报。
