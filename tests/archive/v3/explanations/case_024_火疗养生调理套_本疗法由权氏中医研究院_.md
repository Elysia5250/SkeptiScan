# Case Analysis

**Input:** 火疗养生调理套。本疗法由权氏中医研究院（已注销）创始人历经三十年研发，融合道医养生精华。我们不做治疗，只做调理。内服外用相结合：外用火疗打开毛孔排毒通道，内服灵芝孢子粉激活自愈力。

**Risk Level:** 8/10

**Verdict:** scam

---

### 风险分析报告

**产品名称**：火疗养生调理套

**风险等级**：8（高风险，接近诈骗）

**主要风险信号**：

1. **虚假权威背书**：产品声称由“权氏中医研究院（已注销）”创始人研发。机构已注销意味着其合法性存疑，使用已注销机构名称制造权威感是典型的欺诈手法。

2. **伪科学概念**：
   - “道医养生精华”：道医并非现代医学认可的学科，常被用于包装伪科学产品。
   - “火疗打开毛孔排毒通道”：排毒是伪科学概念，人体有肝脏、肾脏等自然排毒系统，火疗无法打开所谓的排毒通道，且可能造成烫伤。
   - “灵芝孢子粉激活自愈力”：灵芝孢子粉虽有一定保健作用，但“激活自愈力”属于夸大功效，缺乏临床证据。

3. **模糊医疗界限**：宣传语“不做治疗，只做调理”试图规避医疗监管，但实际暗示产品具有调理疾病的效果，容易误导消费者，尤其是老年人。

4. **内服外用组合**：将外用火疗与内服灵芝孢子粉结合，增加产品复杂性和购买理由，但两者均无可靠科学依据支持其宣称的功效。

**事实核查建议**：
- 查询权氏中医研究院的工商注册信息，确认其注销状态及历史。
- 搜索灵芝孢子粉的临床研究，看是否有证据支持“激活自愈力”。
- 了解火疗是否被任何正规医疗机构认可为有效疗法。

**结论**：该产品利用已注销机构名称、伪科学概念和夸大功效进行营销，风险极高，不建议购买。消费者应选择有正规资质、科学依据的产品，并咨询专业医生。
