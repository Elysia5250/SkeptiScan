# Case Analysis

**Input:** 南极磷虾油胶囊。不替代药品治疗。上海某医院（研究合作协议，非临床验证）的初步实验表明，36例高血脂志愿者服用3个月后，总胆固醇平均下降1.2mmol/L。适宜人群：血脂偏高者。

**Risk Level:** 5/10

**Verdict:** not_scam

---

## 风险分析报告

### 产品概述
南极磷虾油胶囊，宣称适宜血脂偏高者，并引用与上海某医院的研究合作协议（非临床验证）作为支持，声称36例高血脂志愿者服用3个月后总胆固醇平均下降1.2mmol/L。

### 风险信号识别
1. **研究合作协议误导**：文字中明确写“研究合作协议，非临床验证”，但普通消费者可能忽略此细节，误以为有医院背书。这种表述模糊了研究合作与正式临床验证的界限，属于营销技巧。
2. **小样本数据**：仅36例志愿者，且未说明是否随机、双盲、有安慰剂对照。在营养学研究中，小样本、无对照的结果可靠性较低，不能作为有效证据。
3. **具体数值暗示疗效**：给出“下降1.2mmol/L”的具体数值，容易让消费者认为效果确切，但实际缺乏大规模研究支持。
4. **保健品定位**：产品明确为“胶囊”，未声称是药品，但降血脂效果宣传容易让消费者误以为可替代药物。

### 评分依据
- **风险等级5**：属于“可疑”范畴。产品没有明显的伪科学术语（如量子、磁疗等），也没有虚假权威认证，但利用医院合作和具体数值进行暗示性宣传，存在夸大效果的风险。
- 未发现疾病治疗承诺（如“治愈高血脂”），因此未达到7分以上。
- 但消费者需警惕：保健品不能替代药品，尤其对于高血脂这类慢性病，擅自停用药物可能带来健康风险。

### 建议
- 购买前确认产品有国家保健食品批准文号（蓝帽子）。
- 咨询医生或药师，不要仅凭宣传数据决定使用。
- 理性看待“研究合作协议”——合作不等于验证，更不等于推荐。

### 结论
该产品属于合法保健品，但宣传方式存在误导性，风险中等。建议消费者谨慎对待，尤其老年人应避免因宣传而停用正规药物。
