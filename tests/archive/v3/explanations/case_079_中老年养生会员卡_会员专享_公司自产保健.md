# Case Analysis

**Input:** 中老年养生会员卡。会员专享：公司自产保健品全场8折、优先预约国内专家（退休副主任医师视频问诊5分钟）、定期旅游。本卡一经售出，概不退换。每月15日会员日，全场买二送一！仅限会员参加。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 5/10

**Verdict:** not_scam

---

## 风险分析报告

### 总体评估
该商品为中老年养生会员卡，风险等级为 **5/10**，属于**可疑**级别。虽然未发现明显的虚假医疗或投资诈骗特征，但存在多处模糊宣传和不利于消费者的条款，需要谨慎对待。

### 风险信号分析

1. **模糊的专家服务**：
   - 声称“优先预约国内专家（退休副主任医师视频问诊5分钟）”，但未明确专家名单、资质认证来源，以及“优先”的具体含义。退休副主任医师的资质无法直接验证，且5分钟视频问诊的实际医疗价值有限。

2. **不透明的旅游服务**：
   - “定期旅游”未说明旅游目的地、频率、费用承担方式（是否包含在会员费中），可能为诱导消费的附加条件。

3. **价格不透明**：
   - “市场价仅供参考，实际价格以门店为准”这一条款使消费者无法预先了解真实价格，容易在购买后遭遇价格变动。

4. **限制消费者权利**：
   - “本卡一经售出，概不退换”违反《消费者权益保护法》中关于预付卡退款的合理规定，属于不公平格式条款。

5. **营销策略**：
   - 使用“会员专享”“仅限会员参加”等话术制造稀缺感，诱导冲动消费。
   - 保健品折扣可能鼓励过度购买，而保健品本身并非必需品。

### 未发现的风险
- 无直接虚假医疗承诺（如“治愈疾病”）。
- 无投资回报承诺。
- 无明显的伪科学术语（如量子、磁疗）。

### 建议
- **核实公司背景**：查询该公司是否在工商部门注册，是否有保健品销售资质。
- **检查保健品批号**：要求提供具体保健品的批准文号，并在国家药监局官网查询。
- **明确服务细节**：要求书面说明专家问诊的预约流程、旅游的具体安排和费用。
- **警惕不可退款条款**：该条款可能不合法，建议与商家协商修改或放弃购买。

### 结论
该会员卡风险中等，主要问题在于信息不透明和限制消费者权利。建议消费者在充分了解所有细节前不要购买，尤其是老年人应避免被“专家”“旅游”等宣传吸引而冲动消费。
