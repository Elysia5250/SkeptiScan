# Case Analysis

**Input:** 体验课低价引流-正式课高价。体验营期间可免费体验：针灸、拔罐、艾灸各一次。但名额有限，仅限60岁以上人士报名。调理方案费用：基础套餐19800元/年，尊享套餐39800元/年。市场价仅供参考，实际价格以门店为准。

**Risk Level:** 7/10

**Verdict:** scam

---

## 风险分析报告

### 风险等级：7（高风险，接近诈骗）

### 主要风险信号

1. **低价引流+高价转化**：体验课免费或低价，但后续推销19800-39800元/年的套餐，这是典型的营销陷阱，利用老年人贪图小便宜的心理。

2. **年龄限制**：仅限60岁以上人士，说明目标群体是健康焦虑高、信息辨别能力较弱的老年人，容易受到情感操控。

3. **价格不透明**：声称“市场价仅供参考，实际价格以门店为准”，为后续加价或隐藏费用留下空间。

4. **缺乏疗效证据**：虽然未直接宣称“治愈疾病”，但“调理方案”暗示能改善健康，而针灸、拔罐、艾灸属于中医理疗，对某些慢性病可能有辅助作用，但并非万能。高价套餐若未明确说明具体服务内容和预期效果，则存在虚假宣传风险。

### 营销手法分析

- **稀缺性**：“名额有限”制造紧迫感，促使老年人冲动消费。
- **免费体验**：让老年人先获得“甜头”，降低防备心理。
- **套餐分级**：基础/尊享套餐暗示不同疗效，但实际可能并无本质区别。

### 事实核查建议

- 查询该机构是否在卫健委备案，是否具备中医诊所资质。
- 核实操作人员是否持有《中医医师资格证》或《中医确有专长证书》。
- 对比正规医院或社区健康服务中心的中医理疗价格（通常单次针灸/拔罐/艾灸在50-150元之间，年度套餐远高于市场价）。
- 搜索该机构名称+“骗局”、“投诉”等关键词。

### 最终建议

**不推荐购买**。老年人应优先选择公立医院或正规社区卫生服务中心的中医理疗服务，价格透明且有保障。对于任何要求预付高额费用的“调理方案”，应视为高风险，避免上当。
