# Case Analysis

**Input:** 【央视推荐】红外线治疗仪，利用远红外光波穿透皮下组织，消炎镇痛、活血化瘀。治疗颈椎病、腰椎间盘突出、关节炎。不用吃药不用手术！今日下单仅需999（原价5999）。

**Risk Level:** 8/10

**Verdict:** scam

---

## 详细分析

### 风险评分：8/10（高度可疑，接近诈骗）

### 主要风险信号

1. **虚假疗效宣称**：产品声称能治疗颈椎病、腰椎间盘突出、关节炎等慢性疾病，这些疾病通常需要综合治疗（药物、理疗、手术等），而该产品暗示仅靠红外线照射即可治愈，属于典型的虚假医疗广告。根据《广告法》第十七条，除医疗、药品、医疗器械广告外，禁止其他任何广告涉及疾病治疗功能。

2. **伪科学包装**：使用“远红外光波穿透皮下组织”等术语，虽然远红外线在理疗中确有促进局部血液循环的作用，但宣称能“消炎镇痛、活血化瘀”并治疗具体疾病，缺乏高质量临床证据支持。此类产品常利用消费者对物理疗法的模糊认知夸大效果。

3. **虚假权威背书**：标注“央视推荐”但未提供任何具体节目名称、播出时间或可查证链接。央视作为国家级媒体，不会随意推荐未经验证的医疗器械。此类手法常见于欺诈性广告。

4. **营销陷阱**：原价5999元现价999元，大幅降价制造稀缺感和紧迫感，诱导冲动消费。同时“不用吃药不用手术”迎合患者对简单疗法的渴望，降低警惕。

### 事实核查结果

- 经搜索，未发现任何央视节目明确推荐该品牌或型号的红外线治疗仪。
- 国家药品监督管理局（NMPA）数据库中未查到该产品（或类似描述产品）的医疗器械注册信息。
- 权威医学机构（如梅奥诊所、中国康复医学会）指出，远红外线理疗仅作为辅助手段，不能替代正规治疗，且对颈椎病、腰椎间盘突出等疾病的疗效证据不足。

### 结论

该产品属于高风险欺诈性商品，利用虚假权威背书和伪科学概念，针对慢性病患者进行营销。强烈建议不要购买，并提醒家人尤其是老年人提高警惕。
