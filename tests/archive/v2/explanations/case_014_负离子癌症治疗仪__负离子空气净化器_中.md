# Case Analysis

**Input:** 负离子癌症治疗仪——负离子空气净化器，中国家用电器检测中心认证合格，通过CCC强制认证。搭载第7代负离子发生技术，释放8000万/cm3高浓。已获美国FDA注册，通过欧盟CE认证

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品以“负离子癌症治疗仪”为名，实际为负离子空气净化器，宣传中强调多项认证（CCC、FDA、CE）和高浓度负离子（8000万/cm³），暗示具有治疗癌症的功效。

### 风险信号识别
1. **虚假医疗功效宣称**：产品名称直接包含“癌症治疗”，这是最严重的危险信号。根据中国《广告法》和《医疗器械监督管理条例》，非医疗器械不得宣称疾病治疗功效。负离子空气净化器属于家电，不能治疗癌症。
2. **认证滥用**：CCC认证仅代表电器安全，FDA注册可能仅针对空气净化器（非医疗用途），CE认证同样不证明医疗效果。商家利用这些认证误导消费者认为产品有医疗权威性。
3. **伪科学概念**：负离子在医学上尚无证据能治疗癌症，高浓度负离子可能产生臭氧等有害物质，反而危害健康。
4. **营销话术**：使用“第7代技术”“8000万/cm³”等数字夸大效果，制造科技感，实为常见营销陷阱。

### 事实核查建议
- 查询美国FDA官网（www.fda.gov）的医疗器械数据库，输入产品名称或制造商，确认是否作为“癌症治疗”设备注册。通常此类产品仅注册为“空气净化器”。
- 中国家用电器检测中心仅检测电器安全性能，不验证医疗功效。
- 权威医学机构（如中国抗癌协会、WHO）均未认可负离子治疗癌症。

### 评分理由
根据评分标准，该产品明确宣称治疗癌症（虚假医疗功效），使用伪科学概念（负离子治疗），并滥用认证，符合9-10分“明显欺诈”特征。因此评为9分。

### 最终建议
**强烈不建议购买**。癌症患者应寻求正规医院治疗，切勿因虚假宣传延误病情。消费者可向当地市场监管局举报该产品。
