# Case Analysis

**Input:** 蜂胶抗癌黄金胶囊——蜂胶王浆胶囊，国家食品药品监督管理局批准（批准文号：国食健字G2024XXXX），中国蜂产品协会推荐品牌。中国农业大学蜂。国药准字Z2025XXXX，GMP认证，三甲医院临床验证有效率91.3%

**Risk Level:** 9/10

**Verdict:** scam

---

## 风险分析报告

### 产品概述
该产品宣称是“蜂胶抗癌黄金胶囊”，同时标注了“国食健字G2024XXXX”（保健食品批准文号）和“国药准字Z2025XXXX”（药品批准文号），并声称由中国蜂产品协会推荐、中国农业大学参与、三甲医院临床验证有效率达91.3%。

### 风险信号识别

1. **批准文号矛盾**：同一产品不可能同时拥有保健食品和药品批准文号。国食健字和国药准字是两种完全不同的审批体系，前者仅允许保健功能声称，后者才可用于治疗疾病。同时标注两者是明显的伪造行为。

2. **文号格式异常**：正规批准文号有固定格式，例如国食健字G+4位年份+4位序号，但示例中的“G2024XXXX”年份后直接跟4个X，不符合实际编号规则。国药准字Z2025XXXX同样存在格式问题。

3. **虚假疗效声称**：非药品（保健食品）不得宣称治疗疾病，更不允许使用“抗癌”字样。声称“三甲医院临床验证有效率91.3%”属于严重违规，且未提供任何医院名称、验证报告或文献来源。

4. **权威背书滥用**：中国蜂产品协会通常不会直接推荐具体产品，中国农业大学蜂业研究机构也极少为商业产品背书。此类声称多为虚构。

5. **GMP认证混淆**：GMP认证是药品生产质量管理规范，保健食品也有类似要求，但该产品同时标注GMP认证和国药准字，进一步加剧了混淆。

### 风险等级判定
综合以上分析，该产品存在多项严重欺诈特征：伪造批准文号、虚假疗效声称、滥用权威背书、混淆药品与保健品概念。风险等级评定为**9分**，属于高度危险的欺诈性产品。

### 建议
- **消费者**：立即停止购买和使用，已购买者可向当地市场监管局举报。
- **老年人**：此类产品常针对慢性病或癌症患者，利用患者求医心切的心理，务必提高警惕。
- **平台**：应下架该商品，并核查商家资质。
