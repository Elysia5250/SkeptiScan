# Case Analysis

**Input:** 蜂胶抗癌黄金胶囊——蜂胶王浆胶囊，国家食品药品监督管理局批准（批准文号：国食健字G2024XXXX），中国蜂产品协会推荐品牌。中国农业大学蜂。国药准字Z2025XXXX，GMP认证，三甲医院临床验证有效率91.3%

**Risk Level:** 9/10

**Verdict:** scam

---

## 详细风险分析

### 1. 批准文号矛盾
产品同时标注了“国食健字G2024XXXX”和“国药准字Z2025XXXX”。
- **国食健字**是保健食品批准文号，保健食品不得宣传疾病治疗作用（如抗癌）。
- **国药准字**是药品批准文号，格式通常为“国药准字H/Z/S+8位数字”，但年份2025尚未到来，且该文号无法在官方数据库查到。
- 同一产品不可能同时拥有保健食品和药品批准文号，这明显是伪造或混淆概念。

### 2. 虚假疗效宣称
“三甲医院临床验证有效率91.3%”是典型的虚假数据。
- 任何正规抗癌药物或保健食品的临床验证结果都会公开发表在医学期刊上，并注明具体医院、样本量、试验方法。
- 该产品未提供任何可查证的医院名称或文献编号，91.3%的有效率远超大多数抗癌药物，极不真实。

### 3. 权威背书可疑
“中国蜂产品协会推荐品牌”和“中国农业大学蜂”等表述模糊，未提供具体推荐函或合作证明。正规协会推荐通常会在官网公示，但该信息无法核实。

### 4. 法律违规
根据《食品安全法》和《广告法》，保健食品不得声称具有疾病预防或治疗功能。该产品明确宣称“抗癌”，已严重违法。

### 5. 风险等级判定
综合以上因素，该产品存在伪造批准文号、虚假疗效数据、违法宣传治疗作用、利用权威背书欺诈等多项高风险信号，符合9-10级“明显欺诈”标准。

### 最终建议
**立即停止购买和使用**，并向当地市场监督管理局或拨打12315举报。消费者应通过正规医院和药房购买药品，切勿相信任何宣称能治愈癌症的保健品。
